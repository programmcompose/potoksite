    Словарь Терминов #

    Словарь основных терминов музыкального производства.

    A #

    ADSR (Attack, Decay, Sustain, Release) — огибающая envelopы звука. Четыре параметра, определяющие, как звук развивается во времени.

    Подробнее: Основные параметры компрессора включают Attack и Release →

    Ableton Live — DAW от компании Ableton, популярна для электронной музыки.

    Подробнее: Выбор DAW в Этапе 0 → · Сравнение DAW →

    Amplifier (Амплификатор, Усилитель) — устройство для усиления аудиосигнала до уровня, достаточного для воспроизведения в колонках.

    Амплитуда (Amplitude) — величина колебания звуковой волны. Определяет громкость. Измеряется в дБ.

    Подробнее: Физика звука — Амплитуда →

    Аналоговый звук — непрерывный сигнал, в отличие от цифрового.

    Подробнее: Аналоговая и цифровая форма звука →

    Auto-Tune — автоматическая коррекция высоты тона вокала. Известен характерным «роботизированным» эффектом (T-Pain, Auto-Tune effect).

    Подробнее: Pitch и обработка вокала →

    Аудиоинтерфейс — устройство для подключения микрофонов и инструментов к компьютеру. Выполняет A/D и D/A конвертацию.

    Подробнее: Аудиоинтерфейс — ключевое оборудование →

    B #

    Band-pass Filter (Пропускающий фильтр) — фильтр, пропускающий полосу частот между нижней и верхней границей, отсекая всё остальное.

    Подробнее: Тональная коррекция — фильтры →

    Bitcrush (Биткраш) — эффект снижения битности и/или частоты дискретизации. Создаёт «разбитое», 8-битное звучание.

    Подробнее: Дисторшн и овердрайв →

    BPM (Beats Per Minute) — количество ударов в минуту. Определяет темп.

    Подробнее: BPM и жанры музыки → · Заметки о BPM в Этапе 0 →

    Bit Depth (Битность) — количество бит для представления каждого сэмпла. Влияет на динамический диапазон.

    Подробнее: Битность и динамический диапазон →

    Bus (Шина) — виртуальный канал для группировки и обработки нескольких треков.

    C #

    CD-Quality — стандарт качества аудио: 16-bit / 44.1 кГц.

    Подробнее: Стандарты частоты дискретизации →

    Chorus (Хорус) — модулятивный эффект, «умножающий» сигнал с лёгким расстройством, создающий объёмный, «хоровой» тембр.

    Подробнее: Пространственные эффекты — Chorus →

    Clip (Клип) — фрагмент аудио или MIDI на таймлайне DAW.

    Подробнее: Клиппинг при записи — чего избегать →

    Compressor (Компрессор) — плагин для уменьшения динамического диапазона.

    Подробнее: Компрессор при сведении → · Подборка компрессоров →

    DAW (Digital Audio Workstation) — программа для создания музыки на компьютере.

    Подробнее: Выбор DAW — полное руководство → · Подготовка к продакшну → · Схема подключения оборудования к DAW →

    D #

    Delay (Задержка) — эффект, создающий повторяющиеся повторения звука с затуханием. Базовый пространственный эффект.

    Подробнее: Пространственные эффекты — Delay →

    Distortion (Дисторшн) — сильное нелинейное искажение сигнала. Используется для агрессивного, «жёсткого» тембра.

    Подробнее: Дисторшн и овердрайв →

    E #

    Expander (Экспандер) — плагин, увеличивающий динамический диапазон: тихие звуки становятся ещё тише. Обратный компрессору эффект.

    Подробнее: Динамическая обработка — Expander →

    EQ (Equalizer, Эквалайзер) — плагин для корректировки частотного спектра.

    Подробнее: Эквалайзер — инструмент сведения → · Подборка EQ-плагинов →

    F #

    Fade — плавное изменение громкости (fade-in, fade-out).

    Flanger (Фленджер) — модулятивный эффект, смешивающий сигнал с его задержанной копией, создающий эффект «пролёта самолёта».

    Подробнее: Пространственные эффекты — Flanger →

    Frequency (Частота) — количество колебаний в секунду. Измеряется в Гц. Определяет высоту тона.

    Подробнее: Частота — основа физики звука → · Частотные диапазоны →

    Fuzz (Фаззер) — экстремальная форма дисторшн с полным «разрушением» формы волны. Характерен для гаражного рока иston-метала.

    Подробнее: Дисторшн и овердрайв →

    G #

    Gain — усиление сигнала. Не путать с громкостью (volume).

    Подробнее: Настройка gain при записи →

    Gain Staging — правильное распределение уровней сигнала на всех этапах.

    Granular (Гранулярный синтез) — техника, разбивающая аудио на микроскопические «зёрна» (granules) и обрабатывающая их независимо.

    Подробнее: Сэмпл- и временная обработка — Granular →

    Groove — ощущение «движения» в ритме. Часто достигается свингом и синкопой.

    Подробнее: Groove, синкопа и свинг →

    Harmonizer (Гармонизатор) — плагин, генерирующий гармонические голоса на основе входного сигнала.

    Подробнее: Pitch и обработка вокала →

    H #

    Headroom — запас между пиковым уровнем сигнала и 0 dBFS. Рекомендуется 3–6 dB.

    Подробнее: Headroom в dBFS →

    Hi-Hat (Хай-хэт) — перкуссионный инструмент из двух цимбал.

    Подробнее: Hi-Hat в ритм-секции →

    High-Pass Filter (HPF) — фильтр, пропускающий частоты выше определённой.

    Подробнее: HPF при сведении →

    I #

    Impulse Response (IR) — образец акустических свойств помещения. Используется в convolution-ревербераторах.

    Insert — плагин, вставленный на конкретный трек.

    L #

    LFO (Low-Frequency Oscillator, Низкочастотный осциллятор) — генератор сигнала низкой частоты для модуляции параметров (громкость, частота, тембр).

    Подробнее: Модулятивные эффекты — LFO →

    LPF (Low-Pass Filter, Низкочастотный фильтр) — фильтр, пропускающий частоты ниже определённой и отсекающий высокие.

    Подробнее: Тональная коррекция — фильтры →

    Latency (Латентность) — задержка между входом и выходом сигнала. Измеряется в миллисекундах.

    Подробнее: Latency аудиоинтерфейса → · Задержка звука и буфер →

    Limiter (Лимитер) — компрессор с очень высоким ratio (∞:1). Предотвращает клиппинг.

    Подробнее: Лимитер — финальный этап мастеринга →

    LUFS (Loudness Units Full Scale) — стандарт измерения громкости.

    Подробнее: Целевые уровни LUFS →

    M #

    MIDI (Musical Instrument Digital Interface) — протокол передачи музыкальной информации (не аудио!).

    Подробнее: MIDI-клавиатура и MIDI в продакшне →

    Mix (Микс) — результат сведения всех элементов трека.

    Подробнее: Создание микса → · Workflow создания трека →

    Mixing (Сведение) — процесс балансировки элементов в миксе.

    Подробнее: Сведение — полное руководство → · Зачем нужно сведение →

    Monitor (Монитор) — студийная акустическая система для контроля звука.

    Подробнее: Студийные мониторы → · Подбор места для мониторов →

    Mastering (Мастеринг) — финальная обработка микса перед релизом.

    Подробнее: Мастеринг — полное руководство → · Мастеринг в workflow →

    Multi-effects (Мульти-эффекты) — плагины, объединяющие несколько эффектов в одной цепи (последовательно, параллельно или комбинированно).

    Подробнее: Экспериментальные эффекты →

    N #

    Noise Gate (Нойз-гейт) — плагин, подавляющий сигнал ниже заданного порога. Убирает фоновый шум, шипение, гул.

    Подробнее: Динамическая обработка — Noise Gate →

    O #

    Overdrive (Овердрайв) — мягкое нелинейное искажение, имитирующее перегрузку лампового предусилителя. Тёплее и «округлее» дисторшн.

    Подробнее: Дисторшн и овердрайв →

    P #

    Panning (Панорамирование) — размещение звука в стереополе (лево/право).

    Подробнее: Пространство при сведении →

    Phaser (Фейзер) — модулятивный эффект, создающий движущиеся впадины и пики в спектре через фазовый сдвиг.

    Подробнее: Пространственные эффекты — Phaser →

    Pitch Shifter (Пич-шифтер) — плагин для изменения высоты тона сигнала без изменения темпа.

    Подробнее: Pitch и обработка вокала →

    Plugin (Плагин) — программное расширение DAW (VST, AU, AAX).

    Подробнее: Подборка плагинов для изучения → · Установка VST-плагинов →

    Preamp (Предусилитель) — устройство для усиления слабого сигнала микрофона до рабочего уровня.

    Подробнее: Preamp в аудиоинтерфейсе →

    R #

    Reverb (Реверберация) — эффект, имитирующий отражения в помещении.

    Подробнее: Реверберация при сведении → · Подборка реверб-плагинов →

    Ring Modulator (Ринг-модулятор) — эффект, умножающий два сигнала, создающий металлические, безобертонные тембры.

    Подробнее: Модулятивные эффекты — Ring Mod →

    S #

    Sample Rate (Частота дискретизации) — количество измерений в секунду. Стандарт: 44.1 кГц или 48 кГц.

    Подробнее: Частота дискретизации →

    Saturation (Сатурация) — мягкое искажение, добавляющее «тёплоту».

    Подробнее: Сатурация в цепочке мастеринга → · Плагины сатурации →

    Sidechain (Сайдчейн) — техника, при которой один сигнал управляет параметрами другого (часто: kick «качает» bass).

    Подробнее: Sidechain в компрессии →

    Spectrum (Спектр) — распределение энергии сигнала по частотам.

    Подробнее: Визуализация частотного спектра →

    Stereo (Стерео) — двухканальный звук (лево + право).

    Stutter (Статтер) — эффект заикания: быстрое повторение коротких фрагментов звука, создающий «глитч-ритм».

    Подробнее: Сэмпл- и временная обработка →

    T #

    Time Signature (Временная подпись) — количество ударов в такте и тип ноты-единицы (например, 4/4).

    Подробнее: Временные подписи →

    Time-stretching (Тайм-стретчинг) — изменение длительности/темпа аудио без изменения высоты тона.

    Подробнее: Сэмпл- и временная обработка →

    Transient (Транзиент) — начальная часть звуковой волны (атака).

    Подробнее: Работа с транзиентами →

    Track (Трек) — отдельный канал в DAW для аудио или MIDI.

    Подробнее: Организация треков в DAW →

    Tremolo (Тремоло) — модулятивный эффект, периодически изменяющий амплитуду (громкость) сигнала.

    Подробнее: Модулятивные эффекты — Tremolo →

    True Peak — максимальный уровень сигнала, включая интерполяционные пики.

    Подробнее: True Peak при мастеринге →

    V #

    Vibrato (Вибрато) — модулятивный эффект, периодически изменяющий высоту тона. Придаёт звуку «живость» и выразительность.

    Подробнее: Модулятивные эффекты — Vibrato →

    VST (Virtual Studio Technology) — стандарт аудио-плагинов.

    Подробнее: VST-плагины — подборка → · Установка VST →

    Volume (Громкость) — воспринимаемая громкость звука.

    Waveform (Форма волны) — визуальное представление звуковой волны.

    Подробнее: Форма волны — тембр звука →
