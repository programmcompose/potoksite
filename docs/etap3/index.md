Третий этап — переход от простых битов к сложным композициям. Вы научитесь создавать мелодии, работать с VST-синтезаторами, чопить сэмплы и усложнять драмку.

## Что в этом этапе

1. **Первая мелодия** — создание мелодии с нуля, поиск тональности
2. **VST Синтезаторы** — работа с синтезаторами, пресеты, настройка
3. **Мелодии из One-Shot** — создание мелодий из одиночных сэмплов
4. **Мелодия на луп** — работа с готовыми лупами, адаптация
5. **Тренировка мелодий** — упражнения для развития мелодического мышления
6. **Chop мелодий** — нарезка и перестановка мелодических сэмплов
7. **Рендер и Consolidate** — экспорт и объединение дорожек
8. **Усложнение драмки** — вариации, заполнение, паттерны
9. **BoomBap Биты** — классический хип-хоп стиль
10. **Rock Beats / NFS** — рок-влияние в битах
11. **Жесткие биты** — агрессивные паттерны, дисторшн
12. **Обработки FX** — эффекты для усложнения микса
13. **Структура трека** — построение от интро до аутро
14. **Фишки для спидрана** — ускорение рабочего процесса
15. **Мастеринг** — финальная обработка трека этапа

!!! important
    **Главная цель этапа:** создать несколько усложнённых битов в разных жанрах, применяя все изученные техники.
