# Chop мелодий

Chopping — техника нарезки мелодии на фрагменты и их ритмического rearrange'а.

## Ритмика chop-мелодий

Chop — это не просто нарезка. Это **ритмическое переосмысление** мелодии.

### Принципы chop

- Берёте мелодию и режете её на **короткие фрагменты**
- Расставляете фрагменты в **новом ритмическом порядке**
- Создаёте **неожиданные паттерны** из знакомых нот

## Рендер мелодии перед chop

Перед нарезкой нужно **рендернуть** (экспортировать) мелодию как аудио.

### Почему рендер важен

- Chop работает с **аудио**, а не MIDI
- Рендер «фиксирует» звук с всей обработкой
- Аудио проще резать и манипулировать

## Fruity Slicer (FL Studio)

Встроенный инструмент для автоматической нарезки.

### Как использовать

1. Загрузите аудио в **Playlist**
2. Добавьте **Fruity Slicer** на трек
3. Нажмите **Auto-slice** — программа нарезает по акцентам
4. Каждый слайс появляется как отдельная нота в пиано-ролле
5. Редактируйте порядок, длительность, высоту слайсов

### Настройки Slicer

| Параметр | Описание |
|---|---|
| **Slice mode** | Auto (по акцентам) или Manual (вручную) |
| **Note spacing** | Расстояние между нотами в пиано-ролле |
| **Retrig mode** | Как слайс реагирует на повторное триггерирование |

## Ableton (Simpler + Slice)

### Slice to new MIDI track

1. Выделите аудио в **Arrangement View**
2. Нажмите **Ctrl+M** (Slice to new MIDI track)
3. Ableton автоматически нарезает аудио по транзам
4. Каждый слайс загружается в **Simpler**
5. Редактируйте в **Piano Roll**

### Ручной chop в Ableton

1. Разрежьте клип вручную (Ctrl+T)
2. Перетащите фрагменты в **Simpler**
3. Настройте каждый слайс индивидуально
4. Создавайте паттерны в **Drum Rack**

!!! tip
    **Chop — это творчество, а не математика.** Не пытайтесь быть «правильным» — экспериментируйте с порядком, высотой и длительностью слайсов.
