# Тренировка написания мелодий

Регулярная практика — ключ к развитию навыка написания мелодий.

## Импровизация

Импровизация — свободное создание мелодий без подготовки.

### Как тренировать импровизацию

1. **Установите ограничение по времени** — 5 минут на мелодию
2. **Выберите случайную тональность** — не думайте, действуйте
3. **Используйте один инструмент** — ограничьте выбор
4. **Не останавливайтесь** — даже если «не получается»

### Упражнения для импровизации

| Упражнение | Описание |
|---|---|
| **Одна октава** | Пишите мелодию, используя только одну октаву |
| **Три ноты** | Создайте мелодию из трёх нот |
| **Случайный BPM** | Генерируйте случайный темп и пишите под него |
| **Слепая запись** | Закройте глаза и играйте на клавиатуре |

## Добавляем мелодию на сэмпл из паков

Сэмпл-паки — отличный источник идей.

### Процесс работы с паками

1. **Откройте пак** и прослушайте содержимое
2. **Выберите 2-3 сэмпла**, которые сочетаются
3. **Определите тональность** сэмплов
4. **Напишите мелодию**, дополняющую сэмплы
5. **Экспериментируйте** с комбинациями

### Советы по работе с паками

- Не перебирайте сэмплы бесконечно — выберите и работайте с ними
- Комбинируйте unexpected пары (пианино + синтезатор)
- Меняйте высоту сэмплов для создания вариаций
- Используйте **сэмплы как вдохновение**, не как основу

!!! important
    **Практика каждый день по 30 минут лучше одной 5-часовой сессии раз в неделю.** Формируйте привычку.
