# Рендер и Consolidate

Рендер — экспорт аудио из DAW. Consolidate — объединение аудиофрагментов в один файл. Критически важные навыки для сэмплирования.

## Зачем рендер и consolidate

| Причина | Описание |
|---|---|
| **Сэмплирование** | Рендерите мелодию → режьте → сэмплируйте фрагменты |
| **Оптимизация** | Уменьшает нагрузку на CPU |
| **Фиксация звука** | Сохраняет обработку и эффекты |
| **Коллаборация** | Отправляете аудио, а не проект |

## FL Studio

### Render (экспорт)

1. Нажмите **File → Export**
2. Выберите формат: **WAV** (качество) или **MP3** (размер)
3. Установите **24-bit** для максимального качества
4. Нажмите **Export**

### Consolidate (объединение)

1. Выделите MIDI-клип или аудиофрагменты
2. Нажмите **Right click → Bounce renderable to new audio clip**
3. Результат появляется в **Playlist** как единый аудиофайл
4. Используйте **Consolidate** для объединения нескольких фрагментов

### Горячие клавиши FL Studio

| Действие | Клавиши |
|---|---|
| Render | **Ctrl+R** |
| Bounce to audio | **Right click → Bounce** |
| Consolidate | **Ctrl+T** (на аудио) |

## Ableton

### Render (Export)

1. Нажмите **File → Export → Audio**
2. Выберите **WAV**, **24-bit**
3. Установите **Ignore Routing** (если нужно всё сразу)
4. Нажмите **Export**

### Consolidate

1. Выделите MIDI- или аудиофрагменты в **Arrangement View**
2. Нажмите **Ctrl+J** (Consolidate)
3. Фрагменты объединяются в один аудиофайл
4. Используйте **Flatten** для удаления автоматизации

### Горячие клавиши Ableton

| Действие | Клавиши |
|---|---|
| Export Audio | **Ctrl+Shift+M** |
| Consolidate | **Ctrl+J** |
| Flatten | **Ctrl+Shift+D** |

!!! important
    **Всегда рендерите в WAV 24-bit.** MP3 теряет качество при каждом перекодировании. Для сэмплирования качество критично.
