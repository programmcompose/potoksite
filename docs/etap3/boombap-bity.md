## Сэмплирование лупов

Основа BoomBap — **сэмплирование** винтажных записей.

### Процесс сэмплирования

1. Найдите винтажную пластинку или запись (соул, фанк, джаз)
2. Выберите фрагмент (2-4 такта)
3. Зациклите фрагмент
4. Скорректируйте BPM под проект
5. Добавьте ударные поверх

### Где искать лупы

- **Виниловые пластинки** — аутентичный звук
- **Splice** — фильтруйте по "Boom Bap", "Vintage"
- **YouTube** — старые записи (для обучения)
- **Сэмпл-паки** — специализированные BoomBap-паки

## Винтажные семплы

### Характеристика винтажного звука

| Характеристика | Почему важно |
|---|---|
| **Теплота** | Аналоговая запись warmer |
| **Шум** | Виниловый шип — часть характера |
| **Неидеальность** | Лёгкие искажения добавляют жизнь |
| **Ограниченный диапазон** | Старые записи не были «стерильными» |

### Имитация винтажности

- **Lo-fi EQ** — срежьте высокие частоты (low-pass 10-12 кГц)
- **Виниловый шум** — добавьте crackle сэмпл
- **Варп** — лёгкое растяжение для «плавающего» BPM
- **Сатурация** — тёплое искажение

## Ритмика и Грув

### Грув — душа BoomBap

- **Swing 60-70%** — выраженный кач
- **Humanize** — каждый удар чуть смещён
- **Velocity** — разная громкость хэтов
- **Неидеальная сетка** — живые барабаны не идеальны

## Живой звук

### Запись живых барабанов

- **Найдите барабанщика** или используйте сессионные записи
- **Запишите отдельно** каждый элемент (kick, snare, hats)
- **Time-stretch** под BPM проекта
- **Смешайте** живые барабаны с программными

### Почему живой звук

- **Неповторимость** — каждая запись уникальна
- **Эмоция** — живой музыкант вносит чувства
- **Авторитет** — продюсеры уровня Pete Rock, DJ Premier используют живые барабаны

## Сэмплирование Slice и Chop

### Serato Sample

Сэмплер, созданный для DJ-стиля работы.

- **Hot cues** — быстрые точки срабатывания
- **Key detection** — автоматическое определение тональности
- **Filter + Drive** — встроенная обработка
- **Ideal для** — виниловой имитации

### Fruity Slicer

Описание см. в главе [Chop мелодий](chop-melodiy.md).

Для BoomBap используйте:

- **Мелодические лупы** → нарезка → rearrange
- **Вокальные лупы** → chop → ритмический вокал

### Sampler Ableton

- Загрузите луп в **Sampler**
- Нарезайте вручную или через **Slice to MIDI**
- Настраивайте ** ADSR** для BoomBap-характера
- Добавляйте **Vinyl Long Beat** для винтажности

!!! tip
    **BoomBap — это не про сложность. Это про характер.** Простой кик, снейр и сэмплированная мелодия бьют сильнее перегруженного продакшна.
