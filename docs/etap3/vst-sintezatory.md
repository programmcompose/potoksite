# VST Синтезаторы

Синтезатор — инструмент для создания звука с нуля. Правильный выбор синта определяет характер вашей мелодии.

## Какой синтезатор выбрать

### Критерии выбора

| Критерий | Что важно |
|---|---|
| **Простота** | Для начала — интуитивный интерфейс |
| **Звук** | Теплый, чистый, характерный |
| **Пресеты** | Готовые пресеты ускоряют работу |
| **Гибкость** | Возможность тонкой настройки |

## Встроенные синтезаторы Ableton

### Operator

FM-синтезатор. Идеален для:

- Перикуссии и колокольчиков
- Пластиковых, цифровых звуков
- Басов с характерным окрасом

### Wavetable

Wavetable-синтезатор. Универсальный инструмент:

- Широкий спектр звуков
- Плавные переходы между волнами
- Отлично для лидов и пэдов

### Analog

Аналоговый эмулятор:

- Тёплые, «жирные» звуки
- Классические басовые пресеты
- Простой и понятный интерфейс

### Sampler

Сэмплер для работы с one-shot сэмплами:

- Загрузка любых аудиофайлов
- Настройка ADSR и фильтров
- Идеален для клавишных и ударных

## Repro-5

Популярный бесплатный синтезатор от Mistic.

- **5 осцилляторов** — богатый звук
- **Простой интерфейс** — легко начать
- **Отличный для** — лидов, пэдов, басов
- Бесплатный, что делает его доступным для всех

## Zenology

Библиотека звуков от Output.

- **100 000+ пресетов** — огромный выбор
- **Категоризация** — легко найти нужный звук
- **Интеграция** — работает как VST и standalone
- **Для чего** — лиды, пэды, текстыры, FX

## SERUM 2

Легендарный wavetable-синтезатор от Xfer Records.

### Почему SERUM

- **Визуальный wavetable editor** — видите форму волны
- **Огромное сообщество** — тысячи пресетов
- **Маппинг** — анимация параметров по времени
- **Пресеты** — от басов до лидов, всё покрыто

### SERUM для начинающих

1. Начните с готовых пресетов
2. Изучайте, какие параметры влияют на звук
3. Модифицируйте пресеты под себя
4. Создавайте свои звуки с нуля

## Пресеты для синтезаторов

### Где искать пресеты

- **Splice** — огромная библиотека
- **YouTube** — бесплатные паки
- **Telegram-чаты** — обмен между продюсерами
- **Сайты производителей** — официальные пресеты

### Как работать с пресетами

1. Не используйте пресет «как есть» — адаптируйте его
2. Меняйте тональность под свой трек
3. Корректируйте ADSR под контекст
4. Добавляйте свою обработку (EQ, сатурация)

## Корректировка ADSR

ADSR — огибающая звука (Attack, Decay, Sustain, Release).

| Параметр | Что делает | Пример |
|---|---|---|
| **Attack** | Время нарастания | Короткий — ударный звук, длинный — пэды |
| **Decay** | Время затухания до уровня sustain | Определяет «удар» ноты |
| **Sustain** | Уровень удержания | Насколько громкая нота при удержании |
| **Release** | Время затвора | Как быстро звук исчезает после отпускания |

!!! important
    **ADSR — это 50% характера звука.** Один и тот же пресет звучит совершенно по-разному с разными настройками ADSR.
