# Rock Beats / NFS

Рок-биты и NFS (Need For Speed) стиль — агрессивные, энергичные продакшны с живыми инструментами.

## Альтернатива 2000-х

### Влияние жанра

Музыка 2000-х определила звук целого поколения битов.

| Группа | Влияние на битмейкинг |
|---|---|
| **Deftones** | Атмосферные гитары, тяжёлые басы, эмоциональные переходы |
| **Limp Bizkit** | Агрессивная драмка, фанк-ритмика, энергия |
| **WISP** | Меланхоличные мелодии, дисторшн-текстуры |

### Что заимствовать

- **Гитарные риффы** — сэмплируйте или воспроизводите
- **Драм-паттерны** — двойные бочки, тяжёлые фиры
- **Атмосфера** — пространство между ударами

## Драмка

### Лайв драмка + Бит

Комбинация живых барабанов и программных элементов.

### Процесс

1. **Запишите** или найдите сессионные живые барабаны
2. **Подгоните BPM** под проект
3. **Добавьте** программный кик для удара
4. **Усиьте** снейр слоем 808 или синтезированного снейра
5. **Сбалансируйте** живые и программные элементы

### Советы по микшу

- Живые барабаны на **заднем плане** — атмосфера
- Программные элементы **вперёд** — удар и чёткость
- **EQ** — разделите частоты живых и программных элементов
- **Панорамирование** — живые барабаны шире, программные уже

## Сэмплы гитар и FX

### Лупы и подбор баса

| Тип гитары | Применение |
|---|---|
| **Clean** | Мелодии, атмосфера |
| **Distortion** | Агрессия, энергия |
| **Chorus** | Пространство, глубина |
| **Reverb** | Эмоциональность |

### Работа с гитарными лупами

1. Найдите луп в нужной тональности
2. Определите BPM и скорректируйте
3. EQ — срежьте низы (ниже 100 Гц), чтобы не конфликтовать с басом
4. Добавьте сатурацию для характера

### Атмосферные гитары

- **Delay** — эхо для пространства
- **Реверб** — большой холл для глубины
- **Chorus** — ширина и движение
- **Low-pass фильтр** — «далёкий» звук

### Дисторшн гитары

- **Cabinet simulation** — реалистичный звук гитарного комбо
- **Multiband компрессия** — контроль динамики
- **Сатурация** — дополнительная агрессия
- **EQ** — boost в 2-4 кГц для «кусачести»

## Сведение

### Рок и лайв звучание

| Параметр | Рок-микс | Электронный микс |
|---|---|---|
| **Динамика** | Больше динамики (меньше компрессии) | Больше компрессии |
| **Пространство** | Широкая панорама | Центрированный |
| **Реверб** | Больше, естественнее | Меньше, контролируемо |
| **Громкость** | Не так критично | Loudness war |

### Принципы рок-сведения

1. **Не сдавливайте** — рок живёт динамикой
2. **Живые инструменты** — панорамируйте широко
3. **Реверб** — используйте для «комнатного» звука
4. **Гитары** — двойная запись (double tracking) для ширины

!!! important
    **Рок-бит — это энергия, а не точность.** Не бойтесь «грязи» — она добавляет характера.
