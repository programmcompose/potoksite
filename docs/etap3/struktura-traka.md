# Построение бита / Структура

Хороший бит — это не только好听 звуки. Это **структура**, которая ведёт слушателя от начала до конца.

## Куплет / Припев

### Базовая структура

| Секция | Длительность | Описание |
|---|---|---|
| **Интро** | 4-8 тактов | Завлечение, настройка |
| **Куплет (Verse)** | 8-16 тактов | Основная мелодия/рэп |
| **Припев (Chorus/Hook)** | 8 тактов | Кульминация, запоминаемость |
| **Куплет 2** | 8-16 тактов | Вариация темы |
| **Припев 2** | 8 тактов | Повтор кульминации |
| **Аутро** | 4-8 тактов | Завершение |

### Различия куплет/припев

| Элемент | Куплет | Припев |
|---|---|---|
| **Энергия** | Умеренная | Максимальная |
| **Инструменты** | Меньше элементов | Все элементы |
| **Громкость** | Ниже | Выше |
| **Мелодия** | Проще | Ярче, сложнее |

## Интро / Бридж

### Интро

Первое впечатление — самое важное.

### Типы интро

| Тип | Описание |
|---|---|
| **Минималистичное** | 1-2 элемента, постепенное заполнение |
| **Эффектное** | Сразу все элементы, максимум энергии |
| **Атмосферное** | Реверб, эмбиент, медленное нарастание |
| **Сэмплированное** | Вокал, цитата, узнаваемый фрагмент |

### Бридж

Переходная секция между куплетом и припевом.

- **Изменение** мелодии или ритма
- **Накопление** энергии перед припевом
- **Breakdown** — удаление элементов для контраста

## Переход от лупа к финальному треку

Луп — 4-8 тактов. Трек — 2-4 минуты. Как превратить одно в другое?

### Стратегия развития

1. **Интро**: Только атмосфера + 1 элемент
2. **Построение**: Постепенное добавление элементов
3. **Кульминация**: Все элементы, максимум энергии
4. **Спад**: Удаление элементов
5. **Финал**: Возврат к интро или резкая остановка

### Техники перехода

| Техника | Применение |
|---|---|
| **Fill** | Драм-филл перед сменой секции |
| **Riser** | Нарастающий звук (white noise sweep) |
| **Breakdown** | Удаление ударных на 1-2 такта |
| **Drop** | Возврат всех элементов после паузы |
| **Automation** | Автоматизация фильтров, громкости |

## Анализ треков

### Работа с референсами

Референс — профессиональный трек, с которым вы сравниваете свой.

### Что анализировать

| Параметр | Что смотреть |
|---|---|
| **Структура** | Где начинаются/заканчиваются секции |
| **Энергия** | Где пик, где спад |
| **Инструменты** | Какие элементы когда появляются |
| **Переходы** | Как трек переходит между секциями |
| **Громкость** | Уровень мастеринга |

### Практика анализа

1. Выберите трек-референс в вашем жанре
2. Откройте его в DAW
3. Отметьте точки смены секций (маркеры)
4. Проанализируйте частотный спектр (анализатор)
5. Попробуйте воспроизвести структуру в своём проекте

!!! tip
    **Анализ референсов — навык, который отделяет любителей от профессионалов.** Тратьте 20% времени на анализ и 80% на создание.
