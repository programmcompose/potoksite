# Пишем мелодию на готовый луп

Луп — зацикленный аудиофрагмент с готовой мелодией. Написание мелодии поверх лупа — отличный тренировочный метод.

## Дополнение (Complement)

Не повторяйте луп — **дополняйте** его.

### Принципы дополнения

| Техника | Описание |
|---|---|
| **Контрапункт** | Параллельная мелодия, идущая в противофазе |
| **Ответ** | Луп задаёт фразу — вы отвечаете своей мелодией |
| **Заполнение** | Играйте в паузах лупа, не перекрывая его |
| **Усиление** | Дублируйте луп с вариациями на ключевых моментах |

## Атмосфера

Создание атмосферы — слой мелодии, задающий настроение.

### Инструменты для атмосферы

- **Пэды** — длинные, затухающие звуки на фоне
- **Струнные** — виолончель, скрипка для эмоциональности
- **Эмбиент-лупы** — фоновые текстуры
- **FX-звуки** — ризеры, свипы, переходы

### Как добавить атмосферу

1. Выберите 1-2 атмосферных элемента
2. Поставьте их на заднем плане (тише основной мелодии)
3. Добавьте реверб для глубины
4. Используйте фильтры для контроля частот

## Быстрота (Speed)

Работа с темпом мелодии влияет на восприятие.

### Техники изменения скорости

| Техника | Эффект |
|---|---|
| **Double-time** | Мелодия в 2 раза быстрее — энергия, напряжение |
| **Half-time** | Мелодия в 2 раза медленнее — тяжесть, глубина |
| **Triplets** | Троичное деление — грув, фанковый оттенок |
| **Syncopation** | Сдвиг акцентов — неожиданность, интерес |

### В DAW

- **FL Studio**: Измените длину нот в пиано-ролле или используйте **Time Stretch**
- **Ableton**: Измените **Length** на MIDI-нотах или используйте **Complex Pro** warping

## Практический подход

1. Найдите луп, который вам нравится
2. Определите его тональность и BPM
3. Напишите **дополняющую** мелодию (не копируйте)
4. Добавьте **атмосферный** слой
5. Экспериментируйте со **скоростью** и ритмом
6. Сбалансируйте громкости

!!! tip
    **Не перегружайте.** Одна сильная мелодия лучше трёх средних. Умение делать паузы — признак зрелого продюсера.
