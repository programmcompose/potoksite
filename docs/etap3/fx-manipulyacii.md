# Обработки FX и Манипуляции

FX-плагины — мощный инструмент для трансформации звуков. Правильная обработка превращает обычный звук в уникальный.

## Shaper Box 3

Multiband-процессор от IXPlug.

### Что умеет

| Функция | Описание |
|---|---|
| **Multiband distortion** | Разное искажение для разных частот |
| **Bitcrusher** | Цифровые артефакты, lo-fi |
| **Filter** | Мульти-фильтры с резонансом |

### Применение

- **Кик** — multiband distortion для «кусачести»
- **Снейр** — bitcrusher для lo-fi характера
- **Мелодия** — ring mod для экспериментальных текстур
- **Бас** — oscillator для добавления суб-слоя

## EFX Motions

Анимационный плагин для автоматизации параметров.

### Возможности

- **LFO** — модуляция параметров по времени
- **Шаговые секвенсоры** — программируемая автоматизация
- **Пресеты анимации** — готовые движения
- **Маппинг** — привязка к MIDI

### Практическое применение

1. **Анимация фильтра** — свипы и переходы
2. **Пульсация реверба** — динамическое пространство
3. **Движение панорамы** — автоматическое перемещение в stereo-поле
4. **Модуляция дисторшна** — пульсирующая агрессия

## BEAM

Stereo-имиджер от Output.

### Что делает

| Функция | Описание |
|---|---|
| **Stereo widening** | Расширение стерео-поля |
| **Mid/Side обработка** | Отдельная обработка центра и боков |
| **Phase correction** | Исправление фазовых проблем |
| **Prism** | Пространственные эффекты |

### Когда использовать

- **Мелодии** — ширина для присутствия
- **Пэды** — максимальное расширение
- **Вокал** — умеренная ширина
- **Kick/Bass** — моно (не расширяйте низкие частоты!)

!!! warning
    **Низкие частоты (ниже 150 Гц) всегда в моно.** Расширение низов создаёт фазовые проблемы и потерю энергии.

## PORTAL

Resonance-процессор от Output.

### Что умеет PORTAL

- **Convolution distortion** — уникальное искажение на основе импультсных ответов
- **Irreverb** — реверсированный реверб
- **Spectral freeze** — «замораживание» звука
- **Пресеты** — от тонкой обработки до полной трансформации

### Применение PORTAL

| Ситуация | Пресет | Эффект |
|---|---|---|
| **Переход** | Riser | Нарастающее напряжение |
| **FX** | Glitch | Цифровые артефакты |
| **Текстура** | Freeze | Атмосферный фон |
| **Мелодия** | Light distortion | Характер без разрушения |

## Реверс мелодии и стретчинг

### Реверс (Reverse)

Обратное воспроизведение мелодии.

- **Вступление** — реверс основной мелодии
- **Переходы** — реверс-свипы
- **FX** — реверс-хиты перед drop

### Стретчинг (Stretching)

Растяжение звука во времени.

| Техника | Эффект |
|---|---|
| **2x stretch** | Тягучий, глубокий звук |
| **0.5x stretch** | Быстрый, тревожный звук |
| **Extreme stretch** | Атмосферный, эмбиентный |
| **Time-stretch с артефактами** | Glitch-эффект |

## Фильтр мелодии для текстуры

Фильтры — простой, но мощный инструмент.

### Типы фильтров

| Фильтр | Звук | Применение |
|---|---|---|
| **Low-Pass** | Тёплый, приглушённый | Lo-fi, атмосфера |
| **High-Pass** | Тонкий, воздушный | Удаление низов |
| **Band-Pass** | Узкий диапазон | Телефонный эффект |
| **Notch** | Вырез частоты | Удаление проблемной частоты |

### Автоматизация фильтра

1. **Cutoff свип** — от закрытого к открытому (или наоборот)
2. **Resonance boost** — акцент на cutoff-частоте
3. **Filter LFO** — пульсирующий фильтр

!!! important
    **FX — это не замена плохому звуку.** Хорошая обработка усиливает хороший звук. Начните с качественного исходного материала.
