# Мелодии из One-Shot

One-shot — одиночный звуковой сэмпл (нота, удар, звук), который можно воспроизводить на разных высотах.

## Сэмплы и редактирование

### Что такое one-shot

One-shot — это **одиночная нота**, записанная как аудиофайл. В отличие от лупа, он не зациклен и воспроизводится один раз.

### Популярные типы one-shot

| Тип | Пример | Применение |
|---|---|---|
| **Пианино** | A4 piano note | Мелодии, аккорды |
| **Орган** | Hammond one-shot | Фанк, соул мелодии |
| **Синтезатор** | Pad, lead | Электронные мелодии |
| **Струнные** | Violin, cello | Кинематографичные биты |
| **808** | 808 bass note | Басовые линии |

### Где брать one-shot сэмплы

- **Splice** — фильтруйте по категории "Melodic" → "One Shots"
- **Сэмпл-паки** — каждый серьёзный пак содержит one-shot секцию
- **Своя запись** — запишите инструмент или синтезатор

## Определение ноты

Перед использованием one-shot важно знать, **какую ноту** он издаёт.

### Как определить ноту

| Способ | DAW | Описание |
|---|---|---|
| **Spectron** | FL Studio | Анализатор спектра — показывает ноту |
| **Note Detector** | Ableton | Автоматическое определение ноты |
| **Newtone** | FL Studio | Запись → анализ |
| **Слух** | Любая | Тренируете — определяете на слух |

### Настройка root note

1. Загрузите one-shot в сэмплер
2. Определите исходную ноту (например, A4)
3. Установите **Root Note** в сэмплере на эту ноту
4. Теперь при игре на MIDI-клавиатуре ноты будут правильными

## Корректировка ADSR

One-shot сэмплы особенно чувствительны к ADSR-настройкам.

### Настройки для разных жанров

| Жанр | Attack | Decay | Sustain | Release |
|---|---|---|---|---|
| **Trap** | Короткий | Средний | Низкий | Короткий |
| **Boom Bap** | Средний | Средний | Средний | Средний |
| **Lo-fi** | Длинный | Длинный | Высокий | Длинный |
| **Drill** | Короткий | Короткий | Низкий | Очень короткий |

### В Ableton (Simpler/Sampler)

- Откройте устройство **Simpler** или **Sampler**
- Найдите секцию **Envelope**
- Настройте **A/D/S/R** ползунки
- Экспериментируйте в реальном времени

### В FL Studio (Fruity Sampler)

- Откройте **Fruity Sampler**
- Найдите **ADSR** секцию
- Настройте параметры
- Используйте **Fade In/Out** для плавности

!!! tip
    **Короткий release** = чёткий, отрывистый звук. **Длинный release** = затухающий, атмосферный звук.

## Обязательные эксперименты: Pitch и Warp

### Pitch-сдвиг

Изменение высоты one-shot открывает новые возможности:

- **+5 полутонов** — более яркий, «игрушечный» звук
- **-7 полутонов** — тёмный, угрюмый звук
- **Микро-сдвиги** (±1-2 полутона) — добавление напряжения

### Warp-эффекты

Изменение скорости воспроизведения:

- **Ускорение** — энергичный, тревожный звук
- **Замедление** — тяжёлый, глубокий звук
- **Rubberbanding** — экстремальное растяжение для FX

!!! important
    **Pitch и Warp — ваши лучшие друзья.** Не бойтесь экспериментировать — самые интересные звуки рождаются из неожиданных комбинаций.
