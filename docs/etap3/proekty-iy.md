# Мои проекты от IY

Разбор реальных треков и практический анализ продакшна.

## FL Studio и Ableton

### Сравнение DAW

| Параметр | FL Studio | Ableton Live |
|---|---|---|
| **Паттерны** | Pattern-based workflow | Clip-based workflow |
| **Пiano Roll** | Лучший на рынке | Хороший, но уступает FL |
| **Встроенные плагины** | Fruit-плагины | Device-система |
| **Сэмплирование** | Fruity Sampler | Simpler/Sampler |
| **Сведение** | Mixer с routингом | Audio tracks + Return |
| **Live-выступления** | Ограничено | Отлично (Session View) |

## Треки-референсы для разбора

### Как проводить разбор

1. **Выберите трек** в вашем жанре
2. **Откройте в DAW** — импортируйте как аудио
3. **Поставьте маркеры** на смене секций
4. **Проанализируйте частоты** — используйте спектральный анализатор
5. **Определите структуру** — интро, куплет, припев, аутро
6. **Обратите внимание** на переходы и FX

### Что искать при разборе

| Элемент | Вопросы для анализа |
|---|---|
| **Драмка** | Какие паттерны? Какой swing? Где перкуссия? |
| **Мелодия** | Какая тональность? Какие инструменты? Chop? |
| **Бас** | 808 или синт? Слайды? Слои? |
| **Структура** | Как долго каждая секция? Как переходы? |
| **Сведение** | Что вперёд, что назад? Громкость? |
| **FX** | Какие эффекты? Где применяются? |

### Практическое задание

1. Выберите **3 трека** от разных продюсеров
2. Проведите разбор каждого по чек-листу выше
3. Запишите **наблюдения** — что работает, что нет
4. Попробуйте **воспроизвести** структуру одного из треков
5. Сравните свой результат с оригиналом

!!! important
    **Разбор чужих треков — самый быстрый способ учиться.** Вы начинаете понимать, почему профессиональные биты звучат «дорого».
