# Мастер

Мастеринг — финальная обработка трека перед релизом. Цель: баланс, громкость, готовность к платформам.

## Баланс частот

### Tonal Balance

Tonal Balance Controller — плагин для визуализации частотного баланса.

### Как использовать

1. Установите **Tonal Balance Controller** на мастер-канал
2. Загрузите **референсный трек** или выберите профиль жанра
3. Слушайте свой трек и наблюдайте за дисплеем
4. Корректируйте EQ, чтобы баланс соответствовал профилю

### Профили жанров

| Жанр | Низкие частоты | Средние | Высокие |
|---|---|---|---|
| **Trap** | Много (808) | Умеренно | Яркие |
| **Boom Bap** | Умеренно | Тёплые | Приглушённые |
| **House** | Глубокий кик | Чистые | Яркие хэты |
| **Drill** | Тяжёлый бас | Агрессивные | Резкие |

### Частотный чек-лист

- [ ] Суб-бас (20-60 Гц) — контроль, не перегруз
- [ ] Низкие (60-250 Гц) — фундамент бита
- [ ] Средние (250-4000 Гц) — мелодия, вокал
- [ ] Высокие (4000-20000 Гц) — яркость, воздух

## Ozone 12

iZotope Ozone 12 — комплексный мастеринг-пак.

### Модули Ozone

| Модуль | Назначение |
|---|---|
| **Maximizer** | Лимитер, финальная громкость |
| **Dynamic EQ** | Частотозависимая компрессия |
| **Imager** | Стерео-ширина |
| **Exciter** | Верхние гармоники, яркость |
| **EQ** | Коррекция баланса |
| **Dynamics** | Компрессия мастер-канала |
| **Reverb** | Финальное пространство |

### Порядок модулей

```
Master → EQ → Dynamic EQ → Compressor → Exciter → Imager → Maximizer
```

## Клиппинг и Лимитер

### Лимитер

Последний плагин в цепи. Защита от клиппинга.

### Настройки лимитера

| Параметр | Значение | Описание |
|---|---|---|
| **Ceiling** | -0.3 dB | Максимальный уровень |
| **Threshold** | По ситуации | Уровень срабатывания |
| **Ratio** | ∞:1 | Бесконечное ограничение |
| **Release** | Auto | Автоматический релиз |

### Клипперы для жира

Клиппер — более агрессивная альтернатива лимитеру.

### Почему клиппер

- **Более «жирный»** звук при той же громкости
- **Аналоговое** ограничение сигнала
- **Меньше артефактов** компрессии

### Применение клиппера

1. Поставьте клиппер **перед** лимитером
2. Установите threshold так, чтобы «срезало» 1-3 dB
3. Лимитер после клиппера — финальная защита
4. Результат: **громче и жирнее** без потери качества

!!! warning
    **Не гонитесь за громкостью.** Громкий трек с плохим балансом хуже тихого трека с хорошим балансом. Качество > громкость.
