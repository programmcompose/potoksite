# Аккорды и гармония

## Что такое аккорд?

**Аккорд** — это одновременно звучащие три или более нот. Большинство аккордов
строятся из **терций** — нот через одну в гамме.

## Мажорные и минорные аккорды

### Мажорный аккорд (Major)

Формула: **корень + большая терция + чистая квинта**

Пример C Major (Cmaj или просто C):
```
C + E + G
```
Звучание: яркое, радостное, стабильное

### Минорный аккорд (Minor)

Формула: **корень + малая терция + чистая квинта**

Пример C Minor (Cm):
```
C + D# + G
```
Звучание: мрачное, грустное, напряжённое

!!! tip
    Разница между мажором и минором — всего в **одном полутоне** (терция).
    Это самый мощный инструмент в гармонии.

## Базовые аккорды в C мажоре

| Степень | Аккорд | Ноты | Тип |
|---------|--------|------|-----|
| I | C | C-E-G | Мажор |
| ii | Dm | D-F-A | Минор |
| iii | Em | E-G-B | Минор |
| IV | F | F-A-C | Мажор |
| V | G | G-B-D | Мажор |
| vi | Am | A-C-E | Минор |
| vii° | Bdim | B-D-F | Уменьшенный |

## Популярные аккордовые прогрессии

### Прогрессия I – V – vi – IV

Самая популярная прогрессия в поп-музыке:

```
C → G → Am → F
```

Используется в: Let It Be (Beatles), With Or Without You (U2),
Don't Worry Be Happy (Lionel Richie)

### Прогрессия vi – IV – I – V

```
Am → F → C → G
```

Используется в: Someone Like You (Adele), Stay With Me (Sam Smith)

### Прогрессия I – vi – IV – V

"50s progression" — классика рока и джаза:

```
C → Am → F → G
```

## Септаккорды

Септаккорды добавляют **четвёртую ноту** (септиму), создавая более
сложное и «красивое» звучание.

| Тип | Формула от C | Ноты | Звучание |
|-----|-------------|------|----------|
| Cmaj7 | C-E-G-B | C-E-G-B | Тёплое, джазовое |
| Cm7 | C-D#-G-Bb | C-D#-G-Bb | Меланхоличное |
| C7 | C-E-G-Bb | C-E-G-Bb | Натяжение, блюз |
| Cminmaj7 | C-D#-G-B | C-D#-G-B | Мистическое |
