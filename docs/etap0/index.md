Перед началом создания музыки нужно выбрать DAW, настроить оборудование и рабочее место.

## Выбор DAW

DAW (Digital Audio Workstation) — главная программа для создания музыки.

### FL Studio 25
Интуитивный шаг-секвенсор и обширная экосистема плагинов. Отлично для электронной музыки, хип-хопа и попа.

### Ableton Live 12
Session View для импровизации и лупинга. Идеальна для живых выступлений и экспериментального продакшна.

### Другие DAW
Logic Pro, Reaper, Cubase, Studio One. Главное — выбрать одну и углубиться.

### Как выбрать
- Попробуйте демо-версии и создайте первый проект
- Интерфейс должен быть понятным — вы проведёте за программой сотни часов
- Логика организации проектов и дорожек должна быть вам близка
- Проверьте системные требования и совместимость с вашим оборудованием

## Оборудование

### Компьютер
WIN или MAC — обе платформы подходят. MAC часто предпочтительнее из-за оптимизации.

### Наушники и мониторы
Мониторные наушники с ровной АЧХ — минимум для начала. Студийные мониторы дают более точную картину — начните с одной пары.

### Дополнительное (опционально)
- **Микрофон** — для вокала и живых инструментов
- **MIDI-клавиатура** — для игры мелодий и управления плагинами
- **Аудиокарта** — для качественной записи и вывода звука

## Рабочее место

- Тихое место с минимальным фоновым шумом
- Мониторы на уровне ушей, образуя равносторонний треугольник
- Всё оборудование под рукой, без путаницы

## Аудио драйверы

ASIO — стандарт для минимальной задержки. На Windows используйте ASIO4All или драйвер вашей аудиокарты. Внешняя аудиокарта обеспечит лучшую задержку и качество.

## Установка программ и плагинов

- Начните с базового набора плагинов из комплекта DAW
- Устанавливайте последовательно: DAW → настройка → плагины
- Начните с бесплатных сэмпл-паков
- Проверяйте чат курса — там рекомендации по плагинам и сэмплам

## Оптимизация

- Закрывайте ненужные программы перед запуском DAW
- Буфер: 128-256 сэмплов для записи, 512-1024 для сведения

## Заметки

Xmind — ментальные карты. Obsidian — заметки со связями. Тетрадь — основные мысли, термины, идеи, порядок действий для написания музыки и сведения.
