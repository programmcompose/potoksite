# Этап №0 — Подготовка к продакшну

Перед тем как начать создавать музыку, важно правильно подготовиться: выбрать программу, настроить оборудование и рабочее место.

## Выбор основной программы DAW

DAW (Digital Audio Workstation) — это главная программа, в которой вы будете создавать музыку.

### FL Studio 25
Популярная DAW с интуитивным интерфейсом, мощным шаг-секвенсором и обширной экосистемой плагинов. Отлично подходит для электронной музыки, хип-хопа и поп-музыки.

### Ableton Live 12
Профессиональная DAW с уникальным Session View для импровизации и лупинга. Идеальна для живых выступлений, электронной музыки и экспериментального продакшна.

### Другие DAW
Logic Pro, Reaper, Cubase, Studio One — каждая имеет свои преимущества. Главное — выбрать одну и углубиться в неё.

## Какую DAW выбрать

### Первый взгляд и ощущение
Откройте демо-версию, создайте простой проект. Ощущения на первых минутах имеют значение.

### Визуальная составляющая
Интерфейс должен быть понятным и удобным. Вы будете проводить за программой сотни часов.

### Логика работы
Как программа организует проекты, дорожки, микшер, навигацию — всё должно быть логично для вас.

### Удобство и технические нюансы
Системные требования, стабильность, поддержка форматов, совместимость с вашим оборудованием.

## Необходимое оборудование

### WIN или MAC
Обе платформы подходят для музыкального продакшна. MAC часто предпочтительнее для студийной работы благодаря оптимизации.

### Наушники
Мониторные наушники с ровной частотной характеристикой — минимум для начала работы.

### Колонки — мониторы
Студийные мониторы дают более точное представление о звуке. Начните с одной пары.

### Дополнительное (необязательное)

- **Микрофон** — для записи вокала и живых инструментов
- **MIDI-клавиатура** — для игры мелодий и управления плагинами
- **Аудиокарта** — внешняя звуковая карта для качественного вывода звука

## Рабочее место

### Комната и рабочий стол
Выберите тихое место с минимальным фоновым шумом.

### Подбор места для мониторов
Мониторы должны стоять на уровне ушей, образуя равносторонний треугольник с вашим ухом.

### Расстановка оборудования
Организуйте пространство так, чтобы всё было под рукой и не создавало путаницы.

## Аудио драйверы

### ASIO
ASIO — стандарт аудио драйверов для минимальной задержки звука. На Windows используйте ASIO4All или драйвер вашей аудиокарты.

### Внешняя звуковая карта
Встроенная звуковая карта подходит для начала, но внешняя аудиокарта обеспечит лучшую задержку и качество звука.

## Установка программ и VST плагинов

### Не обязательно ставить всё и сразу
Начните с базового набора плагинов, которые идут в комплекте с вашей DAW.

### Ставить программы последовательно
Установите DAW, настройте её, затем постепенно добавляйте плагины.

### Наличие сэмпл-паков
Сэмплы — основа продакшна. Начните с бесплатных пакетов, затем расширяйте коллекцию.

### Материалы в обсуждениях к этапу
Проверяйте чат и дополнительные материалы — там есть рекомендации по конкретным плагинам и сэмпл-пакам.

## Оптимизация компа и ресурсов

### ЦП
Закройте ненужные программы перед запуском DAW. Музыкальный продакшн требует ресурсов процессора.

### Задержка звука (буфер)
Настройте буферный размер: 128-256 сэмплов для записи, 512-1024 для сведения.

## Программы для заметок

### Xmind
Отличный инструмент для создания ментальных карт и планирования проектов.

### Obsidian
Мощная программа для заметок с поддержкой связей между записями.

### Тетрадка
Иногда простая тетрадь работает лучше всего — записывайте идеи, BPM, аккорды.

Telegram и Discord чаты — основное место для вопросов и общения с участниками.

Домашние задания — ключевой элемент обучения. Выполняйте их в срок и получайте фидбек.
