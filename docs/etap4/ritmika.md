# Ритмика

Ритм — это **время** музыки. Даже самая красивая мелодия станет скучной без интересного ритмического рисунка.

## Ритм Мелодии

Ритм мелодии определяется не только тем, **какие** ноты вы играете, но и **когда** и **как долго**.

### Длительности нот

| Длительность | Доля такта (при 4/4) | Визуально в Piano Roll |
|-------------|---------------------|----------------------|
| Целая нота | 4 доли | 16 блоков |
| Половинная | 2 доли | 8 блоков |
| Четвертная | 1 доля | 4 блока |
| Восьмая | 0.5 доли | 2 блока |
| Шестнадцатая | 0.25 доли | 1 блок |

### Ритмические приёмы

- **Синкопа** — акцент на слабую долю (создает «кач» и грув)
- **Легато** — плавное соединение нот без пауз
- **Стаккато** — короткие, отрывистые ноты
- **Тирада** — серия быстрых нот подряд

## Кач (Groove/Swing)

**Кач** — это «человеческое» отклонение от идеального ритма, которое делает музыку живой.

### Как создать кач

| Метод | Описание |
|-------|----------|
| **Swing** | Чётные восьмые ноты задерживаются (50-70% swing) |
| **Humanize** | Случайное смещение timing и velocity |
| **Ghost notes** | Тихие «призрачные» ноты между основными ударами |
| **Micro-timing** | Ручное смещение нот на 1-5 мс |

!!! tip
    В FL Studio используйте **Groove Toolbar**. В Ableton — **Groove Pool**. Готовые пресеты swing добавят жизни в механический ритм.

## Паузы и свободное пространство

**Пауза** — это не пустота. Пауза — это инструмент, который:

- Создаёт **ожидание** перед важной нотой
- Даёт «передышку» слушателю
- Подчёркивает следующую ноту (контраст тишины и звука)

### Правило пространства

```
Мелодия без пауз = стена текста без абзацев
Мелодия с паузами = структурированный рассказ
```

!!! important
    **Меньше — больше.** Новички заполняют каждый такт нотами. Опытные продюсеры оставляют пустоту, чтобы каждая нота «дышала».

## Основные ошибки

| Ошибка | Почему плохо | Как исправить |
|--------|-------------|--------------|
| **Перегрузка нотами** | Мелодия теряет фокус, «каша» | Удалите 30% нот — оставьте только ключевые |
| **Однообразный ритм** | Мелодия «плоская», предсказуемая | Чередуйте длинные и короткие ноты |
| **Нет пауз** | Нет воздуха, утомляет слух | Вставьте паузы перед акцентами |
| **Игнорирование swing** | Механический, «роботизированный» звук | Добавьте 5-15% humanize или swing |
| **Все ноты одной громкости** | Нет динамики, нет эмоций | Варьируйте velocity (100-120 для акцентов, 60-80 для фона) |
