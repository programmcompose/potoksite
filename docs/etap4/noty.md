# Ноты

Ноты — это алфавит музыкального языка. В западной музыке используется **7 основных нот**, которые повторяются в разных октавах.

## 7 нот

| Нота (лат.) | Нота (рус.) | Частота (4-я октава, Гц) |
|------------|-------------|-------------------------|
| **C** | До (C) | 261.63 |
| **D** | Ре (D) | 293.66 |
| **E** | Ми (E) | 329.63 |
| **F** | Фа (F) | 349.23 |
| **G** | Соль (G) | 392.00 |
| **A** | Ля (A) | 440.00 |
| **B** | Си (B) | 493.88 |

!!! tip
    **A4 = 440 Гц** — это стандартная настройка. Нота Ля первой октавы звучит с частотой 440 герц, и все остальные ноты выстраиваются относительно неё.

## Диезы и Бемоли

Между соседними нотами есть промежуточные звуки — **полутон**. Они обозначаются знаками альтерации:

| Знак | Название | Действие | Пример |
|------|----------|---------|--------|
| **\#** | Диез | Повышает ноту на полутон (+0.5) | C\# — До-диез |
| **b** | Бемоль | Понижает ноту на полутон (-0.5) | Db — Ре-бемоль |

### Энгармонизм

Одна и та же нота может иметь два названия:
- **C\# = Db** (одна клавиша на пианино, два имени)
- **F\# = Gb**
- **B = Cb** (в определённом контексте)

## Тон и Полутон

Расстояние между нотами измеряется в **полутонах**:

- **Полутон** — минимальное расстояние (соседние клавиши на пианино)
- **Тон** — два полутона (между C и D есть C\#, значит расстояние = 1 тон)

### Расстояния в натуральном звукоряде

```
C — D : 1 тон
D — E : 1 тон
E — F : 0.5 тона (полутон!)
F — G : 1 тон
G — A : 1 тон
A — B : 1 тон
B — C : 0.5 тона (полутон!)
```

Пример: как звучит гамма До мажор (C-D-E-F-G-A-B-C)

!!! important
    Полутона между **E—F** и **B—C** — это ключевой момент. Именно поэтому на пианино между некоторыми белыми клавишами нет чёрных.

## Как выглядит на пианино / Piano Roll

### Пианино

```
Белые клавиши: C  D  E  F  G  A  B  C  (повтор)
Чёрные клавиши: C# D#    F# G# A#     (между белыми, кроме E-F и B-C)

[ C ][C#][ D ][D#][ E ][ F ][F#][ G ][G#][ A ][A#][ B ][ C ]
  ↑ чёрные клавиши повышены на полутон относительно левой белой
```

### Piano Roll в DAW

В пиано-ролле ноты отображаются как блоки на сетке:

- **По вертикали** — высота ноты (чем выше, тем выше звук)
- **По горизонтали** — время и длительность ноты
- **Каждая строка** — одна полутоновая ступень

!!! tip
    В FL Studio и Ableton включайте **Scale Highlight** — это подсветит ноты выбранной гаммы и упростит работу с пиано-роллом.
