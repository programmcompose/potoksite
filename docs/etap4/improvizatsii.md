# Импровизации

Импровизация — это **практика** теории. Всё, что вы изучили о нотах, интервалах и аккордах, обретает смысл, когда вы начинаете импровизировать.

## Мелодии

### Как импровизировать мелодию

1. **Выберите тональность** — например, A minor
2. **Играйте только ноты гаммы** — это гарантирует гармонию
3. **Начните с тоники** — A в нашем случае
4. **Двигайтесь постепенно** — соседние ноты звучат плавнее
5. **Возвращайтесь к тонике** — это создаёт ощущение завершенности

### Упражнение: 4-нотный мотив

```
Шаг 1: Придумайте мотив из 4 нот в тональности
Шаг 2: Повторите его, сместив на октаву вверх
Шаг 3: Измените одну ноту в повторении
Шаг 4: Добавьте паузу перед третьим повторением
```

!!! tip
    Ограничьте себя: используйте только 4-5 нот из гаммы. Ограничение рождает креативность.

## Тренировка аккордов

### Упражнение: прогрессия за 5 минут

1. Откройте пиано-ролл
2. Включите Scale Highlight (A minor)
3. Поставьте 4 аккорда: Am → C → G → F
4. Попробуйте заменить один аккорд на септаккорд
5. Поиграйте с порядком аккордов

### Упражнение: инверсии

Возьмите аккорд Am (A-C-E) и перерасставьте ноты:

```
Root:     A (бас) — C — E
1-я инв:  C (бас) — E — A
2-я инв:  E (бас) — A — C
```

Слушайте, как меняется ощущение от каждой расстановки.

## Запись

### Почему важно записывать

- **Идеи улетучиваются** — за 5 минут вы забудете мелодию
- **Запись = материал** — каждый скетч может стать основой бита
- **Прогресс виден** — сравнивая старые и новые записи, вы видите рост

### Процесс записи импровизации

1. **Включите запись** перед тем, как начать играть
2. **Импровизируйте 2-3 минуты** без остановки
3. **Остановитесь** — не редактируйте в моменте
4. **Прослушайте** — выделите лучший фрагмент (4-8 тактов)
5. **Сохраните** — назовите файл с датой и тональностью

!!! note
    Не стремитесь к совершенству при записи. Первая версия — это «сырой материал». Редактирование ишлифование придёт позже.

## MIDI паттерны

**MIDI паттерн** — готовый шаблон мелодии или аккордовой последовательности.

### Как использовать паттерны

| Подход | Описание |
|--------|----------|
| **Изучение** | Разберите паттерн ноту за нотой — поймите логику |
| **Модификация** | Измените 20% нот — сделайте паттерн своим |
| **Комбинация** | Соедините мелодию из одного паттерна с аккордами из другого |
| **Ремикс** | Замените тембр, темп, ритмику — узнайте, что работает |

### Где брать MIDI паттерны

- Встроенные библиотеки DAW (FL Studio, Ableton)
- Бесплатные паки в Telegram/Discord сообществах
- Splice MIDI
- Собственные накопленные паттерны

!!! important
    Паттерны — это не чит, это **учебный материал**. Как художник копирует мастера, чтобы понять технику, так и вы изучаете чужие паттерны, чтобы понять гармонию.
