# Тембры звука

**Тембр** — это «окраска» звука. Одна и та же нота на пианино и на гитаре звучит по-разному — это различие и есть тембр. Выбор тембра определяет 50% восприятия мелодии.

## Яркость

**Яркость** — распределение энергии звука по частотам. Яркий звук — много высоких гармоник, тёмный — больше низких.

| Яркость | Характер | Где использовать |
|---------|----------|-----------------|
| **Яркий (Bright)** | Острый, пронзительный, «блистающий» | Лиды, хуки, акцентные звуки |
| **Средний (Mid)** | Сбалансированный, естественный | Пэды, аккордовые инструменты |
| **Тёмный (Dark)** | Мягкий, тёплый, «бархатный» | Фоновые слои, басовые инструменты |

### Управление яркостью

- **EQ** — срежьте высокие частоты для темнения, поднимите для осветления
- **Filter** — low-pass filter затемняет, high-pass осветляет
- **Oscillator waveform** — sawtooth яркий, sine тёмный, square средний
- **Saturation** — добавляет гармоники, делает звук ярче

## Стилевое сочетание

Тембр должен соответствовать **стилю трека**:

| Стиль | Подходящие тембры | Примеры инструментов |
|-------|-------------------|---------------------|
| **Trap** | Тёмные пэды, яркие лиды, тяжёлый 808 | DX7, Omnisphere, 808-бас |
| **Boom Bap** | Винтажные, «пыльные», тёплые | Сэмплы пианино, фанк-гитара |
| **R&B** | Мягкие, «воздушные», гладкие | Rhodes, струнные, вокал |
| **Фонк** | Агрессивные, искажённые, «грязные» | Distorted 808, detuned синты |
| **Lo-fi** | Приглушённые, «кассетные», тёплые | Lo-fi пианино, виниловый шум |

!!! tip
    Не ставьте яркий лид поверх яркого пэда — они будут «сражаться» в одних и тех же частотах. Сочетайте яркий + тёмный для контраста.

## Читаемость в миксе

**Читаемость** — насколько легко слух различает инструмент в общем миксе.

### Факторы читаемости

| Фактор | Влияние |
|--------|---------|
| **Частотное место** | Каждый инструмент занимает свою «полку» |
| **Динамический диапазон** | Контраст громкости выделяет инструмент |
| **Пространственное позиционирование** | Панорама и реверб создают глубину |
| **Ритмическое разделение** | Инструменты играют в разное время |

### Как улучшить читаемость

1. **EQ carve** — вырежьте частоты, где «живёт» другой инструмент
2. **Sidechain** — уступайте место главному инструменту
3. **Слои по регистрам** — низкий, средний, высокий — каждый на своём уровне
4. **Меньше инструментов** — 3 хорошо подобранных звука лучше 10 перекрывающихся

## Сочетания с другими инструментами

### Классические пары

| Комбинация | Почему работает |
|------------|----------------|
| **Пианино + Струнные** | Контраст атаки (резкое + плавное) |
| **Синтезатор + Вокал** | Электронное + органическое |
| **Гитара + Пэды** | Точечный + растянутый звук |
| **808 + Акустический бас** | Гибрид цифрового и аналогового |

!!! note
    Тембр — это не только выбор инструмента, но и его **обработка**. Один и тот же синтезатор может звучать как мягкий пэд или как агрессивный лид — всё зависит от настроек.
