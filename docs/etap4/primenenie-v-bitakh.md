# Применение в битах и треках

Теория обретает смысл, когда вы применяете её в реальных битах. Разберём, как разные аккордовые подходы работают в конкретных стилях.

## Красивые аккорды

«Красота» — субъективна, но есть прогрессии, проверенные десятилетиями:

| Прогрессия | Тональность | Ощущение |
|-----------|------------|---------|
| **I — V — vi — IV** | Мажор | Эпичное, «киношное» |
| **i — vii° — IV — V** | Минор | Драматичное, напряжённое |
| **I — IV — V — vi** | Мажор | Классическое, «гимновое» |
| **i — III — iv — VI** | Минор | Мечтательное, «облачное» |

### Секрет «красоты»

Красивые аккорды строятся на:
- **Плавных переходах** (общие ноты между аккордами)
- **Контрасте** (мажор ↔ минор, свет ↔ тень)
- **Ожидании и разрешении** (напряжение → покой)

## Поп и минималистик

### Поп-музыка

- **Мажорные прогрессии** — позитив, радиоформат
- **Простые триады** — без перегрузки расширениями
- **Повторяющиеся паттерны** — 4-аккордовый цикл на весь трек

**Пример поп-прогрессии:**
```
C → G → Am → F  (повтор × 4)
```

### Минимализм

- **2-3 аккорда** на весь трек
- **Пол на минимуме** — аккорды как текстура, а не фокус
- **Пространство** — пустота важнее заполненности

**Пример минималистичной прогрессии:**
```
Am → F  (повтор × 8, с вариациями тембра)
```

!!! tip
    Минимализм — это не «меньше делать», а «делать меньше, но качественнее». Каждый аккорд в минимализме несёт максимум смысла.

## Криповые сочетания

Для мрачных, зловещих, «неудобных» битов:

| Приём | Как сделать | Пример |
|-------|-----------|--------|
| **Тритон** | Интервал 6 полутонов | C + F\# в одном аккорде |
| **Уменьшённые аккорды** | Терция + уменьшённая квинта | Cdim = C + Eb + Gb |
| **Хроматическое движение** | Корень идёт по полутонам | C → C\# → D → D\# |
| **Обратный бас** | Бас движется вниз, аккорды вверх | Am (бас A) → Am/G (бас G) → Am/F (бас F) |
| **Кластеры** | Ноты через полутон подряд | C + C\# + D |

### Где используется

- **Дрил-биты** — тритоны, уменьшённые аккорды
- **Хоррор-саундтреки** — кластеры, хроматизм
- **Фонк** — диссонирующие басовые линии

## Эмоциональные

Эмоциональные прогрессии играют на **контрасте** и **неожиданности**:

### Меланхолия

```
Am → F → C → G  (минор → мажор и обратно)
```

Контраст между минорным началом и мажорным «окном» создаёт ощущение надежды в грусти.

### Ностальгия

```
F → G → Em → Am  (обратное движение к минору)
```

Начало с мажора и «падение» в минор — как воспоминание, которое тает.

### Тревога

```
Em → E → Em → E  (параллельный минор/мажор)
```

Чередование минорного и мажорного аккордов с одним корнем — эффект «нервности».

!!! important
    Эмоция в музыке рождается не из отдельных аккордов, а из их **последовательности** и **контекста**. Один и тот же аккорд может звучать по-разному в зависимости от того, что было до него и что придёт после.
