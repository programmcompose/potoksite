# Применение в битах / треках

Теория обретает смысл, когда вы применяете её в реальных продакшенах. Разберём практические сценарии.

## Красивые аккорды

### Формула красивого звучания

1. Выберите **минорную тональность** (Am, Cm, Em)
2. Постройте **7-аккорды** (maj7, min7) вместо простых триад
3. Добавьте **9-ю ступень** для воздушности
4. Используйте **перевёрнутые аккорды** для плавности

### Пример прогрессии

```
Тональность: A minor

Am9  →  Fmaj7  →  Cadd9  →  G7
(A C E G D) (F A C E) (C E G A) (G B D F)
```

!!! tip
    Секрет «дорогого» звучания — **простота + качество тембра**. 4 аккорда с хорошим саундом лучше 16 аккордов с плохим.

## Поп и минималистик

### Поп-формула

```
I – V – vi – IV  (C – G – Am – F)
```
- Универсальна, работает в любой тональности
- Миллионы хитов построены на этой прогрессии

### Минималистичный подход

- **2-3 аккорда** на весь трек
- Акцент на **ритм и тембр**, а не на сложность прогрессии
- **Повторение** с небольшими вариациями

```
Пример минимализма:
Am → F (×4 повторения, меняя тембр каждый раз)
```

## Криповые сочетания

### Приёмы для мрачного звучания

| Приём | Описание | Пример |
|---|---|---|
| **Тритон** | Интервал 6 полутонов | C → F# |
| **Уменьшённые аккорды** | Нестабильность | Cdim: C Eb Gb |
| **Хроматизм** | Движение по полутонам | C → C# → D |
| **Фригийский лад** | b2 ступень создаёт тёмный колорит | A Bb C D E F G |
| **Диссонанс** | Конфликтующие ноты | Maj7 + b9 |

### Криповая прогрессия

```
Cm → Cdim → Fm → Bb7
(мрачно → напряжённо → грустно → зловеще)
```

!!! warning
    Криповость — в дозе. Слишком много диссонанса = хаос. Используйте точечно.

## Эмоциональные

### Карта эмоций и прогрессий

| Эмоция | Прогрессия | Тональность |
|---|---|---|
| **Грусть** | i – VI – III – VII | минор |
| **Надежда** | i – iii – IV – V | минор → мажор |
| **Ностальгия** | I – vi – IV – V | мажор |
| **Тревога** | i – bII – bIII – i | минор + хроматизм |
| **Спокойствие** | I – IV – V – I | мажор, медленно |
| **Стыд/Вина** | i – iv – VII – III | минор, диссонанс |

### Эмоциональный переход

```
[Грусть]  Am → F → C → G
[Надежда] Am → C → F → G → C (переход в мажор)
```

!!! note
    Смена тональности в середине трека (минор → мажор) создаёт эффект «просветления» — мощный эмоциональный приём.
