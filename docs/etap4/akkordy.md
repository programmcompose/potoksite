# Аккорды

**Аккорд** — это три или более нот, звучащих одновременно и образующих единое гармоническое целое. Аккорды — это «слова» музыкального языка.

## Логика построения аккордов

Аккорды строятся методом **терцового наложения** — берём ноты через одну ступень гаммы:

### Пример: C Major аккорд

```
Гамма C Major:  C   D   E   F   G   A   B
Ступени:         I  II  III  IV  V   VI  VII

Аккорд C: берём I + III + V = C + E + G
```

### Мажорный vs Минорный аккорд

| Тип | Формула | Пример (C) | Интервалы от корня |
|-----|---------|-----------|-------------------|
| **Мажор (maj)** | корень + большая терция + квинта | C + E + G | +4pt, +7pt |
| **Минор (min)** | корень + малая терция + квинта | C + Eb + G | +3pt, +7pt |

!!! tip
    Квинта (+7 полутонов) одинакова в обоих типах. Разница только в **терции**: 4 полутона = мажор, 3 полутона = минор.

## Больше 3-х нот

Аккорд из трёх нот — это база. Добавление нот обогащает звучание:

| Аккорд | Состав | Добавлена нота | Ощущение |
|--------|--------|---------------|----------|
| **Cmaj7** | C + E + G + B | 7-я ступень | Мягкое, мечтательное, джазовое |
| **Cadd9** | C + E + G + D | 9-я ступень | Открытое, воздушное |
| **Cmin7** | C + Eb + G + Bb | минорная 7-я | Блюзовое, глубокое |
| **Cdim** | C + Eb + Gb | уменьшённая квинта | Напряжённое, переходное |
| **Caug** | C + E + G\# | увеличенная квинта | Таинственное, неустойчивое |

### Когда добавлять ноты

- **В поп-музыке** — maj7 и add9 для «красоты»
- **В хип-хопе** — min7 для глубины бита
- **В фонке** — 9-я и 11-я для «широты» звука
- **В дрилле** — минорные септаккорды для мрачности

## Бас в аккорде

**Басовая нота** — самая низкая нота в аккорде. Она определяет «фундамент» гармонии.

### Корневой бас vs Не-корневой бас

| Тип | Описание | Пример |
|-----|----------|--------|
| **Root position** | Бас = корень аккорда | C / C (нота C внизу) |
| **Инверсия** | Бас ≠ корень аккорда | C / E (нота E внизу, аккорд всё ещё C) |

!!! important
    Смена басовой ноты **меняет ощущение** аккорда, даже если состав нот тот же. Это мощный инструмент для создания плавных переходов.

## Как расставлять ноты

### Октавное распределение

```
Высокий регистр:     E (верхняя терция) — яркость, мелодичность
Средний регистр:     C + G (корень + квинта) — тело аккорда
Низкий регистр:      C (корень в басах) — фундамент
```

### Принципы расстановки

1. **Не дублируйте корень** — если бас уже играет корень, в аккорде можно его убрать
2. **Терция наверху** — терция определяет цвет аккорда, она должна быть слышна
3. **Избегайте «густоты» в низких частотах** — ниже 200 Гц оставляйте только бас
4. **Space is sound** — пустота между нотами так же важна, как и сами ноты

## Гармонический «пол» трека

Аккорды создают **гармонический фон** — основу, на которой строится весь трек.

- **Аккордовая прогрессия** повторяется циклически
- **Мелодия** «живёт» внутри аккордов (берёт из них ноты)
- **Бас** следует за корнями аккордов или играет контрмелодию
- **Драмка** задаёт ритмический каркас поверх гармонии

!!! note
    В битмейкинге аккорды — это «пол», на котором «ходят» мелодия, бас и барабаны. Если пол кривой (диссонирующие аккорды без цели), весь трек будет звучать хаотично.
