# Тональность

**Тональность** — это система нот, объединённых общим центром (тоникой). Она определяет, какие ноты «свои», а какие «чужие» в рамках трека.

## Что такое тональность

Представьте, что все 12 нот — это доступные цвета. Тональность — это выбор **палитры**: вы берёте не все цвета, а только те, что хорошо сочетаются между собой.

- **Тоника** — главная нота, «домой», к которому всё возвращается
- **Ступени** — положение каждой ноты относительно тоники (1-я, 2-я, 3-я...)
- **Гамма (Scale)** — набор нот, входящих в тональность

## Виды тональностей

### Мажор

Мажорная тональность — светлая, открытая, позитивная.

**Формула мажорной гаммы (в тонах):**
```
Тон — Тон — Полутон — Тон — Тон — Тон — Полутон
```

**Пример: C Major (До мажор)**
```
C — D — E — F — G — A — B — C
```

Все белые клавиши пианино от C до C — это мажорная гамма без единого диеза и бемоля.

### Минор

Минорная тональность — тёмная, меланхоличная, эмоциональная.

**Формула натурального минора:**
```
Тон — Полутон — Тон — Тон — Полутон — Тон — Тон
```

**Пример: A Minor (Ля минор)**
```
A — B — C — D — E — F — G — A
```

!!! tip
    **A minor** — относительный минор к **C major**. Они используют одни и те же ноты, но тоника разная. Это называется «родственные тональности».

### Лады

Помимо мажора и минора существуют **лады** (modes) — варианты расстановки полутонов в гамме. Каждый лад получается из мажорной (ийонийской) гаммы **одним изменением**: повышением или понижением определённой ступени.

Все примеры ниже — относительно **C Major** (C D E F G A B C).

#### Ионийский (Ionian) — чистый мажор

```
C — D — E — F — G — A — B — C
```

Формула: `Т — Т — п — Т — Т — Т — п`

Это исходная мажорная гамма. Все ступени «натуральные» — без изменений.

#### Дорийский (Dorian) — минорный лад с мажорной шестой

```
C — D — E♭ — F — G — A — B♭ — C
```

Отличие от мажора: ** lowered 3-я ступень (E→E♭), lowered 7-я ступень (B→B♭)**.
Минорный характер (из-за lowered 3-й), но **мажорная 6-я ступень (A)** даёт более светлый оттенок, чем чистый минор.

#### Миксолидийский (Mixolydian) — мажор с lowered 7-й

```
C — D — E — F — G — A — B♭ — C
```

Отличие от мажора: ** lowered 7-я ступень (B→B♭)**.
Мажорный характер (3-я ступень на месте), но lowered 7-я снимает сильную тягу к тонике — звучит более расслабленно, «грубовато».

#### Лидийский (Lydian) — мажор с raised 4-й

```
C — D — E — F♯ — G — A — B — C
```

Отличие от мажора: ** raised 4-я ступень (F→F♯)**.
Мажорный характер, но raised 4-я создаёт ощущение «поднятия», «космичности» — интервал между 4-й и 5-й становится увеликванным.

#### Эолийский (Aeolian) — чистый минор

```
C — D — E♭ — F — G — A♭ — B♭ — C
```

Отличие от мажора: ** lowered 3-я (E→E♭), lowered 6-я (A→A♭), lowered 7-я (B→B♭)**.
Тёмный, меланхоличный характер — натуральный минор.

#### Фригийский (Phrygian) — минор с lowered 2-й

```
C — D♭ — E♭ — F — G — A♭ — B♭ — C
```

Отличие от мажора: ** lowered 2-я (D→D♭), lowered 3-я (E→E♭), lowered 6-я (A→A♭), lowered 7-я (B→B♭)**.
 lowered 2-я создаёт полутоновый интервал в начале (C→D♭) — характерный тёмный, экзотический, напряжённый звук.

| Лад | Характер | Изменение относительно мажора | Где используется |
|-----|----------|------------------------------|-----------------|
| **Ионийский (Ionian)** | Чистый мажор, светлый | Без изменений — эталон | Всё |
| **Дорийский (Dorian)** | Минорный, но с «поднятым» цветом | lowered 3-я, lowered 7-я | Фанк, R&B, хип-хоп |
| **Миксолидийский (Mixolydian)** | Мажорный с «грубоватым» оттенком | lowered 7-я | Рок, блюз, трэп |
| **Лидийский (Lydian)** | Мечтательный, «космический» | raised 4-я | Эмбиент, саундтреки |
| **Эолийский (Aeolian)** | Чистый минор | lowered 3-я, lowered 6-я, lowered 7-я | Популярная музыка |
| **Фригийский (Phrygian)** | Тёмный, экзотический | lowered 2-я, lowered 3-я, lowered 6-я, lowered 7-я | Дрил, фонк, метал |

## Устойчивые и неустойчивые ступени

В любой тональности ноты делятся на две группы:

### Устойчивые ступени (Тоника)

| Ступень | Название | Роль |
|---------|----------|------|
| **I (1-я)** | Тоника | Центр, «домой», покой |
| **III (3-я)** | Медианта | Определяет мажор/минор |
| **V (5-я)** | Доминанта | Прочная опора |

### Неустойчивые ступени

| Ступень | Название | Роль |
|---------|----------|------|
| **II (2-я)** | Супертоника | Переход, движение |
| **IV (4-я)** | Субдоминанта | Контраст, расширение |
| **VI (6-я)** | Субмедианта | Эмоциональный оттенок |
| **VII (7-я)** | Ведущий тон | Тяга к тонике, напряжение |

!!! important
    Музыка движется за счёт **напряжения и разрешения**: неустойчивые ступени создают напряжение, устойчивые — снимают его. Это основа всех аккордовых прогрессий.

## Эмоциональность между нот

Каждая ступень несёт свою «эмоциональную нагрузку»:

```
I    — покой, устойчивость, «я дома»
II   — лёгкое напряжение, движение вперёд
III  — эмоциональный цвет (мажор = радость, минор = грусть)
IV   — расширение, «окно открылось»
V    — максимальное напряжение, ожидание возвращения
VI   — контраст, смена настроения
VII  — сильная тяга к разрешению, «почти дома»
```

Понимание эмоциональности ступеней позволяет **конструировать настроение** трека через выбор аккордов.
