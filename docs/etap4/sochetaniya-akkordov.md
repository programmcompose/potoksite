# Сочетания аккордов

Один аккорд — это слово. Сочетание аккордов — это предложение. **Аккордовая прогрессия** — последовательность аккордов, создающая движение и эмоцию.

## Мажорные и минорные аккорды

### Мажорный аккорд (Major)

Формула: **корень + большая терция + чистая квинта**

Пример C Major (Cmaj или просто C):
```
C + E + G
```
Звучание: яркое, радостное, стабильное

<audio class="audio-player" controls preload="none" src="../../assets/audio/chord-c-major.ogg"></audio>

### Минорный аккорд (Minor)

Формула: **корень + малая терция + чистая квинта**

Пример C Minor (Cm):
```
C + D# + G
```
Звучание: мрачное, грустное, напряжённое

<audio class="audio-player" controls preload="none" src="../../assets/audio/chord-c-minor.ogg"></audio>

!!! tip
    Разница между мажором и минором — всего в **одном полутоне** (терция).
    Это самый мощный инструмент в гармонии.

## Как ставить аккорды друг за другом

###基本原则 плавного перехода

1. **Общие ноты** — если два аккорда имеют общую ноту, переход между ними плавный
2. **Постепенное движение** — корневые ноты идут по соседним ступеням
3. **Напряжение → Разрешение** — неустойчивые аккорды ведут к устойчивым

### MIDI Паки

Готовые MIDI-файлы с прогрессиями — отличный способ учиться:

- Анализируйте, **почему** прогрессия звучит хорошо
- Меняйте **одну ноту** и слушайте разницу
- Меняйте **тембр** — одни и те же аккорды звучат по-разному на разных инструментах

### I – V – vi – IV

Самая популярная прогрессия в поп-музыке:

```
C → G → Am → F
```

<audio class="audio-player" controls preload="none" src="../../assets/audio/progression-iv-vi-iv.ogg"></audio>

Используется в: **Let It Be** (The Beatles), **With Or Without You** (U2), **Don't Worry Be Happy** (Lionel Richie)

### vi – IV – I – V

```
Am → F → C → G
```

<audio class="audio-player" controls preload="none" src="../../assets/audio/progression-vi-iv-i-v.ogg"></audio>

Используется в: **Someone Like You** (Adele), **Stay With Me** (Sam Smith)

### I – vi – IV – V

"50s progression" — классика рока и джаза:

```
C → Am → F → G
```

<audio class="audio-player" controls preload="none" src="../../assets/audio/progression-50s.ogg"></audio>

### Примеры из треков по жанрам

| Трек / Жанр | Прогрессия | Почему работает |
|-------------|-----------|----------------|
| Pop ballad | I — V — vi — IV | Классика, универсальность |
| Hip-Hop | i — bIII — VII | Мрачность, минимум нот |
| R&B | i — iii — VI — VII | Плавность, общие ноты |
| Trap | i — III — v | Контраст мажор/минор |

## Самые красивые переходы

### Таблица аккордов в C Major

| Ступень | Аккорд | Тип |
|---------|--------|-----|
| I | C (C-E-G) | мажор |
| ii | Dm (D-F-A) | минор |
| iii | Em (E-G-B) | минор |
| IV | F (F-A-C) | мажор |
| V | G (G-B-D) | мажор |
| vi | Am (A-C-E) | минор |
| vii° | Bdim (B-D-F) | уменьшенный |

### I — vi — iii — V

```
C  →  Am  →  Em  →  G
```

Классическая «дядя Бобби» прогрессия в обратном порядке. Каждый аккорд делит общие ноты с предыдущим, создавая **каскадный эффект**.

### i — iii — vi

```
Am  →  Cm  →  Fm
```

Мрачная, кинематографичная прогрессия. Все аккорды минорные, корневые ноты движутся по терциям.

### i — iii — iv — v

```
Am  →  Cm  →  Dm  →  Em
```

Полный минорный круг — последовательное прохождение всех минорных ступеней. Звучит как «путь» или «путешествие».

!!! tip
    Эти прогрессии работают в **любой тональности**. Просто сдвиньте все аккорды на нужное количество полутонов.

## Добавочные ступени

### 7-я и 9-я

| Степень | Полутоны от корня | Пример (от C) | Эффект |
|---------|-----------------|--------------|--------|
| **7-я** | 10-11 pt | Bb или B | Добавляет «глубину» и «воздух» |
| **9-я** | 14-15 pt | D или D\# | Расширяет аккорд, делает его «шире» |

### Зачем нужны

- **7-я** убирает «плоскость» триады, делает аккорд объёмнее
- **9-я** добавляет мелодичность внутри самого аккорда
- Вместе они создают **современное, «дорогое» звучание**

### Частые ошибки

| Ошибка | Проблема | Решение |
|--------|----------|---------|
| 7-я конфликтует с вокалом | Диссонанс намеренно неприятный | Проверьте, какие ноты использует мелодия |
| Слишком много расширений | Аккорд «перегружен» | Оставьте 1-2 расширения на аккорд |
| 9-я в низком регистре | Мутит низкие частоты | 9-ю ставьте выше корня |

!!! important
    Расширения (7, 9, 11, 13) — это специи. Немного — вкусно, слишком много — пересол.
