# Использование Нейросетей для написания музыки

Нейросети — мощный инструмент **вдохновения и обучения**. Они не заменят продюсера, но ускорят процесс.

## Что могут нейросети

### Генерация аккордовых прогрессий

| Инструмент | Что делает |
|---|---|
| **Suno AI** | Генерирует полноценные треки по описанию |
| **Udio** | Создаёт музыку из текстового промпта |
| **AIVA** | Композиторская нейросеть для классики и саундтреков |
| **Chordbot** | Генерация аккордовых прогрессий |
| **Mubert** | Бесконечная генерация по жанру/настроению |

### Анализ и советы

- **Определение тональности** — загрузите аудио, AI определит ключ
- **Предложение аккордов** — введите тональность, получите прогрессии
- **Мелодические идеи** — сгенерируйте мотив, затем доработайте

## Как использовать правильно

### ✅ Правильно

- **Источник идей** — используйте сгенерированные прогрессии как отправную точку
- **Обучение** — анализируйте, какие аккорды предлагает AI и почему
- **Быстрый прототип** — создайте набросок за 5 минут, затем доработайте
- **Преодоление writer's block** — когда нет идей, пусть AI предложит варианты

### ❌ Неправильно

- **Слепое копирование** — не используйте сгенерированное «как есть»
- **Замена обучения** — AI не заменит изучение теории
- **Игнорирование контекста** — AI не знает, что «звучит хорошо» в вашем жанре

## Практический workflow с AI

```
1. Опишите идею нейросети
   "Мрачная trap-прогрессия в C minor, BPM 140"

2. Получите 3-5 вариантов прогрессий

3. Выберите лучший вариант

4. Доработайте в DAW:
   - Измените 2-3 ноты
   - Поменяйте тембр
   - Добавьте свои мелодии

5. Результат — ваш трек с AI-вдохновением
```

!!! tip
    Используйте AI как **коллегу**, а не как исполнителя. Спрашивайте идеи, но принимайте решения сами.

## Советы по промптам

### Хороший промпт

```
"Аккордовая прогрессия для trap-бита в A minor, 
4 аккорда, с 7-ми и 9-ми ступенями, 
мрачное настроение, BPM 140"
```

### Плохой промпт

```
"Сделай бит"
```

### Шаблоны промптов

| Задача | Промпт |
|---|---|
| Прогрессия | "[Тональность] прогрессия для [жанра], [кол-во] аккордов, [настроение]" |
| Мелодия | "Мелодический мотив в [тональности], [инструмент], [ритм]" |
| Анализ | "Определи тональность и аккорды этого трека" |
| Вариация | "Вариация этой прогрессии: [аккорды], сделай её [характеристика]" |

!!! note
    Нейросети развиваются быстро. Экспериментируйте с разными инструментами и находите свой workflow.
