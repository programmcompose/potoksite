# Интервалы

**Интервал** — это расстояние между двумя нотами, измеряемое в полутонах. Интервалы — это «атомы» гармонии: из них строятся аккорды, мелодии и вся музыкальная структура.

## Как измеряются интервалы

Интервал определяется от **первой ноты к второй**:

- **По полутонам** — точное количественное расстояние
- **По ступеням** — качественное название (терция, квинта и т.д.)

## Самые важные интервалы

### Минорная терция (minor 3rd)

| Параметр | Значение |
|----------|---------|
| **Полутоны** | 3 полутона |
| **Пример** | C → Eb |
| **Ощущение** | Тёмное, грустное, напряжённое |
| **Где встречается** | Минорные аккорды, блюз, фанк |

### Мажорная терция (major 3rd)

| Параметр | Значение |
|----------|---------|
| **Полутоны** | 4 полутона |
| **Пример** | C → E |
| **Ощущение** | Яркое, радостное, открытое |
| **Где встречается** | Мажорные аккорды, поп, хип-хоп |

!!! important
    **Разница между мажором и минором** — всего в одном полутонах! Терция определяет «цвет» аккорда: мажорная = светлый, минорная = тёмный.

### Квинта (5th)

| Параметр | Значение |
|----------|---------|
| **Полутоны** | 7 полутонов |
| **Пример** | C → G |
| **Ощущение** | Прочное, устойчивое, «пустое» |
| **Где встречается** | Power-аккорды (рок), базовая гармония |

Квинта создаёт ощущение «каркаса» — она не говорит, мажор это или минор, но даёт фундамент.

### Октава (Octave)

| Параметр | Значение |
|----------|---------|
| **Полутоны** | 12 полутонов |
| **Пример** | C4 → C5 |
| **Ощущение** | Та же нота, но выше (ниже) |
| **Где встречается** | Вокал, арпеджио, расширение мелодий |

Октава — это «повтор» ноты на другом уровне. Ноты в октаве имеют одинаковое имя и идеально сочетаются.

## Таблица интервалов

| Интервал | Полутоны | Пример (от C) | Характер |
|----------|---------|--------------|----------|
| Унисон | 0 | C → C | Совпадение |
| Малая секунда | 1 | C → C\# | Натяжение, трение |
| Большая секунда | 2 | C → D | Мягкое движение |
| Малая терция | 3 | C → Eb | Тёмное, минорное |
| Большая терция | 4 | C → E | Светлое, мажорное |
| Кварта | 5 | C → F | Открытое, неустойчивое |
| Тритон | 6 | C → F\# | Диссонирующее, «дьявольское» |
| Квинта | 7 | C → G | Прочное, чистое |
| Малая септима | 10 | C → Bb | Блюзовое |
| Большая септима | 11 | C → B | Напряжённое, джазовое |
| Октава | 12 | C → C | Повтор, расширение |

!!! tip
    Для начала запомните **терции** (3 и 4 полутона) и **квинту** (7 полутонов) — на этих трёх интервалах строится 90% аккордов в современной музыке.
