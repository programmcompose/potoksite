# Использование Нейросетей для написания музыки

Нейросети — не замена продюсера, а **инструмент** для ускорения рабочего процесса и генерации идей.

## Что умеют нейросети

| Задача | Что может ИИ | Что ИИ не умеет |
|--------|-------------|----------------|
| **Аккордовые прогрессии** | Генерировать бесконечные варианты | Понимать контекст и намерение |
| **Мелодии** | Создавать мелодические линии | Чувствовать эмоцию «из души» |
| **Тембры** | Предлагать сочетания звуков | Знать ваш уникальный стиль |
| **Советы** | Давать теоретические объяснения | Заменить опыт и слух |

## Аккорды от нейросетей

### Как запросить прогрессию

```
Пример промпта:
"Предложи минорную аккордовую прогрессию для мрачного
хип-хоп бита в тональности F minor. Используй септаккорды."
```

### Что делать с результатами

1. **Проверьте теорию** — убедитесь, что аккорды логичны
2. **Поэкспериментируйте** — измените 1-2 аккорда на свои
3. **Поменяйте инверсии** — ИИ редко предлагает интересные инверсии
4. **Добавьте расширения** — 7, 9, 11 по вашему вкусу

!!! tip
    Используйте нейросети для **генерации стартовых идей**, а не финальных решений. ИИ даёт направление, вы — доводите до совершенства.

## Советы от нейросетей

### Теоретические вопросы

Нейросети отлично отвечают на вопросы вроде:

- «Какие аккорды совместимы с Am7?»
- «Как сделать прогрессию мрачнее?»
- «Какой лад использовать для фанк-бита?»
- «Почему тритон звучит напряжённо?»

### Анализ вашего бита

Опишите свой трек нейросети:

```
"У меня бит в A minor, прогрессия Am → F → C → G.
Мелодия использует ноты A, C, E. Как усложнить
гармонию, не потеряв простоту?"
```

## Ограничения и предостережения

| Ограничение | Как работать с ним |
|------------|-------------------|
| **ИИ не слышит** | Проверяйте все предложения на слух |
| **Шаблонные ответы** | Спрашивайте конкретнее, уточняйте контекст |
| **Ошибки в теории** | Критически проверяйте информацию |
| **Нет «вкуса»** | Финальный выбор всегда за вами |

!!! important
    Нейросеть — это **ассистент**, а не автор. Используйте её для генерации идей, проверки теории и поиска вдохновения. Но финальное решение о том, что звучит хорошо — всегда за вами.
