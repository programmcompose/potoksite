# Музыка для фильма

Фильмовая музыка — это служение истории. Она не должна перетягивать внимание, но без неё сцена теряет 50% эмоциональной силы.

## Функциональная музыка

### Музыка не тянет на себя

**Золотое правило:** зритель не должен замечать музыку как отдельный элемент. Она должна быть неотделима от картинки.

| Принцип | Описание |
|---------|---------|
| **Подчинение** | Музыка служит сцене, а не наоборот |
| **Прозрачность** | Слушатель чувствует эмоцию, но не анализирует аккорды |
| **Своевременность** | Музыка появляется и уходит в нужный момент |
| **Интенсивность** | Громкость и плотность соответствуют сцене |

### Написание основной темы

**Подход к фильмовой теме:**

1. **Определите главную эмоцию фильма** — что зритель должен почувствовать?
2. **Создайте мотив** — 4-8 нот, которые определяют фильм
3. **Варируйте тему** — она должна работать в разных контекстах
4. **Повторяйте** — тема становится ассоциацией с персонажем или историей

### Примеры тем

| Фильм | Тема | Инструмент |
|-------|------|-----------|
| **Star Wars** | Heroic fanfare | Brass, полный оркестр |
| **Jaws** | 2 ноты, напряжение | Струнные, хроматика |
| **Schindler's List** | Трагедия | Соло скрипка |
| **Interstellar** | Эпик, космос | Орган, оркестр |

## Настроение

### Подбор под видео

**Анализ сцены:**

| Элемент сцены | Музыкальная реакция |
|-------------|-------------------|
| **Диалог** | Тихий фон, минимум инструментов |
| **Экшн** | Быстрый темп, перкуссия, диссонанс |
| **Романтика** | Лиричная тема, струнные, пиано |
| **Тревога** | Tremolo, хроматика, низкие регистры |
| **Триумф** | Полный оркестр, мажор, высокий регистр |

### Изменение атмосферы музыкой

**Приёмы смены настроения:**

- **Смена тональности** — мажор → минор (или наоборот)
- **Темповый сдвиг** — замедление или ускорение
- **Тембральный контраст** — смена инструментов
- **Динамический скачок** — тишина → громко

### Смена аккордов или тональности

**Модуляция в кино:**

| Приём | Эффект | Когда использовать |
|-------|--------|-------------------|
| **Родственная модуляция** | Плавный переход | Развитие сцены |
| **Хроматическая модуляция** | Напряжение | Конфликт, поворот |
| **Квартовое обращение** | Свежесть | Новая глава |
| **Возврат в тонику** | Разрешение | Финал сцены |

## Аранжировка

### Основная тема

**Вариации темы:**

| Вариант | Инструменты | Контекст |
|---------|-----------|---------|
| **Лёгкая** | Пиано соло | Интро, интимные сцены |
| **Средняя** | Струнные + пиано | Развитие |
| **Полная** | Полный оркестр | Кульминация |
| **Тёмная** | Минор, низкие регистры | Конфликт |

### Нарастание

**Техники нарастания:**

1. **Instrumental build-up** — постепенное добавление инструментов
2. **Dynamic crescendo** — увеличение громкости
3. **Harmonic tension** — диссонанс перед разрешением
4. **Rhythmic intensity** — увеличение плотности ритма

## Эмбиент и текстуры

### Роль эмбиента в кино

- **Фон для диалогов** — мягкие пэды, атмосфера
- **Переходы между сценами** — текстуры, звуковые мосты
- **Подводное напряжение** — низкочастотные дроны
- **Эмоциональная окраска** — цветовые текстуры

### Создание текстур

| Техника | Описание |
|---------|---------|
| **Pad layering** | Наложение нескольких пэдов |
| **Granular synthesis** | Зернистые текстуры |
| **Field recordings** | Полевые записи, атмосфера |
| **Reverb tails** | Длинный реверб, пространство |
| **Reverse effects** | Реверс-эффекты, сны |

!!! important
    **Главный принцип фильмовой музыки:** если музыка мешает диалогу — она слишком громкая. Если зритель анализирует ваши аккорды вместо того, чтобы переживать за персонажа — вы переборщили с аранжировкой.
