# Построение оркестрового ритма

Ритм в оркестре — это не только ударные. Каждый инструмент может нести ритмическую функцию.

## Остинато

### Что такое остинато

Остинато — повторяющийся ритмический или мелодический паттерн, создающий пульсацию и напряжение.

### Типы остинато

| Тип | Инструменты | Пример |
|-----|------------|--------|
| **Струнное** | Violin, Viola spiccato | «Jaws» тема |
| **Пиано** | Арпеджио, блоки | Эмбиент, фон |
| **Духовой** | Pulsing brass | Эпик, трейлер |
| **Перкуссия** | Timpani, snare | Марши, экшн |

### Создание остинато

1. **Выберите паттерн** — 2-4 такта
2. **Определите инструмент** — кто несёт ритм
3. **Зациклите** — повторение с вариациями
4. **Варируйте** — меняйте динамику, артикуляцию

## Акцентировка и пульсация

### Акценты

| Тип акцента | Обозначение | Использование |
|------------|-----------|--------------|
| **sfz** | Фортиссимный акцент | Удар, драматизм |
| **F** | Громко | Кульминация |
| **Accent mark** | > | Ритмический акцент |
| **Unaccented** | — | Фон, заполнение |

### Пульсация

**Уровни пульсации:**

| Уровень | Длительность | Инструменты |
|---------|-------------|-------------|
| **Медленный** | 1 удар / такт | Бас, timpani |
| **Средний** | 2-4 удара / такт | Струнные, пиано |
| **Быстрый** | 8+ ударов / такт | Percussion, hi-hat |

### Ритмические слои

**Стратегия наложения:**

| Слой | Инструмент | Паттерн |
|------|-----------|---------|
| **1. Фундамент** | Bass drum, Timpani | 1-я доля |
| **2. Пульс** | Strings ostinato | 8-е ноты |
| **3. Деталь** | Snare, Hi-hat | 16-е ноты |
| **4. Акцент** | Cymbals, Brass | Ключевые доли |

### Ритм и динамика

- **Crescendo** — нарастание ритмической плотности
- **Diminuendo** — уменьшение, расслабление
- **Ritardando** — замедление, подготовка к финалу
- **Accelerando** — ускорение, экшн

### Практические приёмы

| Приём | Описание |
|-------|---------|
| **Layering** | Наложение ритмических слоёв |
| **Call & Response** | Ответные фразы между инструментами |
| **Syncopation** | Смещение акцентов |
| **Polyrhythm** | Несколько ритмов одновременно |
| **Tic-Tac** | Быстрые короткие удары |

!!! tip
    **Ритмическое нарастание:** начинайте с одного инструмента и постепенно добавляйте слои. К кульминации все ритмические слои работают вместе.

!!! important
    Ритм в оркестре — это не сетка. Humanize, velocity, и микро-сдвиги делают ритм живым. Идеальная сетка звучит механически.
