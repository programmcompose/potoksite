# Библиотеки Kontakt — Подбор и настройка

Kontakt — самая популярная платформа для сэмплов в индустрии саундтреков. Правильный подбор библиотек определяет 80% качества вашего оркестрового звука.

## Настройка Kontakt

### Базовая настройка

Kontakt работает как хост для сэмплов. Правильная настройка — фундамент качества звука.

| Параметр | Рекомендация |
|----------|-------------|
| **Буфер** | 1024-2048 сэмплов для баланса задержки и нагрузки |
| **Потребление RAM** | Мониторьте через Task Manager, не перегружайте |
| **NNTP (Native Native True Positives)** | Включайте для библиотек, которые его поддерживают |
| **Прелоад** | Загружайте библиотеки до начала работы |

### Управление памятью

- **Fast Load** — быстрая загрузка, но больше потребление RAM
- **Standard Load** — баланс между скоростью и памятью
- **Lazy Load** — экономит память, загружает при первом использовании

!!! tip
    Закрывайте неиспользуемые инструменты. 64 GB RAM — комфортный минимум для полноценного оркестра.

## В чём разница библиотек

### Топ-вендоры

| Вендор | Специализация | Цена | Качество |
|--------|--------------|------|----------|
| **KMI (Krotkov Music Instruments)** | Струнные, духовые | Бюджет | Хороший старт |
| **LSI (Lior Shamgar Instruments)** | Гибридный оркестр | Средний | Отличный баланс |
| **Spitfire Audio** | Оркестр, эмбиент | Премиум | Индустриальный стандарт |
| **Orchestral Tools** | Полный оркестр | Премиум | Топ-качество |
| **VSL (Vienna Symphonic Library)** | Классика, кино | Премиум | Золотой стандарт |

### Сравнение подходов

- **Библиотека-оркестр (SSpitfire, OT, VSL)** — полный набор инструментов, дорого, максимальное качество
- **Отдельные инструменты (KMI, LSI)** — выборочно, бюджетно, достаточно для начала
- **Гибридные (LSI, Output)** — оркестр + синты, универсально

## Какие выбрать

### Для старта

1. **KMI Strings** — струнные, основа оркестра
2. **KMI Brass / Winds** — духовые секции
3. **KMI Percussion** — ударные и перкуссия
4. **KMI Piano** — рояль

### Для продвинутого уровня

1. **Spitfire Albion** — полный пакет оркестра
2. **LSI Hybrid Strike** — эпик, трейлер
3. **Spitfire Atmospheres** — эмбиент, текстуры

### Минимальный набор для саундтрека

| Инструмент | Рекомендация |
|-----------|-------------|
| Струнные | KMI Strings / Spitfire Strings |
| Духовые | KMI Brass / LSI Session Strings |
| Пиано | KMI Piano / Spitfire Piano |
| Перкуссия | KMI Percussion / LSI Percussion |
| Эмбиент | Spitfire Atmospheres / Output Exosphere |

!!! important
    **Не гонитесь за количеством.** Лучше одна качественная библиотека, чем десять средних. Начните с KMI, вырастите до Spitfire.
