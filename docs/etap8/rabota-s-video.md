# Работа с видеорядом и техническая часть

Финальный блок этапа: как музыка встраивается в видео-продукцию и какие технические навыки нужны.

## Работа с видеорядом

### Кино

**Специфика:**

| Аспект | Описание |
|--------|---------|
| **Тайминг** | Музыка синхронизирована с кадрами |
| **Диалоги** | Музыка не конфликтует с речью |
| **Сцены** | Каждый блок — отдельная музыкальная секция |
| **Эмоции** | Музыка усиливает, а не заменяет |

**Workflow:**
1. Получение таймкодов и описания сцен
2. Создание temp score (временная музыка для референса)
3. Написание оригинальной музыки
4. Синхронизация с картинкой
5. Сведение с учётом диалогов и SFX

### Игры

**Специфика:**

| Аспект | Описание |
|--------|---------|
| **Интерактивность** | Музыка реагирует на действия игрока |
| **Зацикливание** | Треки должны бесшовно повторяться |
| **Слои** | Вертикальный композинг (stems) |
| **Состояния** | Исследование, бой, опасность |

**Вертикальный композинг:**

| Слой | Состояние |
|------|---------|
| **1** | Фон, исследование |
| **2** | + Струнные, напряжение |
| **3** | + Перкуссия, подготовка к бою |
| **4** | + Полный оркестр, бой |

### Анимация / Трейлеры

**Анимация:**
- Более стилизованная музыка
- Мультяшные элементы (пицicato, woodwinds)
- Яркие тембральные контрасты

**Трейлеры:**
- Короткие, интенсивные треки
- Бит-дропы, взрывы
- SFX как часть музыки

### Саунд-дизайн

**Что такое саунд-дизайн:**

Создание уникальных звуков, которые не существуют в природе.

| Техника | Описание |
|---------|---------|
| **Сэмплирование** | Запись реальных звуков, обработка |
| **Синтез** | Создание звуков с нуля |
| **Layering** | Наложение нескольких звуков |
| **Processing** | FX, реверб, дисторшн |
| **Foley** | Имитация звуков действий |

## Подходы

### Композиторские подходы

| Подход | Описание | Когда |
|--------|---------|-------|
| **Тема-first** | Сначала мелодия, потом оркестровка | Когда есть чёткая идея |
| **Texture-first** | Сначала атмосфера, потом тема | Эмбиент, фон |
| **Rhythm-first** | Сначала ритм, потом мелодия | Экшн, трейлер |
| **Harmony-first** | Сначала аккорды, потом развитие | Лирика, драма |

### Итеративный процесс

1. **Черновик** — быстрая наброска
2. **Оркестровка** — выбор инструментов
3. **Аранжировка** — развитие, динамика
4. **Сведение** — баланс, пространство
5. **Финализация** — мастеринг, экспорт

## Техническая часть

### Сведение и Плагины

#### Необходимые плагины

| Категория | Плагины |
|-----------|--------|
| **EQ** | FabFilter Pro-Q 3, Pultec |
| **Компрессия** | SSL Bus Compressor, 1176, LA2A |
| **Реверб** | Valhalla Room/Verb, Plate |
| **Delay** | Soundtoys EchoBoy |
| **Сатурация** ** | Saturn 2, Decapitator |
| **Мастеринг** ** | Ozone 11, Triton |
| **Утилиты** ** | Youlean Loudness Meter, Reference |

#### Технический пайплайн

1. **Session template** — готовый шаблон в DAW
2. **Track organization** — группировка, цвета, маршрутизация
3. **Bus processing** — обработка на группах
4. **Mix bus** — финальное склеивание
5. **Master bus** — контроль громкости
6. **Export** — рендер в нужном формате

### Экспорт для видео

| Формат | Использование |
|--------|--------------|
| **WAV 24-bit / 48kHz** | Стандарт для видео |
| **WAV 24-bit / 44.1kHz** | Стриминг, музыка |
| **Stems** | Отдельные группы для пост-продакшна |
| **Score mix** | Музыка без диалогов и SFX |
| **M&E (Music & Effects)** | Музыка + эффекты без диалогов |

!!! important
    **Техническая часть — это фундамент.** Даже гениальная композиция пропадёт при плохом сведении. Инвестируйте время в изучение плагинов и пайплайна.
