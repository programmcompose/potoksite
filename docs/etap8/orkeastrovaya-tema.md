# Пишем оркестровую тему

Оркестровая тема — это мелодия, способная работать с полным оркестром. Разберём атомарные элементы её создания.

## Звукоряд (тональность или лад)

### Выбор тональности

| Тональность | Характер | Примеры |
|------------|---------|---------|
| **C Major** | Свет, чистота | Героика, триумф |
| **D Major** | Энергия, мощь | Эпик, оркестр |
| **A Minor** | Грусть, драма | Трагедия |
| **E Minor** | Напряжение | Опасность |
| **F Major** | Тёплый, лиричный | Надежда |
| **G Minor** | Тёмный, драматичный | Конфликт |

### Выбор лада

- **Ионийский** — стандартный майор
- **Эолийский** — натуральный минор
- **Дорийский** — минор с надеждой
- **Фригийский** — экзотика, опасность
- **Лидийский** — магия, сновидение
- **Миксолидийский** — свобода, эпик

## Тип движения

### Восходящее движение

- **Эмоция:** надежда, рост, триумф
- **Техника:** постепенный подъём по ступеням
- **Пример:** тема из «Титаника»

### Нисходящее движение

- **Эмоция:** грусть, потеря, упадок
- **Техника:** спуск по ступеням, каскады
- **Пример:** тема из «Списка Шиндлера»

### Зигзагообразное движение

- **Эмоция:** неопределённость, конфликт
- **Техника:** чередование подъёмов и спусков
- **Пример:** темы из триллеров

## Аккорды

### Построение аккордов к теме

1. **Определите тонику** — главная нота темы
2. **Выберите 3-4 аккорда** — I, IV, V, vi
3. **Расставьте по тактам** — 1 аккорд на 2-4 такта
4. **Проверьте мелодию** — все ли ноты вписываются

### Расширенные аккорды для кино

- **maj7** — мечтательность, ностальгия
- **min7** — джазовая драма
- **dom7** — напряжение, движение
- **add9** — открытость, воздушность
- **sus2 / sus4** — ожидание, предвкушение

## Ритм

### Ритмические паттерны

| Паттерн | Использование |
|---------|-------------|
| **Длинные ноты** | Лиризм, величие |
| **Рубато** | Свобода, экспрессия |
| **Ритмические акценты** | Энергия, драйв |
| **Синкопы** | Неожиданность |
| **Триоли** | Поток, движение |

### Темп

| Темп | Жанр |
|------|------|
| **40-60 BPM** | Эпик, трагедия |
| **60-80 BPM** | Лирика, драма |
| **80-100 BPM** | Героика, действие |
| **100-120 BPM** | Экшн, погоня |

!!! tip
    **Проверка темы:** сыграйте свою тему на пиано в унисон. Если она звучит хорошо без оркестровки — тема рабочая.
