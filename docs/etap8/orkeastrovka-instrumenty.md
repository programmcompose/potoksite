# Оркестровка и основные инструменты

Оркестр — это не один инструмент, а целая армия звуков. Понимание каждого раздела — ключ к грамотной оркестровке.

## Струнные / Strings

Струнные — основа оркестра. Они несут мелодию, гармонию и эмоции.

### Секции струнных

| Секция | Диапазон | Роль |
|--------|---------|------|
| **Violin I** | Высокий | Ведущая мелодия |
| **Violin II** | Высокий | Гармоническая поддержка, контрапункт |
| **Viola** | Средний | Заполнение гармонии, теплота |
| **Cello** | Низкий | Мелодия и бас |
| **Double Bass** | Очень низкий | Низкочастотный фундамент |

### Артикуляции струнных

- **Legato** — плавное связное извлечение, основа мелодии
- **Spiccato** — отрывистое, ритмическое
- **Pizzicato** — щипок, акцентировка
- **Tremolo** — тремоло, напряжение
- **Sul Pont / Sul G** — тембральные эффекты

!!! tip
    Legato — ваша рабочая лошадка. 80% оркестровых тем строятся на legato-струнных.

## Духовые / Winds

Духовые добавляют цвет, характер и мощь.

### Семейства духовых

| Инструмент | Характер | Роль |
|-----------|---------|------|
| **Flute** | Лёгкий, прозрачный | Мелодия, атмосфера |
| **Oboe** | Ностальгический | Лирические мелодии |
| **Clarinet** | Гибкий, тёплый | Соло, заполнение |
| **Bassoon** | Низкий, комичный | Бас, характер |
| **French Horn** | Тёплый, мощный | Гармония, эпик |
| **Trumpet** | Яркий, героический | Фанфары, тема |
| **Trombone** | Тёмный, мощный | Драма, низ |
| **Tuba** | Очень низкий | Фундамент |

### Группировка духовых

- **Woodwinds** — флейты, гобое, кларнеты, фаготы
- **Brass** — валторны, трубы, тромбоны, туба

## Пиано

Пиано — универсальный инструмент саундтрека.

### Роль пиано

- **Соло** — интимные моменты, основная тема
- **Аккомпанемент** — арпеджио, блоки
- **Текстура** — пэды, мягкие аккорды

### Диапазон

| Зона | Использование |
|------|--------------|
| **Низ** | Поддержка баса, аккорды |
| **Средний** | Гармония, аккомпанемент |
| **Высокий** | Мелодия, бриллиантовые акценты |

## Арфа / Harp

Арфа создаёт волшебство и воздушность.

- **Арпеджио** — текучие пассажи
- **Glissando** — глайссандо, магия
- **Аккорды** — мягкая гармоническая поддержка

## Перкуссия / Percussion

Перкуссия задаёт ритм, драму и масштаб.

### Типы перкуссии

| Инструмент | Роль |
|-----------|------|
| **Snare Drum** | Ритм, маршевость |
| **Bass Drum** | Мощь, удары |
| **Timpani** | Драматические акценты |
| **Cymbals** — Crash, Ride | Транзишены, атмосфера |
| **Triangle** | Блеск, акценты |
| **Tambourine** | Ритм, энергия |

## Синтезаторы и Сэмплы

Гибридный саундтрек сочетает оркестр и электронику.

### Роль синтов

- **Пэды** — фон, атмосфера, текстуры
- **Басы** — низкочастотная поддержка
- **Лиды** — яркие мелодические вставки
- **FX** — переходы, эффекты, переходы

### Популярные инструменты

| Синт | Специализация |
|------|-------------|
| **Omnisphere** | Пэды, текстуры, всё |
| **Serum** | Басы, лиды, саб-басы |
| **Spitfire Atmospheres** | Эмбиент, кинематографичность |
| **Output Exosphere** | Гибридные текстуры |

!!! important
    **Правило гибридного микса:** синты заполняют частотные дыры оркестра, а не конкурируют с ним. Оркестр — фронт, синты — бэк.
