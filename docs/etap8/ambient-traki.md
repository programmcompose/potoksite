# Эмбиент треки

Эмбиент — это музыка, которая создаёт пространство, а не рассказывает историю. Атмосфера, текстуры, бесконечность.

## Атмосферные синты

### Пэды

**Типы пэдов:**

| Тип | Характер | Использование |
|-----|---------|--------------|
| **Warm Pad** | Мягкий, тёплый | Фон, атмосфера |
| **Dark Pad** | Насыщенный, тёмный | Напряжение, драма |
| **Ethereal Pad** | Воздушный, прозрачный | Мечтание, космос |
| **Choir Pad** | Хоральный | Эпик, духовность |
| **Granular Pad** | Зернистый | Текстуры, эмбиент |

### Создание атмосферы

**Слоистый подход:**

| Слой | Что | Инструмент |
|------|-----|-----------|
| **1. Дрон** | Бесконечная нота | Синт, бас |
| **2. Пэд** | Гармонический фон | Omnisphere, Atmospheres |
| **3. Текстура** | Детали, движение | Granular, field recordings |
| **4. Мелодия** | Тонкая линия | Пиано, флейта, синт |

## Прогрессия аккордов

### Эмбиент-прогрессии

**Медленные, воздушные:**

| Прогрессия | Характер |
|-----------|---------|
| **I — IV — V — vi** (медленно) | Спокойствие |
| **i — bIII — bVI — bVII** | Мрачный эмбиент |
| **Imaj7 — IVmaj7 — vi — III7** | Мечтательный |
| **I — III — IV — V** (каждый на 8 тактов) | Медитативный |

### Темп и длительность

- **Аккорды держатся 4-8 тактов** — медленная смена
- **BPM 40-70** — медитативный темп
- **Размытые переходы** — автоматизация, не резкие смены

## Мягкое наслоение синтов и FX

### Техники наслоения

| Техника | Описание |
|---------|---------|
| **Crossfade** | Плавный переход между пэдами |
| **Volume automation** | Постепенное появление/исчезновение |
| **Filter automation** | Открытие/закрытие фильтра |
| **Reverb automation** | Изменение пространства |
| **Delay feedback** | Нарастание эха |

### FX для эмбиента

| Эффект | Роль |
|--------|------|
| **Reverb** | Пространство, глубина |
| **Delay** | Движение, эхо |
| **Chorus** | Ширина, движение |
| **Granular** | Текстуры, зернистость |
| **Reverse** | Сновидения, ностальгия |
| **Bitcrusher** | Глитч, цифровая эстетика |

## Текстуры

### Создание текстур

**Источники:**

- **Field recordings** — природа, город, пространство
- **Vinyl crackle** — виниловый шум, тёплый фон
- **Tape hiss** — ленточный шум, аналоговость
- **Synthesized noise** — синтезированный шум
- **Processed vocals** — обработанные вокальные сэмплы

### Слоистые текстуры

1. **Базовый слой** — дрон или пэд
2. **Средний слой** — текстура (шум, грануляр)
3. **Верхний слой** — детали (блики, арпеджио)

## Мелодика и Сведение

### Мелодика в эмбиенте

**Принципы:**

- **Минимализм** — мало нот, много пространства
- **Повторение** — короткие мотивы, зацикленные
- **Вариации** — микро-изменения в каждом повторе
- **Пространство** — паузы важны как ноты

### Сведение эмбиента

| Параметр | Рекомендация |
|---------|-------------|
| **EQ** | Мягкий срез ниже 30 Hz, лёгкий подъём highs |
| **Компрессия** | Минимальная, 1.5:1, slow attack |
| **Реверб** | Много, 4-8s decay, hall |
| **Delay** | Medium, 1-3 repeats |
| **Панорама** | Широкая стерео-картина |
| **Громкость** | -18 to -16 LUFS (эмбиент не нужен громким) |

### Целевой звук

- **Бесшовный** — нет резких переходов
- **Пространственный** — ощущение глубины
- **Тёплый** — аналоговый характер
- **Бесконечный** — трек можно зациклить

!!! tip
    **Эмбиент-трюк:** запишите звук дождя, огня или комнаты, пропустите через гранулярный синтезатор — получите уникальную текстуру.

!!! important
    Эмбиент — это искусство терпения. Не спешите добавлять элементы. Дайте каждому слою «дышать». Тишина — ваш инструмент.
