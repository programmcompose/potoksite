# Оркестровые аккорды

Оркестровые аккорды — это не просто ноты, это текстура, пространство и эмоция. Разберём структуру, расположение и голосоведение.

## Структура аккордов

### Базовые типы

| Тип | Состав | Характер |
|-----|--------|---------|
| **Трезвучие** | 1-3-5 | Основа, стабильность |
| **Септаккорд** | 1-3-5-7 | Богатство, напряжение |
| **Нонаккорд** | 1-3-5-7-9 | Кинематографичность |
| **Добавочные ступени** | add2, add4, add9 | Воздушность |

### Примеры в C Major

| Аккорд | Ноты | Использование |
|--------|------|--------------|
| **C** | C-E-G | Тоника, покой |
| **Cmaj7** | C-E-G-B | Мечтательность |
| **Cadd9** | C-E-G-D | Открытость |
| **C9** | C-E-G-B-D | Эпик, богатство |

## Расположение

### Тесное расположение

- **Описание:** ноты аккорда в пределах одной октавы
- **Звучание:** плотный, компактный
- **Где использовать:** средний регистр, заполнение

### Широкое расположение

- **Описание:** ноты разнесены на 2+ октавы
- **Звучание:** открытый, воздушный
- **Где использовать:** эпик, кульминация, тема

### Сравнение

| Параметр | Тесное | Широкое |
|---------|--------|---------|
| **Плотность** | Высокая | Низкая |
| **Эмоция** | Интимная | Эпическая |
| **Регистр** | Средний | Широкий |
| **Пример** | Камерная музыка | Трейлеры |

## Обращения

### Что такое обращения

Обращение — переordering нот аккорда, где бас не является тоникой.

| Обращение | Бас | Эффект |
|-----------|-----|--------|
| **Основное** | Тоника | Стабильность |
| **1-е обращение** | Терция | Мягкость, движение |
| **2-е обращение** | Квинта | Неустойчивость, напряжение |

### Практическое применение

- **Плавные переходы** — обращения минимизируют прыжки между аккордами
- **Басовые линии** — обращения создают движение в басу
- **Гармонический интерес** — то же самое звучит иначе

## Голосоведение

### Принцип общих нот

**Правило:** при переходе между аккордами сохраняйте общие ноты на месте.

**Пример:** C → F

| Голос | C | F | Действие |
|-------|---|---|---------|
| **Верхний** | G | G | Общий, держим |
| **Средний** | E | F | Движение на ступень |
| **Нижний** | C | C | Общий, держим |

### Правила голосоведения

1. **Избегайте параллельных октав и квинт** — каждый голос независим
2. **Минимизируйте прыжки** — плавное движение лучше
3. **Не пересекайте голоса** — порядок регистров сохраняется
4. **Октавный диапазон** — не выходите за пределы инструментов

## Задержания

### Sus2 и Sus4

| Тип | Описание | Разрешение |
|-----|---------|-----------|
| **sus2** | 2-я ступень вместо 3-й | → терция (вверх) |
| **sus4** | 4-я ступень вместо 3-й | → терция (вниз) |

### Применение в саундтреке

- **Нарастание напряжения** — sus4 перед разрешением
- **Открытое звучание** — sus2 для воздушности
- **Драматический эффект** — задержка разрешения на полтакта

### Пример прогрессии с задержаниями

```
C  →  Csus4  →  C  →  F  →  Fsus2  →  F  →  G  →  G7
```

!!! tip
    **Оркестровый аккорд ≠ пиано-аккорд.** В оркестре каждая нота — это инструмент или секция. Думайте не нотами, а голосами.

!!! important
    Голосоведение — это то, что отличает любителя от профессионала. Хорошее голосоведение делает аккорды «живыми», каждое движение имеет смысл.
