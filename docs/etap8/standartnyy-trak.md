# Стандартный минутный оркестровый трек

Минутный трек — формат трейлеров, коротких сцен и рекламных роликов. За 60 секунд нужно рассказать историю.

## Саундтрек

### Создание основной темы

#### Структура минутного трека

| Секция | Время | Что происходит |
|--------|-------|---------------|
| **Интро** | 0-10с | Атмосфера, настройка |
| **Тема** | 10-30с | Основная мелодия, раскрытие |
| **Развитие** | 30-45с | Усложнение, нарастание |
| **Кульминация** | 45-55с | Максимум инструментов |
| **Финал** | 55-60с | Разрешение или обрыв |

### Важность темы

Тема — сердце трека. За 20 секунд она должна:
- Зацепить слушателя
- Передать эмоцию
- Запомниться

### Аккорды к теме

**Подход:**
1. Выберите тональность
2. Определите 3-4 аккорда для прогрессии
3. Создайте мелодию поверх аккордов
4. Проверьте, работает ли тема в унисоне

### Развитие темы в инструментах

**Волновая техника:**

| Волна | Инструменты |
|-------|-------------|
| **1** | Пиано / соло струнный |
| **2** | + Струнная секция |
| **3** | + Духовые |
| **4** | + Перкуссия, полный оркестр |

## Трейлер

### Эпик

**Элементы эпического звука:**

- **Брамм-биты** — низкочастотные удары
- **Taiko барабаны** — мощь, масштаб
- **Полный оркестр** — все секции одновременно
- **Хор** — человеческий голос, эпичность

### Яркие и насыщенные темы

**Приёмы:**
- **Широкие интервалы** — октавы, квинты
- **Параллельные сексты** — богатство
- **Мелодические арки** — восходящие фразы
- **Динамические контрасты** — тишина → взрыв

### Драмка

#### Ритмика трейлера

| Элемент | Роль |
|---------|------|
| **Kick** | Пульс, сердцебиение |
| **Snare** | Акценты, маршевость |
| **Taiko** | Мощь, удары |
| **Cymbals** | Транзишены, блеск |
| **Rimshot** | Деталь, энергия |

**Бит-реduction:**
- Начало: минимум драмки
- Середина: добавление слоёв
- Кульминация: полный ударный набор

### SFX

**Звуковые эффекты в саундтреке:**

- **Whoosh** — переходы, движения
- **Hit** — акценты, удары
- **Riser** — нарастание напряжения
- **Impact** — кульминационные удары
- **Reverse Cymbal** — предвкушение

!!! tip
    SFX — это не замена музыке, а её усиление. Используйте sparingly (умеренно), чтобы не перегрузить микс.

!!! important
    **Формула трейлера:** тишина → нарастание → взрыв → тишина. Контраст — ваш лучший друг.
