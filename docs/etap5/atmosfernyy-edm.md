# Атмосферный EDM

Атмосферный EDM — направление, где **пространство, текстура и настроение** важнее танцевального ритма. Это музыка, которая создаёт мир вокруг слушателя.

## Синты

### EDM синты

Особенности синтов в атмосферном EDM:

- **Пэды** — широкие, объёмные, заполняющие пространство
- **Полетные лиды** — мелодии, «парящие» над миксом
- **Текстурные слои** — фоновые элементы, создающие глубину

### Глитчи

Glitch-эстетика:

- **Stutter-эффекты** — прерывистые повторения
- **Bit-crush** — снижение битности для «цифровой» грубости
- **Granular-синтез** — разложение семплов на гранулы
- **Randomization** — случайные модуляции для непредсказуемости

### Жир и плотность

- **Layering** — наложение нескольких слоёв для плотности
- **Сатурация** — добавление гармонических обертонов
- **Параллельная обработка** — compression на параллельных треках
- **Low-end stacking** — суббас + базовый слой + текстурный слой

### Импровизация и лееринг

- **Свободный эксперимент** — без жёстких структур
- **Случайные комбинации** — неожиданные пары звуков
- **Модуляция в реальном времени** — live automation параметров
- **Textural layering** — звуки как «краски» на холсте

## Драмка

Драм-машина в атмосферном EDM:

- **Минимализм** — меньше элементов, больше пространства
- **Half-time и slow-tempo** — 120-130 BPM или медленнее
- **Organic элементы** — реальные ударные, field recordings
- **Пространственная обработка** — реверб, delay, панорамирование

## Основные отличия

Атмосферный EDM отличается от «классического» EDM:

| Параметр | Классический EDM | Атмосферный EDM |
|----------|-----------------|-----------------|
| Фокус | Танцевальный ритм | Настроение и пространство |
| Структура | Buildup → Drop | Плавные переходы |
| Драмка | Плотная, энергичная | Разреженная, текстурная |
| Синты | Яркие, пробивные | Мягкие, объёмные |
| Сведение | Громкий, компрессированный | Динамичный, с глубиной |
