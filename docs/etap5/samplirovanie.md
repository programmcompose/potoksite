# Сэмплирование Slice и Chop

Сэмплирование — искусство **взять существующий звук и превратить его в инструмент**. Slice и Chop — две核心技术, которые лежат в основе EDM-продакшна.

## Slice

**Slice** — разделение сэмпла или лупа на отдельные части по ударным или мелодическим точкам.

### Мелодия

Слайсирование мелодических семплов:

- Разделение по нотам
- Перераспределение по Piano Roll
- Изменение питча и тайминга

### Драмка

Слайс драм-лупов:

- Автоматический слайс по transient-ам
- Ручная настройка точек слайса
- Перестроение ритма из готовых лупов

### Работа с ритмом

- Изменение groove-а
- Добавление свинга
- Синхронизация с BPM проекта

## Chop

**Chop** — ручное нарезание семплов на фрагменты для создания новых паттернов.

### Редактирование семплов

- Trim, fade in/out
- Pitch shifting без изменения скорости
- Time stretching без изменения питча
- Reverse, granular-эффекты

### Работа с лупами

- Loop detection
- Stretching лупов под проект
- Chopping лупов на части
- Создание новых лупов из chopped-фрагментов

## VST чоперы и слайсеры

### Serato Sample

Популярный сэмплер для chop-инга:

- Slice mapping
- Pitch control
- Slew и decay
- Filter и EQ

### Fruity Slicer

Встроенный инструмент FL Studio:

- Автоматический слайс по битам
- Слайс по сэмплам
- Переназначение на ноты
- Pattern-based работа

### Sampler Ableton

Simple/Simpler в Ableton Live:

- One-shot сэмплинг
- Retrig и warp
- Envelope и filter
- Grain mode для текстурирования
