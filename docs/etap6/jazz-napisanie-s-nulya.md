# Джаз — Написание с нуля

Джаз требует понимания гармонии. Но с правильным подходом, вы сможете создавать джазовые arregiroвки.

## Джазовые прогрессии

### Постоянная смена аккордов

В отличие от рока (3-4 аккорда), джаз использует быструю смену гармонии.

| Жанр | Смена аккордов |
|------|---------------|
| **Рок** | Каждый 2-4 такта |
| **Поп** | Каждый 1-2 такта |
| **Джаз** | Каждый полтакта или такт |

### ii-V-I — Основа джаза

**ii-V-I** — фундаментальная прогрессия джаза. Понимание этой цепочки открывает дверь в жанр.

```
В тональности C major:
ii = Dm7  →  V = G7  →  I = Cmaj7

В тональности F major:
ii = Gm7  →  V = C7  →  I = Fmaj7
```

### Как тренировать ii-V-I

1. **Выучите ii-V-I в 4-5 тональностях** по памяти
2. **Играйте джазовые стандарты** — они построены на ii-V-I
3. **Практикуйте daily** — 10 минут в день лучше часа раз в неделю

### Джазовые стандарты для практики

| Стандарт | Прогрессии |
|---------|-----------|
| **Autumn Leaves** | ii-V-I в разных тональностях |
| **All The Things You Are** | Модуляции, ключевые изменения |
| **Summertime** | Медленный темп, простая гармония |
| **Blue Bossa** | Босса-нова ритм, ii-V-I |

## Доп. ступени: 7-9-11-13

Джаз использует **расширенные аккорды** — не три ноты, а четыре и более.

### Структура расширенных аккордов

| Ступень | Название | Пример (C) |
|---------|----------|-----------|
| **5** | Квинта | G |
| **7** | Септима | Bb (минор) / B (мажор) |
| **9** | Нонa | D |
| **11** | Одиннадцатая | F |
| **13** | Тринадцатая | A |

### Примеры аккордов

```
Cmaj7 = C - E - G - B
Dm9 = D - F - A - C - E
G13 = G - B - D - F - A - E
```

### Как использовать

- **Начните с 7-ых аккордов** — основа джазовой гармонии
- **Добавьте 9-ую** — для цвета
- **11 и 13** — продвинутый уровень, для обогащения

## Приятный и мягкий звук

### Философия джаз-звука

| Параметр | Цель |
|----------|------|
| **Тёплота** | Мягкие верха, богатые mid |
| **Пространство** | Умеренный реверб, «клуб» |
| **Динамика** | Живое дыхание, не сжато |

### EQ для джаза

- **Низ:** Тёплый, не агрессивный (80-200 Гц)
- **Mid:** Богатый, характерный (500-2000 Гц)
- **Высокий:** Мягкий roll-off выше 10 кГц

## Важность мелодии

В джазе мелодия — король.

- **Мелодия ведёт**, аккомпанемент следует
- **Свобода** — мелодия не привязана к сетке
- **Синкопы** — смещение акцентов, грув

## Живость и импровизации

### Импровизация в DAW

| Техника | Описание |
|---------|----------|
| **Свободный пиано-ролл** | Не привязывайте к сетке |
| **Variance** | Добавляйте случайность в тайминг |
| **Ad-libs** | Запишите импровизационные вставки |
| **Swing** | Включите swing в DAW (55-65%) |

### Swing

**Swing** — ритмический характер джаза. Краткие ноты укорачиваются, длинные растягиваются.

| Swing % | Характер |
|---------|---------|
| **50%** | Прямой, без swing |
| **55-60%** | Лёгкий swing |
| **65%+** | Явный swing, фанк |

!!! tip
    **Swing — не опция в джазе.** Без swing джаз звучит как «робот, пытающийся играть джаз». Всегда включайте swing 55-65%.
