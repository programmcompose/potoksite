# Джаз — Вариации жанра

Джаз — не монолит. Внутри жанра существуют кардинально разные стили.

## Noir Jazz (Джаз Нуар)

### Что такое Noir Jazz

Атмосферное направление, сочетающее джаз с кинематографичным напряжением.

| Элемент | Описание |
|---------|----------|
| **Темп** | Медленный, тягучий |
| **Гармония** | Тёмная, минорная, диссонансная |
| **Атмосфера** | Дождь, ночной город, одиночество |
| **Инструменты** | Тромбон, тенор-сакс, рояль, кинематографичные пэды |

### Продакшн Noir Jazz

- **Много реверба** — длинный hall, 4-6 секунд
- **Низкочастотное давление** — дрон, суб-бас
- **Минимальная драмка** — brushes, тихий кик
- **Дождь / шум города** — фоновые текстуры

### Референсы

- Triple Seam — современный noir jazz
- Glorium — кинематографичный джаз
- «Blade Runner» soundtrack — Vangelis + джаз

## Быстрый и фанковый джаз

### Up-tempo Jazz

| Характеристика | Описание |
|---------------|----------|
| **Темп** | 140-200 BPM |
| **Энергия** | Высокая, танцевальная |
| **Драмка** | Активная, ride-цимбал, сложные филлы |
| **Импровизация** | Быстрые соло, виртуозность |

### Funk Jazz

| Характеристика | Описание |
|---------------|----------|
| **Ритм** | Фанковый грув, syncopation |
| **Бас** | Активный, мелодичный, фанковый |
| **Клавиши** | Rhodes, органи, фанковые аккорды |
| **Гроув** | «В кармане», танцевальный |

### Продакшн быстрый джаз

- **Чёткая драмка** — меньше реверба, больше punch
- **Компрессия** — умеренная, для контроля динамики
- **Бас вперёд** — фанковый бас — основа
- **Компактный микс** — меньше пространства, больше драйва

### Референсы

- Herbie Hancock — фанк-джаз легенда
- Weather Report — fusion jazz
- Robert Glasper — современный jazz-hop
- Kamasi Washington — эпический современный джаз

!!! note
    **Джаз — самый разнообразный жанр.** От медленного noir до быстрого fusion. Не ограничивайте себя одним стилем — экспериментируйте.
