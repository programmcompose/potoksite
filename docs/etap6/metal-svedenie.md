# Метал — Сведение

Сведение метала — баланс между плотностью и чёткостью. Каждый элемент должен быть слышен в «стене звука».

## Сведение гитар

### Стерео-изображение

| Техника | Описание |
|---------|----------|
| **Pan L50/R50** | Классическое раскрытие ритм-гитар |
| **Triple layer** | L50 + R50 + центр для плотности |
| **Разные пресеты** | Каждый слой с уникальным EQ-характером |

### Плотный низ

- **High-pass filter:** 60-80 Гц (ниже — мусор)
- **Бас и кик** занимают суб-низ (20-80 Гц)
- **Гитары** начинают с 80 Гц, не ниже

### Нюанс 3-4 кГц

| Диапазон | Роль |
|----------|------|
| **3 кГц** | Attack гитары, «ударный» характер |
| **4 кГц** | Presence, прорыв через микс |
| **Boost 3-4 кГц** | +2-3 dB для чёткости риффов |

!!! note
    **3-4 кГц — секретная зона метала.** Небольшой boost в этом диапазоне делает гитары «зубчатыми» и прорезающими. Но не перебарщивайте — резкость утомляет.

## Сведение драмки

### Комната

- **Room-микрофоны** — основа ощущения пространства
- **Blend close + room** — чёткость + размер
- **EQ room:** cut mid для грязи, boost высокочастотный «air»

### Кранч на драмке

- **Сатурация на снейре** — характер, «характерный хлопок»
- **Kick:** лёгкая сатурация для атаки
- **Не перегружайте** — драмка должна оставаться чёткой

### Лееринг (Layering) сэмплов

| Слой | Роль |
|------|------|
| **Библиотека** | Основа, основной звук |
| **Сэмпл-слой** | Дополнительная атака или тело |
| **One-shot** | Усиление кика или снейра |

### Distressor на драмке

**Fairchild Distressor** — плагин для агрессивной компрессии.

- **All Buttons In** — классический «грязный» пресет
- **Red line** — не бойтесь перегруза, это часть звука
- **Применение:** снейр, kick, или весь drum-BUS

## Параллельные обработки

### Parallel Compression

1. Дублируйте drum-bus
2. Жёсткая компрессия на дубле (10-20 dB reduction)
3. Микш: 30-50% compressed + dry
4. Результат: контроль + punch

### Parallel Distortion

- Аналогично компрессии, но с дисторшном
- Добавляет «грязь» без потери чёткости

## Общий микс

### Баланс элементов

| Инструмент | Уровень | Примечание |
|-----------|---------|-----------|
| **Гитары** | Плотные, широкие | L/R pan, layer |
| **Бас** | Чёткий, центрированный | Склейка с киком |
| **Драмка** | Punchy, вперёд | Parallel compression |
| **Вокал** | Яркий, чёткий | Компрессия + EQ |
| **Соло-гитара** | Прорезающая | Presence boost |

### Мастер для метала

- **Компрессия:** Лёгкая, 2-3 dB, для склейки
- **EQ:** Multiband, контроль низа и верха
- **Лимитер:** True Peak -1 dB, LUFS -8 to -6 для метала

!!! important
    **Метал должен быть ГРОМКИМ, но не сплюснутым.** LUFS -8 — это цель, но не ценой динамики. Если трек «стена без дыхания» — открутите лимитер.
