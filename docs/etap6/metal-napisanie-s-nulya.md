# Метал — Написание с нуля

Метал через VST — один из сложных вызовов. Но правильные библиотеки и подход дают убедительный результат.

## Гитары

### Живая Гитара vs VST

| Аспект | Живая гитара | VST |
|--------|-------------|-----|
| **Реализм** | 100% | 80-95% (зависит от библиотеки) |
| **Скорость работы** | Медленно | Быстро |
| **Гибкость** | Фиксированная запись | Любое изменение post-factum |

### Низкий строй

Метал часто использует пониженные строи:

| Строй | Применение |
|-------|-----------|
| **Standard (E)** | Классический метал, скорость |
| **Drop D** | Тяжёлые риффы, одна кнопка |
| **Drop C** | Дэт-метал, джент |
| **B и ниже** | Дум-метал, джент, брутал |

### Shreddage 3

**Shreddage 3** — одна из лучших гитарных библиотек для метала.

- **Артикуляции:** palm-mute, гармонics, bends, vibrato
- **Low-tuning:** встроенные низкие строи
- **Микшированный звук:** готов к работе из коробки
- **Velocity-слои:** естественная динамика

### Session Guitarist

**Session Guitarist: Metal** — альтернатива для быстрого результата.

- **Предзаписанные фразы:** выбираете стиль, библиотека играет
- **Простота:** минимум настроек
- **Стили:** death, thrash, doom, groove

## Бас

### Кранч для баса

| Параметр | Значение |
|----------|---------|
| **Дисторшн** | Средний-высокий, для «грызущего» звука |
| **Бленд Clean/Dirty** | 50/50 — чистота низа + грязь mid |
| **EQ** | Boost 800-1200 Гц для присутствия |

### Чистый бас для суба

Техника двойного баса:

1. **Трек 1:** Чистый бас, только низ (cut выше 300 Гц)
2. **Трек 2:** Перегруженный бас, только mid-high (HPF 300 Гц)
3. **Микш:** together дают плотный и чёткий бас

## Драмка

### Perfect Drums

**Perfect Drums** — популярный выбор для метала.

- **Пресеты:** готовые метал-настройки
- **Микшированные:** звук «из коробки»
- **Гибкость:** индивидуальный контроль каждого элемента

### Библиотеки Kontakt

| Библиотека | Стиль |
|-----------|-------|
| **FruityDrums** | Современный метал, разнообразие |
| **DLX Drums** | Тяжёлый, агрессивный |
| **Infernal Machines** | Экстремальный метал |
| **SL Drums** | Премиум, фотореалистичный |

## Мощность и плотность

### Как достичь «большого» звука

1. **Layer гитар** — минимум 2 стерео гитары + layer дисторшна
2. **Бас + Кик** — склеенный низкочастотный фундамент
3. **Драмка** — плотная, с параллельной компрессией
4. **Сатурация на BUS** — общая «klei» для микса

!!! tip
    **Плотность метала — в layering.** Одна гитара никогда не даст «стены звука». Layer 2-3 гитар с разными пресетами и панорамированием.
