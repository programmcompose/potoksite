# Гранж — Плагины и Сэмпл паки

Правильные инструменты — половина гранж-звука.

## Гитарные библиотеки

### Для гранж-риффов

| Библиотека | Особенности |
|-----------|-----------|
| **Session Guitarist: Garage** | Готовые риффы, гаражный характер |
| **Shreddage 3: Acoustic/Electric** | Гибкость, артикуляции |
| **EZguitar: Rock** | Простота, быстрый результат |
| **Scse3** | Бесплатная, достойное качество |

### Что искать в библиотеке

- **Дисторшн-пресеты** — готовый перегруженный звук
- **Palm-mute артикуляции** — для ритмических риффов
- **Low-tuning** — низкие строи из коробки
- **Round-robin** — естественное звучание при повторении

## Драм-библиотеки

### Гранж-ударные

| Библиотека | Стиль |
|-----------|-------|
| **LA Drums** | Большие, «американские» ударные |
| **SSL 2** | Микшированные, готовые к работе |
| **Addictive Drums 2** | Гибкость, пресеты |
| **EZdrummer** | Простота, стиль гранж есть |

### Что важно для гранж-драмки

- **Большая бочка** — тёплый, полный звук кика
- **Хлесткий снейр** — с характером, не «пластиковый»
- **Room-микрофоны** — ощущение большого помещения
- **Не слишком «чистые»** — гранж не требует стерильного звука

## Эффекты для гранжа

### Обязательные плагины

| Плагин | Назначение |
|--------|-----------|
| **Decapitator** | Сатурация, грязь |
| **Reverb (Valhalla)** | Пространство, атмосфера |
| **Chorus (Soundtoys)** | Толщина на чистых гитарах |
| **Distressor** | Агрессивная компрессия |
| **Portal (Output)** |Glitch-реверб, текстуры |

### Lo-fi плагины

- **RC-20 (Retrocolor)** — кассетные артефакты
- **Distructor (Soundtoys)** — перегруз + эффекты
- **Kemet (Klanghelm)** — бесплатный bitcrush

!!! note
    **Для гранжа не нужны самые дорогие библиотеки.** Гранж — жанр характера, а не идеального звука. Среднего уровня библиотека с правильной обработкой звучит лучше, чем топ-библиотека без обработки.
