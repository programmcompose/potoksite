# Рок — Оживление VST

MIDI-гитары и драмка по умолчанию звучат механически. Задача — добавить человечности.

## Velocity (Сила нажатия)

Velocity — наиболее важный параметр для оживления MIDI.

### Как работает velocity

| Velocity | Звук |
|----------|------|
| **Одинаковое** | Роботизированное, механическое |
| **Вариативное** | Естественное, живое |

### Практика velocity

1. **Не используйте 100 везде** — реальный музыкант не бьёт с одинаковой силой
2. **Акценты** — первые доли такта чуть громче
3. **Случайность** — добавьте ±10-15 к базовому velocity
4. **Ручная правка** — проходите ноту за нотой, корректируя velocity

### Velocity для разных инструментов

| Инструмент | Диапазон | Примечание |
|-----------|----------|-----------|
| **Гитара** | 70-110 | Лёгкие вариации, не слишком резко |
| **Kick** | 85-115 | Акценты на первую долю |
| **Snare** | 95-120 | Стабильно громкий, микро-вариации |
| **Hi-Hat** | 60-100 | Открытые чуть тише закрытых |

## Humanize — Мимо сетки

Идеальный тайминг звучит неестественно. Живой музыкант играет «вокруг» сетки.

### Тайминг

| Техника | Описание |
|---------|----------|
| **Сдвиг назад (Lay Back)** | Ноты чуть позже сетки — расслабленный грув |
| **Сдвиг вперёд (Push)** | Ноты чуть раньше — агрессивный, драйвовый звук |
| **Случайный сдвиг** | ±5-15 мс для каждой ноты |

### Сколько «неидеальности» нужно

- **Лёгкий рок:** ±5-10 мс
- **Альтернативный рок:** ±10-20 мс
- **Гранж / Панк:** ±15-30 мс

!!! tip
    **Меньше — не значит лучше.** Слишком точный тайминг убивает грув. Не бойтесь сдвигать ноты.

## Удачная библиотека

Правильная библиотека делает половину работы за вас.

### Признаки хорошей библиотеки

- **Round-robin** — автоматическая смена сэмплов
- **Velocity-слои** — разные сэмплы для разных сил нажатия
- **Микшированные** — готовый звук с ревербом и компрессией
- **Артিকেции** — приглушённые, гармонические, палм-мьют для гитар

### Как проверить библиотеку

1. Сыграйте простую фразу
2. Послушайте без обработки
3. Если звучит «живо» — библиотека подходит
4. Если «роботизированно» — ищите другую

## Типичные ошибки

| Ошибка | Результат | Решение |
|--------|-----------|--------|
| **Одинаковый velocity** | Механический звук | Ручная правка velocity |
| **Идеальный тайминг** | Нет грува | Humanize ±10 мс |
| **Одна гитара** | Тонкий звук | Layer 2-3 гитар |
| **Слишком много эффектов** | Мутный микс | Убирайте по одному |
| **Нет динамики** | Скучный трек | Варируйте плотность секций |
| **Игнорирование room** | Сухая драмка | Добавьте room-микрофон |

!!! important
    **Оживление — это не плагин.** Нет кнопки «make it human». Работайте с velocity и таймингом вручную — это навык, который приходит с практикой.
