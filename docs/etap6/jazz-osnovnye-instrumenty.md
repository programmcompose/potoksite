# Джаз — Основные инструменты

Джаз — самый гармонически сложный жанр в этом курсе. Понимание инструментов — первый шаг.

## Клавиши

### Роль клавишных в джазе

| Инструмент | Роль |
|-----------|------|
| **Рояль** | Гармония, импровизация, соло |
| **Фарфоровое пианино (Rhodes)** | Мягкий, тёплый звук, фанк-джаз |
| **Орган (Hammond B3)** | Полнота, sustain, классический джаз |

### ВST для клавишных

- **Scarbee Rhodes 2** — лучшее эмуляция Rhodes
- **Spectral Rays** — премиум электропианино
- **Keyscape** — рояль
- **B3 Organ эмуляции** — Scarbee, Surge

## Трубы и духовые

### Роль духовых

| Инструмент | Характер |
|-----------|---------|
| **Труба** | Яркий лид, мелодия |
| **Саксофон** | Мелодичный, экспрессивный |
| **Тромбон** | Низкий регистр, гармония |

### VST для духовых

- **Omnisphere** — библиотека духовых
- **Wavesamples Brass** — реалистичные духовые
- **Dnpro Sax** — специализированный саксофон

## Дабл Бас (Ударный бас)

### Отличие от бас-гитары

| Параметр | Дабл бас | Бас-гитара |
|----------|---------|-----------|
| **Звук** | Тёплый, деревянный | Чёткий, электрический |
| **Техника** | Пizzicato (пальцами) | Пizzicato или медиатор |
| **Роль** | Мелодичный, «говорит» | Ритмический фундамент |

### VST для дабл баса

- **Los Agaves** — премиум, фотореалистичный
- **Session Bass** — простой, рабочий
- **Giga Bass** — Kontakt-библиотека

## Драмка

### Джазовые ударные

| Элемент | Особенности |
|---------|-----------|
| **Ride-_CYМБАЛ_** | Основной ритмический инструмент (не hi-hat) |
| **Brushes** | Мётёлki для тихих секций |
| **Kick** | Меньше доминирования, чем в роке |
| **Snare** | Акценты, не постоянный бит |

### Jazz-drum библиотеки

- **DLX Drums Jazz** — специализированная
- **LA Drums** — есть джазовые пресеты
- **Addictive Drums 2** — jazz-пак

!!! note
    **Джазовая драмка — не о силе, а о груве.** Ride-цимбал создаёт «дробящий» ритм, кик входит диалог с басом. Это не «бит» в привычном смысле.
