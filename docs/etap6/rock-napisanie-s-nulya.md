# Рок — Написание с нуля

Создание рок-трека с нуля через VST требует понимания не только инструментов, но и того, как их комбинировать.

## Подбор библиотеки

Выбор правильной библиотеки — 80% успеха. Плохая библиотека не даст нужного звука ни при какой обработке.

### Критерии выбора

| Критерий | Почему важно |
|----------|-------------|
| **Микшированные сэмплы** | Готовый звук «из коробки», минимум настройки |
| **Вариации** | Несколько取得 каждого удара для естественности |
| **Round-robin** | Автоматическая смена сэмплов при повторении |
| **Размер библиотеки** | Больше сэмплов = больше вариативности |

### Рекомендуемые библиотеки

- **Guitar:** Session Guitarist series, Shreddage 3, EZguitar
- **Drums:** SSL 2, Addictive Drums 2, LA Drums
- **Bass:** Session Bass, EZbass

!!! important
    **Не экономьте на библиотеках.** Дешёвая библиотека с плохими сэмплами потребует больше времени на сведение, чем качественная «из коробки».

## С чего начать: Гитары

### Базовый подход

1. **Определите тональность** — выберите гамму и строй
2. **Напишите рифф или аккорды** — начните с простой ритм-фразы
3. **Добавьте слой** — дублируйте гитару с другим панорамированием
4. **Проверьте плотность** — два слоя должны звучать как «стена»

### Распространённые аккордовые структуры

```
Прогрессия I - V - vi - IV:
C  G  Am  F   (в тональности C)

Прогрессия i - VII - VI - V:
Am  G  F  E   (классический рок)
```

## С чего начать: Драмка

### Рок-паттерн базовый

```
Hi-Hat:  X X X X  X X X X  X X X X  X X X X
Kick:    X . . X  . X . .  X . . X  . X . .
Snare:   . . X .  . . X .  . . X .  . . X .
```

### Принципы рок-драмки

- **Простота** — рок не требует сложных паттернов
- **Сила** — акценты на первую и третью долю
- **Филлы** — томы для перехода между секциями
- **Динамика** — купол тише, припев громче

## Гармония и аккорды

### Рок-гармония

Рок использует относительно простые гармонии, но их эффективность в энергии исполнения.

| Тип аккорда | Использование |
|------------|---------------|
| **Пентатоника** | Основа большинства риффов |
| **Пower-аккорды (5)** | Два звука — тоника и квинта, универсальны |
| **Минорные аккорды** | Мрачность, глубина |
| **Доминантовые (7)** | Напряжение, переходы |

### Советы по гармонии

- Не усложняйте — **3-4 аккорда** достаточно для сильного трека
- **Повторение** — сила рока в повторяющихся, запоминающихся фразах
- **Контраст** — чередуйте плотные и разреженные секции

## Структура рок-трека

### Типичная структура

```
Интро (4-8 тактов)
→ Куплет 1
→ Припев
→ Куплет 2
→ Припев
→ Бридж / Соло
→ Припев (x2)
→ Аутро
```

### Динамика структуры

| Секция | Уровень энергии |
|--------|----------------|
| **Интро** | Низкий, завлекающий |
| **Куплет** | Средний, строит напряжение |
| **Припев** | Максимальный, все инструменты |
| **Бридж** | Контрастный, новый элемент |
| **Финальный припев** | Пик, всё на максимум |
| **Аутро** | Сброс, затухание |

!!! note
    **Структура — это карта для слушателя.** Без чёткой структуры даже лучшие мелодии потеряются. Планируйте динамику трека до начала написания.
