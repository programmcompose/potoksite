# Метал — Жанры

Метал — не один жанр. Каждый поджанр имеет свой подход к продакшну.

## Synth + Metal (Argent Metal)

Синтезаторный метал — гибрид электроники и тяжёлых гитар.

### Характерные черты

| Элемент | Описание |
|---------|----------|
| **Синтезатор** | Мелодический лид, пэд, бас |
| **Гитары** | Ритмическая основа, дисторшн |
| **Электронная драмка** | Гибрид живой и электронной |
| **Атмосфера** | Кинематографичная, эпичная |

### Продакшн-подход

1. **Синтезатор на переднем плане** — мелодия часто на синте, а не на гитаре
2. **Гитары как ритм** — palm-mute риффы, не соло
3. **Электронные элементы** — арпеджио, пэды, текстуры
4. **Пространство** — большое, кинематографичное

### Референсы

- Argent (гипотетический стиль)
- Sync — электроника + метал
- Ill Niño — латин-метал с электроникой

## Современный метал: Nu-Metal и Core

### Nu-Metal

| Характеристика | Описание |
|---------------|----------|
| **Бас** | Доминирующий, мелодичный |
| **Гитары** | Пальм-мьют риффы, не соло |
| **Драмка** | Хип-хоп влияние, сложные паттерны |
| **Вокал** | Речитатив + крик |

### Metalcore / Deathcore

| Характеристика | Описание |
|---------------|----------|
| **Брейкдауны** | Медленные, тяжёлые секции |
| **Скальпинг** | Чередование крика и чистого вокала |
| **Гитары** | Очень низкий строй (B, A, G#) |
| **Драмка** | Блейст-биты, сложные филлы |

### Продакшн-различия

| Жанр | Гитары | Бас | Драмка |
|------|--------|-----|--------|
| **Nu-Metal** | Mid-focused, не слишком тяжёлые | Мелодичный, вперёд | Хип-хоп groove |
| **Metalcore** | Тяжёлые, низкий строй | Плотный, склеен с киком | Быстрый, агрессивный |
| **Deathcore** | Максимальный дисторшн | Суб-бас доминант | Блейст + брейкдаун |

### Советы по жанру

- **Nu-Metal:** бас — звезда. Дайте ему пространство в миксе
- **Metalcore:** контраст между быстрыми и медленными секциями
- **Deathcore:** брейкдауны — ключевые моменты. Дайте им «дышать»

!!! tip
    **Не смешивайте подходы.** Продакшн nu-metal'а отличается от deathcore. Определите поджанр до начала сведения — это определит EQ, компрессию и пространство.
