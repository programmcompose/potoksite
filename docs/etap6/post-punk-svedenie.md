# Пост-Панк — Сведение

Сведение пост-панка — это создание атмосферы через пространство. Реверб не эффект, а инструмент.

## Реверб и атмосфера

### Философия сведения

| Принцип | Описание |
|---------|----------|
| **Пространство — это звук** | Реверб определяет жанр |
| **Сухое вперёд, мокрое назад** | Глубина через wet/dry |
| **Меньше = больше** | Минимализм в аранжировке требует воздуха в миксе |

### Настройка реверба

| Параметр | Значение для пост-панка |
|----------|----------------------|
| **Decay** | 2-6 секунд (длинный) |
| **Pre-Delay** | 20-50 мс (чёткость атаки) |
| **Damp** | Средний (не слишком яркий хвост) |
| **Mix** | 20-40% на гитарах, 30-50% на вокале |

## Гитары в миксе

### Подход

- **Центрированные или слегка панорамированные** — не слишком широко
- **EQ:** Cut 300-500 Гц (грязь), boost 2-4 кГц (характер)
- **Компрессия:** Лёгкая, для удержания динамики

### Слои гитар

| Слой | Обработка |
|------|----------|
| **Основная гитара** | Сухая, чёткая, вперёд |
| **Атмосферная гитара** | Много реверба, дилей, на заднем плане |
| **Дрон / пэд** | Постоянный, заваленный ревербом |

## Вокал

### Пост-панк вокал

- **Сухой, близкий** — вокал вперёд, без лишней атмосферы
- **Короткий реверб** — для связности, не для «эфира»
- **Дилей** — слэп-бэк или короткий echo для глубины

## Драмка

- **Сухая, жёсткая** — минимум реверба на кике и снейре
- **Room-микрофон** — для ощущения пространства
- **Компрессия** — средняя, для контроля динамики

## Общий микс

### Баланс

| Инструмент | Уровень | Позиция |
|-----------|---------|--------|
| **Вокал** | Вперёд | Центр |
| **Бас** | Плотный | Центр |
| **Гитара основная** | Средний | L30/R30 |
| **Гитара атмосферная** | Тихий | Широко |
| **Драмка** | Чёткая | Центр |
| **Реверб** | Слышный, но не доминирующий | — |

!!! important
    **В пост-панке реверб — не украшение, а структура.** Без реверба пост-панк звучит как обычный рок. Реверб создаёт ту самую «холодную» атмосферу, которая определяет жанр.
