# Рок — Вариации жанра

Рок — не монолитный жанр. Внутри него существуют поджанры с кардинально разным звучанием. Рассмотрим два ключевых направления.

## Рок нулевых (2000s Rock)

Рок 2000-х — это эпоха пост-гранжа, альтернативного рока и поп-рока.

### Характерные черты

| Элемент | Описание |
|---------|----------|
| **Гитары** | Широкие стерео-панорамы, наслоение кранча и чистого звука |
| **Драмка** | Крупные, «большая» бочка, хлесткий снейр с длинным хвостом |
| **Бас** | Чёткий, с_mid_-частотным присутствием |
| **Вокал** | Компрессированный, яркий, с реверберацией на хвостах |

### Референсы

- Linkin Park — сочетание рока с хип-хопом
- Foo Fighters — классический альтернативный рок
- The Killers — поп-рок с широкой аранжировкой
- Green Day — панк-рок энергия

### Продюсинг через VST

Для эмуляции звука рока нулевых:

1. **Гитары:** Layer из 2-3 гитарных треков с разным панорамированием
2. **Драмка:** Комбинация сэмплов и записи живых ударных
3. **Пространство:** Средний реверб на гитары, длинный на вокал

## Shoegaze (Шугейз)

Shoegaze — атмосферный поджанр, где гитара становится текстурой, а не инструментом ритма.

### Философия шугейза

- **Стена звука** — гитары создают плотную звуковую стену
- **Размытие границ** — инструменты сливаются в единое полотно
- **Эфирность** — много реверба, дилеев, модуляции

### Характерные элементы

| Элемент | Роль в шугейзе |
|---------|---------------|
| **Реверб** | Основной инструмент создания пространства |
| **Хорус** | Утолщение звука, создание «водного» эффекта |
| **Дисторшн** | Не агрессивный, а текстурирующий |
| **Фидбэк** | Контролируемая обратная связь как инструмент |

### Продюсинг шугейза

- Используйте **много слоёв** гитар с разной обработкой
- **Реверб** — ключевой эффект. Не бойтесь перегружать
- **Мелодии** — простые, повторяющиеся, гипнотические
- **Динамика** — нарастание от тихого к плотной стене звука

### Референсы

- My Bloody Valentine — «Loveless» — эталон жанра
- Slowdive — более мелодичный шугейз
- Alcest — шугейз с элементами блэк-метала

!!! tip
    **Shoegaze в DAW:** секрет «стены звука» — не в количестве эффектов на одном треке, а в наложении множества треков с разными эффектами. Каждый слой добавляет свою текстуру.
