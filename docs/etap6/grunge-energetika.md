# Гранж — Энергетика

Гранж — это не звук, а ощущение. Холод, сырость, эмоциональное давление.

## Холод и сырость

### Что создаёт «холодный» звук

| Элемент | Как достичь |
|---------|-----------|
| **Минорные тональности** | Em, Am, Dm — тёмная палитра |
| **Медленный темп** | 70-90 BPM, тягучесть |
| **Длинные хвосты** | Реверб 3-5 секунд |
| **Низкочастотное давление** | Дрон, суб-бас |

### Сырой характер

- **Не полируйте** — артефакты добавляют характер
- **Грязный шум** — контролируемый background noise
- **Неидеальный тайминг** — humanize ±20 мс

## Эмоциональное напряжение

### Динамика как инструмент

| Секция | Энергия | Обработка |
|--------|---------|----------|
| **Купол** | Тихая, сдержанная | Чистая гитара, минимум эффектов |
| **Припев** | Взрывная | Дисторшн, все инструменты |
| **Бридж** | Напряжение | Постепенное нарастание |
| **Финал** | Максимум | Всё на полную, фидбэк |

### Quiet-Loud-Quiet

Классическая гранж-формула:

```
Тихий купол → Громкий припев → Тихий купол → Громкий припев
```

Этот контраст — ДНК гранжа. Nirvana закрепили этот подход.

## Энергия в продакшне

### Практические приёмы

1. **Компрессия на мастер-бусе** — для «склеивания» энергии
2. **Сатурация на BUS** — общая грязь для всех инструментов
3. **Automation громкости** — ручное управление динамикой секций
4. **Fill и транзишены** — креши, фидбэк, шум между секциями

### Как проверить энергию трека

- Закройте глаза и послушайте: вы чувствуете **контраст** между секциями?
- Трек «дышит»? Есть ли моменты напряжения и разрядки?
- Припев «бьёт» сильнее куплета?

!!! important
    **Энергия гранжа — в контрасте, а не в громкости.** Тихий купол делает громкий припев мощным. Без тишины нет громкости.
