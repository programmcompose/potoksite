# Гранж — Гаражный звук

Гранж — это звук гаража, а не студии. Грязь, сырой характер, необработанный драйв.

## Философия гаражного звука

### Что такое «гаражный» звук

| Характеристика | Описание |
|---------------|----------|
| **Грязь** | Контролируемый шум, перегруз, артефакты |
| **Неидеальность** | Микро-ошибки, которые добавляют характер |
| **Плотность** | Звуки «склеены» перегрузом |
| **Сырой характер** | Минимум полировки, максимум эмоций |

## Как достичь гаражного звука в DAW

### Сатурация и перегруз

| Плагин | Назначение |
|--------|-----------|
| **Decapitator (Soundtoys)** | Гибкая сатурация, от лёгкой до жёсткой |
| **Camridge (Klanghelm)** | Бесплатный перегруз, «грязный» характер |
| **Distressor (Fairchild)** | Компрессия с перегрузом |
| **Saturator (Ableton)** | Встроенный, versatile |

### Практика

1. **Не бойтесь перегруза** — гранж живёт на грани клиппинга
2. **Parallel distortion** — дублируйте трек, перегружайте, микшуйте с оригиналом
3. **Lo-fi эффекты** — биткраш, тап-хеад дилей для «кассетного» звука

## Гитары в гранже

### Подход к гитарному звуку

- **Массивный дисторшн** — не crunch, а полноценный перегруз
- **Низкий строй** — Drop D, Drop C для тяжести
- **Пальм-мьют риффы** — ритмическая основа гранжа

### Структура гитарного слоя

| Слой | Роль |
|------|------|
| **Основной дисторшн** | Тяжёлый, плотный рифф |
| **Чистая вставка** | Контраст в бридже или куплете |
| **Фидбэк / шум** | Атмосфера, переходы |

!!! tip
    **Гранж — это динамика между тишиной и громкостью.** Куплет тихий и чистый, припев — взрыв дисторшна. Этот контраст — ДНК жанра.

## Референсы

- Nirvana — «Nevermind» — эталон гранж-продакшна
- Soundgarden — тяжёлый, сложный гранж
- Alice in Chains — тёмный, мелодичный гранж
- Pearl Jam — более «чистый» гранж
