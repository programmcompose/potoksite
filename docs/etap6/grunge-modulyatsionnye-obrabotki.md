# Гранж — Модуляционные обработки

Модуляционные эффекты придают гранжу характер: толщину, движение, нестабильность.

## Дисторшн и Овердрайв

### Типы перегруза

| Тип | Характер | Применение |
|-----|---------|-----------|
| **Overdrive** | Лёгкий, тёплый | Ритм-гитара, купол |
| **Distortion** | Жёсткий, плотный | Припев, тяжёлые риффы |
| **Fuzz** | Размытый, «расплывчатый» | Экспериментальные вставки |

### Параллельный дисторшн

Техника, где чистый сигнал миксуется с перегруженным:

1. Дублируйте гитарный трек
2. На один примените дисторшн
3. Микшуйте: 60% clean / 40% dirty (или наоборот)
4. Результат: плотность без потери чёткости

## Хорус

Хорус — ключевой эффект гранжа. Он создаёт «мокрый», размытый характер.

### Настройки для гранжа

| Параметр | Значение |
|----------|---------|
| **Rate** | 0.8-2 Hz |
| **Depth** | 50-70% |
| **Mix** | 30-50% |
| **Voices** | 2-3 |

### Где применять

- **Чистые гитары в куплете** — хорус создаёт атмосферу
- **Бас** — лёгкий хорус для «движения»
- **Вокал** — subtilный хорус на хвостах

## Фазер и Флэнжер

### Фазер

- **Эффект:** «Всасывание» звука, движение частот
- **Применение:** текстурные вставки, переходы
- **Настройки:** медленный rate для subtilного эффекта

### Флэнжер

- **Эффект:** «Летающий», вращающийся звук
- **Применение:** гитарные соло, атмосферные секции
- **Настройки:** feedback для интенсивности

## Tremolo и Vibrato

| Эффект | Что модулирует | Роль в гранже |
|--------|---------------|---------------|
| **Tremolo** | Громкость | Ритмический пульс, напряжение |
| **Vibrato** | ПITCH | Эмоциональность, «плачущий» звук |

!!! tip
    **Модуляция — это не «включил и забыл».** Автоматизируйте параметры: наращайте depth в припеве, меняйте rate в бридже. Движение параметров создаёт движение музыки.
