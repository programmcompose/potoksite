# Работа с реальными инструментами

VST — мощный инструмент, но запись реальных инструментов добавляет уникальность и характер.

## Микрофон

### Базовые понятия

| Тип микрофона | Применение |
|--------------|-----------|
| **Динамический** | Гитарные кабинеты, живая драмка, вокал |
| **Конденсаторный** | Акустические инструменты, детальный звук |
| **Ленточный** | Тёплый, винтажный характер |

### Расположение микрофона

- **Близко (2-5 см)** — прямой звук, минимум комнаты
- **Средне (10-30 см)** — баланс прямой/комната
- **Далеко (1м+)** — комната как инструмент

## Запись гитар

### Провод (DI + Кабинет)

| Способ | Описание |
|--------|---------|
| **DI (Direct Input)** | Прямое подключение, чистый сигнал |
| **Микрофон на кабинете** | Классический способ, реальный звук |
| **Комбинация** | DI + микрофон, максимальная гибкость |

### Звуковуха (Audio Interface)

| Параметр | Что важно |
|----------|----------|
| **Preamp** | Чистота, headroom |
| **Instruments** | Минимум 2 входа для stereo |
| **Latency** | Низкая задержка для мониторинга |
| **AD-конвертер** | Качество оцифровки |

### Популярные интерфейсы

- **Focusrite Scarlett** — доступный, надёжный
- **Universal Audio Apollo** — премиум, DSP-обработка
- **Motu Ultralite** — хороший AD-конвертер

## Вопрос шума

### Заземление

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Гудение (Hum)** | Плохое заземление | Проверьте розетку, используйте ground loop isolator |
| **Писк (Buzz)** | Электромагнитные помехи | Уберите источники помех, экранируйте кабели |
| **Щелчки** | Плохой контакт | Проверьте разъёмы, замените кабели |

### Шум на фоне

- **Тишина в комнате** — выключите вентиляцию, кондиционер
- **Акустическая обработка** — пены, панели, ковёр
- **Blanket-метод** — одеяло над гитарным кабинетом для изоляции

### Как убрать шум post-factum

| Плагин | Назначение |
|--------|-----------|
| **RX (iZotope)** | Профессиональная шумоподавление |
| **Gate (шлюз)** | Отрезает сигнал ниже порога |
| **Noise Reduction (Ableton)** | Встроенный, работает хорошо |
| **De-noise (FL Studio)** | Встроенный в DAW |

## Запись в FL Studio

### Маршрутизация треков

1. **Настройте маршруты** — каждый инструмент на отдельный трек
2. **Input monitoring** — включите для записи с мониторингом
3. **Arm для записи** — красная кнопка на треке
4. **Buffer size** — уменьшите для низкой задержки (128 samples)

### Удаление ненужных тейков

- **Punch In/Out** — запись только нужного фрагмента
- **Take-менеджер** — управление несколькими дублями
- **Trim** — обрезка начала и конца тейка

## Запись в Ableton Live

### Редактирование тейков

1. **Split (Ctrl+E)** — разделение тейка
2. **Fade** — плавное затухание на краях
3. **Warping** — подгонка тайминга
4. **Clip Editing** — точная правка в Clip View

### Советы для Ableton

- **Input Recording** — включите «Monitor to Auto» для записи
- **Warper** — используйте «Beats» warp mode для музыкального материала
- **Take Lanes** — для работы с несколькими дублями

!!! important
    **Запись реальных инструментов — навык, который приходит с практикой.** Не расстраивайтесь, если первые записи будут imperfect. Каждый session — опыт. Главное — слушайте результат и корректируйте подход.
