# Пост-Панк — Основные элементы

Пост-панк — жанр, рождённый из панка, но с большей атмосферностью и экспериментальным подходом.

## Гитары и Мелодии

### Роль гитары в пост-панке

| Характеристика | Описание |
|---------------|----------|
| **Минимализм** | Простые, повторяющиеся риффы |
| **Диссонанс** | Непривычные интервалы, напряжённые аккорды |
| **Ритмическая функция** | Гитара как перкуссия, а не мелодический инструмент |

### Подход к написанию

- **Октавные риффы** — две ноты с разницей в октаву, повторяющиеся
- **Приглушённые аккорды** — palm-mute для ритмической плотности
- **Мелодия через повторение** — гипнотический эффект от простой фразы

## Драмка и сочетание электроники

### Живая драмка

- **Прямой, жёсткий звук** — без излишней обработки
- **Маршеобразный бит** — характерный для пост-панка
- **Минимум филлов** — драмка держит ритм, не украшает

### Электронные элементы

| Элемент | Применение |
|---------|-----------|
| **Драм-машина** | Механический, холодный ритм |
| **Сиквенсор** | Повторяющиеся электронные паттерны |
| **Синтезатор-бас** | Альтернатива бас-гитаре |

### Комбинация

Пост-панк часто смешивает живую и электронную драмку:
- Живая драмка для куполов
- Электронные элементы для припевов и бриджей

## Реверб и Пространство

### Философия пространства

Пост-панк — жанр атмосферы. Пространство не украшение, а **основа звука**.

| Тип реверба | Назначение |
|------------|-----------|
| **Hall** | Большое пространство, эфирность |
| **Room** | Интимность, близость |
| **Plate** | Яркость на вокале и гитарах |

### Как использовать

1. **Много реверба** — не бойтесь «мокрого» звука
2. **Dry/Wet баланс** — гитара сухая вперёд, реверб создаёт глубину
3. **Препреверб** — короткий attack для чёткости
