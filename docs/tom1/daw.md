# DAW — Digital Audio Workstation

DAW (Digital Audio Workstation) — это программа для записи, редактирования,
сведения и мастеринга музыки. Выбор DAW — один из первых и самых важных
решений начинающего продюсера.

## Популярные DAW

| DAW | Платформа | Цена | Лучше всего для |
|-----|-----------|------|-----------------|
| **Ableton Live** | Win / Mac | $74.99–$749 | Электронная музыка, лайв-выступления |
| **FL Studio** | Windows | $99–$529 | Hip-hop, EDM, битмейкинг |
| **Logic Pro** | Mac | $199 (единоразово) | Все жанры, профессиональная работа |
| **Reaper** | Win / Mac / Linux | $60–$225 | Кастомизация, бюджет |
| **Pro Tools** | Win / Mac | $29.99/мес | Профессиональная студия, запись |
| **Studio One** | Win / Mac | $99–$399 | Современный интерфейс, драг-энд-дроп |
| **Cubase** | Win / Mac | $99–$579 | Оркестровка, MIDI-производство |

## Сравнение ключевых DAW

### Ableton Live

> [!TIP]
> Ableton Live — выбор №1 для электронной музыки и лайв-выступлений.

- **Session View** — нелинейная работа с клипами
- **Warping** — лучший встроенный твинг
- Огромная библиотека устройств и эффектов
- Seamless интеграция с Max for Live

### FL Studio

- Паттерн-базированный подход
- Лучший встроенный драм-машина (Fruity Drum Machine)
- Бесплатные обновления при покупке lifetime
- Огромное сообщество и туториалы

### Logic Pro

- Лучшее соотношение цена/качество
- Огромная библиотека звуков (~200 ГБ)
- Нативные плагины высокого качества
- Оптимизация под macOS

### Reaper

> [!NOTE]
> Reaper имеет **бессрочную бесплатную пробную версию**.
> Лицензия стоит $60 (discounted) или $225 (commercial).

- Минимальные системные требования
- Полная кастомизация интерфейса
- Scripting на Lua и Python
- Очень низкая латентность

## Системные требования

| Параметр | Минимум | Рекомендуется |
|----------|---------|--------------|
| **CPU** | Dual-core 2.0 ГГц | Quad-core 3.0+ ГГц |
| **RAM** | 4 ГБ | 16 ГБ+ |
| **SSD** | 128 ГБ | 512 ГБ+ NVMe |
| **OS** | Windows 10 / macOS 11+ | Windows 11 / macOS 14+ |
