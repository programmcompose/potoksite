# Сведение с битом

Вокал и бит должны работать как единое целое. Вокал — вперёд, бит — поддержка.

## На что обращать внимание

### Баланс громкости

| Элемент | Уровень относительно вокала |
|---------|---------------------------|
| **Основной вокал** | 0 dB (референс) |
| **Kick** | -3 to -6 dB от вокала |
| **Snare** | -3 to -5 dB от вокала |
| **Бас** | -4 to -8 dB от вокала |
| **Мелодии** | -6 to -12 dB от вокала |
| **FX** | -10 to -20 dB от вокала |

### Наличие свободного места для вокала

| Техника | Как сделать |
|---------|-----------|
| **Sidechain вокала** | Бит приглушается при появлении вокала |
| **EQ карманы** | Срез 1-3 кГц на битах, где сидит вокал |
| **Динамическая эквализация** | Автоматический срез на битах при вокале |

## Регулировка вокала с миксом

### Чёткость бита и вокала

| Проблема | Решение |
|----------|---------|
| **Вокал теряется** | Поднимите presence (2-5 кГц) на вокале |
| **Бит «машет» вокал** | Компрессия на вокале для удержания уровня |
| **Конфликт в середине** | Dynamic EQ на битах в диапазоне вокала |
| **Мутный микс** | High-pass на всех элементах, кроме kick/bass |

### Работа с плотными битами

| Приём | Описание |
|-------|---------|
| **Мультибанд компрессия на бите** | Сжимайте только конфликтующие частоты |
| **Parallel processing** | Грязный бит на отдельном треке, смешивайте аккуратно |
| **Сведение по секциям** | Куплет — меньше бита, припев — больше энергии |

## Сенды и посылы

### Сатураторы на сендах

| Настройка | Эффект |
|-----------|--------|
| **Parallel saturation** | Громкость и характер без искажения основного сигнала |
| **Mix 10-30%** | Слышно уплотнение, но не искажение |

### Планы ревербов

| Реверб | Назначение |
|--------|-----------|
| **Short (0.8s)** | Клей, близкое пространство |
| **Medium (2s)** | Основной реверб вокала |
| **Long (4-6s)** | Атмосфера, хвост припева |

## «Фишка» в голосе

### Сложный интересный объём

| Техника | Как сделать |
|---------|-----------|
| **Слои ревербов** | 2-3 реверба с разным временем и EQ |
| **Делэй + реверб** | Делэй вперёд, реверб назад — глубина |
| **Filter automation** | Открывайте low-pass на кульминациях |

### Дисторшн и эффекты

| Эффект | Когда использовать |
|--------|-------------------|
| **Дисторшн на припеве** | Энергия, агрессия |
| **Войс крий (vocal fry)** | Атмосферные секции |
| **Phaser/Flanger** | Психоделические моменты |
| **Pitch shift** | Роботизированные акценты |

### Пространство

1. **Сухой вокал** — куплет, интимность
2. **Реверб нарастает** — пре-хорус, напряжение
3. **Полное пространство** — припев, эпичность
4. **Делэй-трейл** — переходы между секциями

## Нюансы жанров

### Поп

| Характеристика | Подход |
|---------------|--------|
| **Громкий вокал** | Вокал вперёд, минимум конкуренции |
| **«Чистый» звук** | Минимум сатурации, чистая компрессия |
| **Полировка** | Soothe2,细致的 EQ, прозрачный реверб |

### Рок

| Характеристика | Подход |
|---------------|--------|
| **Грязный и жирный** | Сатурация, дисторшн, ламповая компрессия |
| **Энергия** | Быстрая компрессия, aggressive attack |
| **Живой реверб** | Hall, Room — естественные пространства |

### Атмосферный

| Характеристика | Подход |
|---------------|--------|
| **Склеивание пространства** | Вокал «в» ревербе, а не «над» ним |
| **Длинный хвост** | 4-8s реверб, low-pass на хвосте |
| **Мягкая компрессия** | LA2A, минимум gain reduction |
