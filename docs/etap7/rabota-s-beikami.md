# Работа с бэками

Бэки (backing vocals) создают объём, глубину и эмоциональное напряжение. Правильно сведённые бэки — разница между «домашней записью» и «студийным треком».

## Яркость и баланс

### Роль бэков

| Тип бэка | Назначение |
|----------|-----------|
| **Harmony (гармония)** | Параллельная мелодия, обогащает гармонию |
| **Unison (унисон)** | Та же мелодия, добавляет плотность |
| **Ad-libs** | Свободные импровизации, акценты |
| **Oohs/Aahs** | Атмосферные вокальные подушки |

### EQ бэков

| Диапазон | Действие | Зачем |
|----------|---------|-------|
| **High-pass 100-150 Гц** | Агрессивный срез | Освобождаем место для основного вокала |
| **2-4 кГц cut** | -2-3 dB | Убираем конфликт с основным вокалом |
| **Low-pass 12-15 кГц** | Мягкий срез | Бэки не должны быть ярче основного вокала |

!!! tip "Правило бэков"
    Бэки всегда **тише и темнее** основного вокала. Если бэки конкурируют с лидом — вы что-то делаете не так.

## Панорама

### Стерео-размещение

| Тип бэка | Панорама |
|----------|---------|
| **Harmony 1 (выше)** | L30-L50 |
| **Harmony 2 (ниже)** | R30-R50 |
| **Unison дабл** | L20 + R20 |
| **Ad-libs** | Широко L60-R60 или по центру для акцентов |
| **Oohs/Aahs** | L70-R70 (максимальная ширина) |

### Слойная панорама

1. **Основной вокал** — центрирован, вперёд
2. **Ближние бэки** (unison) — L20/R20
3. **Гармонии** — L40/R40
4. **Атмосферные бэки** — L70/R70

## Вариативность обработки

### Разные цепочки для разных бэков

| Бэк | Обработка |
|-----|----------|
| **Ближний бэк** | Похожая цепочка на лид, но меньше компрессии |
| **Гармония** | Больше реверба, меньше presence |
| **Ad-libs** | Больше эффектов, делэи, дисторшн для акцентов |
| **Атмосферный бэк** | Максимум реверба, low-pass фильтр |

### Делэй на бэках

| Настройка | Эффект |
|-----------|--------|
| **Slap delay** (100-150 ms) | Уплотнение, «приближение» бэка |
| **Tempo-sync 1/8** | Ритмичные повторения |
| **Ping-pong** | Стерео-ширина |

### Реверб на бэках

| Параметр | Значение |
|----------|---------|
| **Больше реверба чем на лиде** | Бэки «уходят назад» в пространстве |
| **Отдельный реверб return** | Контроль пространства бэков независимо |
| **EQ реверба бэков** | Агрессивный high-pass (300-500 Гц) |

## Важность ритмики

### Попадание бэков

| Проблема | Решение |
|----------|---------|
| **Бэки не по сетке** | Вырежьте и сдвиньте, используйте warp |
| **Бэки опережают лид** | Сдвиньте бэки на 10-30 ms назад |
| **Бэки отстают** | Сдвиньте вперёд или перепишите |

### Слои и плотность

1. **Куплет** — минимальные бэки, один слой гармонии
2. **Припев** — полная раскладка, 3-4 слоя бэков
3. **Бридж** — атмосферные бэки, Oohs/Aahs
4. **Финал** — максимум слоёв, ad-libs
