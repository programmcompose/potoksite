# Редакция вокала

Перед обработкой вокал нужно отредактировать: выровнять ритмику, подтянуть ноты, убрать шум.

## Ритмика

### Попадание по сетке

| Проблема | Решение |
|----------|---------|
| **Слова сдвинуты по времени** | Сдвиньте клипы в DAW вручную |
| **Сибилянты (эсы) не по сетке** | Вырежьте и переместите проблемные слоги |
| **Общий сдвиг фразы** | Используйте warp / elastic audio для растяжки |

### Нарезка вокала

1. **Вырежьте паузы** между фразами — чистый вокал без «междустрочного шума»
2. **Оставьте дыхание** где оно добавляет экспрессии
3. **Fade-in/out** на каждом клипе — убирает щелчки на стыках

!!! warning "Не перережьте"
    Слишком агрессивная нарезка убивает естественность вокала. Дыхание и микро-паузы — часть живой подачи.

### Автоматизация громкости

| Зачем | Как |
|-------|-----|
| **Выровнять динамику фраз** | Автоматизируйте volume на каждом слове/фразе |
| **Приглушить перегрузы** | Понижайте громкость на пиковых участках |
| **Подчеркнуть акценты** | Поднимайте громкость на ключевых словах |

**В FL Studio:** Используйте automation clip для volume каждого вокального трека.

**В Ableton:** Используйте Envelope Automation (H-key) на device chain.

## Тюнинг вокала

### Melodyne

| Параметр | Описание |
|----------|---------|
| **Pitch** | Точная настройка каждой ноты |
| **Timing** | Сдвиг ноты по времени |
| **Dynamics** | Громкость отдельной ноты |
| **Formant** | Сохранение тембра при изменении питча |

**Подход:**
1. Импортируйте аудио в Melodyne
2. Просканируйте проблемные ноты
3. Корректируйте **постепенно** — 10-20 центов за раз
4. Сохраняйте **натуральность** — лёгкий детюн добавляет жизни

### Antares Auto-Tune

| Режим | Описание |
|-------|---------|
| **Retune Speed: Fast (0-5)** | Явный «роботизированный» эффект (trap, pop) |
| **Retune Speed: Medium (10-20)** | Естественный тюн, не слышен |
| **Retune Speed: Slow (30+)** | Плавные переходы, минимальная коррекция |

### Настройка скорости тюна

| Скорость | Жанр | Эффект |
|----------|------|--------|
| **0-5 (быстрый)** | Trap, Hyperpop | Слышный Auto-Tune эффект |
| **10-20 (средний)** | Поп, R&B | Невидимая коррекция |
| **30+ (медленный)** | Рок, акустика | Минимальное вмешательство |

### Ручной тюн vs Автоматический

| Метод | Плюсы | Минусы |
|-------|-------|--------|
| **Melodyne (ручной)** | Точный контроль, естественность | Требует времени |
| **Auto-Tune (авто)** | Быстро, предсказуемо | Может звучать «пластмассово» |
| **Комбинация** | Auto-Tune для gross-correction + Melodyne для деталей | Больше работы |

## Очистка от шума

### Типы шума

| Шум | Источник | Решение |
|-----|---------|---------|
| **Фоновый гул** | Кондиционер, вентилятор | RX 11 Voice De-noise |
| **Попы** | Ударные согласные | Поп-фильтр при записи, De-plosive в RX |
| **Щелчки** | Сухие губы, зубы | De-click в RX |
| **Реверб комнаты** | Плохая акустика | De-reverb в RX |

### RX 11 — Audio

| Модуль | Назначение |
|--------|-----------|
| **Voice De-noise** | Удаление фонового шума |
| **De-reverb** | Уменьшение хвоста комнаты |
| **De-plosive** | Устранение «бахов» |
| **De-click** | Удаление щелчков |
| **Spectral De-noise** | Точечное удаление шумов в спектре |

### AI-сервисы

| Сервис | Что делает |
|--------|-----------|
| **iZotope RX AI Assistant** | Автоматический анализ и исправление |
| **Acon Digital Acoustica** | AI-очистка вокала |
| **Landr Mastering** | Базовая обработка вокала онлайн |

### Работа с референсами при чистке

1. **Сравните спектр** чистого референса и вашего вокала
2. **Ищите лишние частоты** — гул, резонансы комнаты
3. **Используйте спектральный дисплей** в RX для точечной чистки
