# FL Studio и Ableton: Запись трека с вокалом

Практическое руководство по созданию полного трека с вокалом в двух DAW.

## FL Studio

### Запись вокала

| Шаг | Действие |
|-----|---------|
| **1. Настройка** | Add type → Microphone, выберите аудио-интерфейс |
| **2. Gain staging** | Протестируйте уровень, целевой пик -12 dBFS |
| **3. Арранжировка** | Расставьте секции: куплет, припев, бридж |
| **4. Запись** | Record arm → Rec → записывайте по секциям |
| **5. Comping** | Take selector для выбора лучших дублей |

### Подстройка под BPM

| Инструмент | Описание |
|-----------|---------|
| **Newtone** | Автоматический тюнинг в FL Studio |
| **Warping** | Elastic Audio для ритмической коррекции |
| **CapCut** | Нарезка аудио по транзиентам |

### Цепочка обработки в FL

1. **EQ (Fruity Parametric EQ 2)** — high-pass, срезы, яркость
2. **Maximus** — компрессия (или сторонний плагин)
3. **Fruity Blood Overdrive** — сатурация
4. **Fruity Love Philter** — деессинг
5. **Fruity Reeverb 2** — пространство
6. **Fruity Delay 3** — повторения

## Ableton Live

### Запись вокала

| Шаг | Действие |
|-----|---------|
| **1. Настройка** | Create audio track, armed, выберите вход |
| **2. Monitoring** | Switch to «In» для мониторинга |
| **3. Warping** | Включите warping для синхронизации с BPM |
| **4. Запись** | Record button → записывайте по секциям |
| **5. Comping** | Compile takes в Arrangement View |

### Подстройка под BPM

| Инструмент | Описание |
|-----------|---------|
| **Warping** | Автоматическая синхронизация с темпом |
| **Complex Pro** | Режим варпинга для вокала |
| **Elastic Audio** | Ручная коррекция ритмики |

### Цепочка обработки в Ableton

1. **EQ Eight** — high-pass, срезы, яркость
2. **Compressor** или **Glue Compressor** — компрессия
3. **Saturator** — сатурация
4. **DeEsser** — сибилянты
5. **Reverb** — пространство
6. **Echo** или **Simple Delay** — повторения

## Разница между FL Studio и Ableton

| Аспект | FL Studio | Ableton Live |
|--------|-----------|-------------|
| **Запись вокала** | Channel rack, Take selector | Audio track, Compile takes |
| **Warping** | Newtone, manual slicing | Complex Pro, автоматический |
| **Автоматизация** | Automation clips | Envelope automation (H-key) |
| **Сенды** | Send tracks в mixer | Return tracks |
| **Тюнинг** | Newtone (встроенный) | Нужен сторонний (Melodyne, Auto-Tune) |
| **Workflow** | Визуальный, паттерн-ориентированный | Линейный, аудио-ориентированный |

## Поиск подачи

### Работа с артистом

1. **Обсудите референсы** — какой стиль подачи
2. **Проба мотивов** — запишите демо-фразы
3. **Эксперимент с ритмикой** — разные расклады слов
4. **Выберите лучшую подачу** — comping из разных дублей

### Проба мотивов и ритмики

| Техника | Описание |
|---------|---------|
| **Freestyle запись** | Артист импровизирует, записываете всё |
| **Мелодические вариации** | 3-4 варианта мелодии на одну фразу |
| **Ритмические расклады** | Одна мелодия, разная ритмика |
| **Ad-lib сессия** | Свободные вокальные акценты поверх основного вокала |

## Практический проект

### Запись трека с нуля

1. **Создайте бит** — kick, snare, хэты, бас, мелодия
2. **Напишите структуру** — куплет, припев, бридж
3. **Запишите основной вокал** — несколько дублей
4. **Отредактируйте** — ритмика, нарезка, тюн
5. **Обработайте** — EQ, компрессия, сатурация, пространство
6. **Запишите бэки** — гармонии, ad-libs
7. **Сведите с битом** — баланс, sidechain, эффекты
8. **Мастеринг** — LUFS, чек на устройствах
9. **Рендер** — финальный трек

!!! tip "Совет"
    Начните с простого: один вокал + минимальный бит. Когда освоите базу, усложняйте: бэки, эффекты, слои.
