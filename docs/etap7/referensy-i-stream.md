# Работа с референсами и Стрим

Анализ профессиональных треков — лучший способ улучшить своё сведение.

## Стили в сведении вокала

### Разбор профессионального микса

| Что анализировать | Инструмент |
|------------------|-----------|
| **Спектр** | RTA анализатор, Youlean |
| **Громкость** | LUFS метр, ADPTR Meter |
| **Стерео-изображение** | Стерео-метр, имиджер |
| **Динамика** | RMS пик метр |

## Повторение обработок из паков

### Как учиться на чужих пресетах

1. **Загрузите пресет** в свой проект
2. **Слушайте разницу** A/B с оригиналом
3. **Анализируйте цепочку** — какой плагин что делает
4. **Адаптируйте под свой вокал** — не копируйте слепо

## Лучшая практика для сведения

### Чек-лист профессионального сведения

- [ ] Запись чистая, без клиппинга
- [ ] Редакция: ритмика, нарезка, тюн
- [ ] Очистка: шум, щелчки, реверб комнаты
- [ ] EQ: high-pass, срез мутности, яркость
- [ ] Компрессия: контроль динамики, 2-3 ступени
- [ ] Сатурация: плотность и характер
- [ ] DeEsser: сибилянты под контролем
- [ ] Пространство: реверб и делэй
- [ ] Бэки: панорама, баланс, эффекты
- [ ] Сведение с битом: баланс, sidechain
- [ ] Мастер: LUFS, true peak, чек на устройствах

## Использование SpectraLayers / Stem Separator

### Разделение треков

| Инструмент | Что делает |
|-----------|-----------|
| **SpectraLayers** | Разделяет трек на стемы: вокал, бит, бас, остальные |
| **Stem Separator** | AI-разделение на компоненты |
| **Demucs** | Бесплатный AI-сепаратор |

### Зачем разделять референсы

1. **Изучить EQ вокала** — посмотрите спектр отдельно
2. **Проанализировать компрессию** — как сжат динамически
3. **Понять пространство** — сколько реверба и делэя
4. **Сравнить громкость** — уровень вокала относительно бита

## ADPTR Meter

### Что такое ADPTR Meter

Бесплатный плагин для анализа микса:

| Функция | Описание |
|---------|---------|
| **Spectrum Analyzer** | Частотный анализ в реальном времени |
| **Loudness Meter** | LUFS измерение |
| **Stereo Imaging** | Визуализация стерео-поля |
| **Phase Correlation** | Проверка моно-совместимости |

### Как использовать

1. **Повесьте на мастер-канал**
2. **Сравните свой микс** с референсом
3. **Ищите различия** в спектре и громкости
4. **Корректируйте** до совпадения характеристик

## Стрим: разбор треков-референсов

### Выбор треков для анализа

| Жанр | Что слушать |
|------|-----------|
| **Поп** | Чистота вокала, прозрачность, полировка |
| **Trap/Hip-Hop** | Громкость вокала, пространство, 808 |
| **Рок** | Грязь, энергия, живые инструменты |
| **Электроника** | Ширина, бас, FX |

### План разбора

1. **Первый прослуш** — общее впечатление
2. **Спектральный анализ** — какие частоты доминируют
3. **Громкость** — LUFS, peak, RMS
4. **Стерео-изображение** — ширина элементов
5. **Поэлементный анализ** — вокал, бит, бас отдельно
6. **Выводы** — что применить в своём миксе
