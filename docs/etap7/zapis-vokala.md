# Запись вокала

Качественная запись — 50% хорошего сведения. Плохо записанный вокал невозможно «спасти» плагинами.

## Подготовка к записи

### Комната

| Параметр | Что делать |
|----------|-----------|
| **Акустика** | Мягкие поверхности, ковры, шторы уменьшают отражения |
| **Шум фона** | Выключите кондиционер, вентиляцию, закройте окна |
| **Изоляция** | Даже шкаф с одеждой работает как временная звукоизоляция |

### Настройка уровня

1. **Попросите артиста пропеть самую громкую часть** трека
2. **Установите gain** так, чтобы пиковый уровень был около **-12 dBFS**
3. **Не допускайте клиппинга** — перегруз на входе не исправить

!!! tip "Правило headroom"
    Оставляйте 6-10 dB headroom между пиковым уровнем и 0 dBFS. Это даёт пространство для обработки без перегруза.

## Выбор микрофона

### Типы микрофонов для вокала

| Тип | Характер | Примеры |
|-----|---------|---------|
| **Конденсаторный** | Чистый, детальный, чувствительный | Neumann U87, AKG C414, Rode NT1 |
| **Ленточный** | Тёплый, винтажный, мягкий | Royer R-121, sE RPC6 |
| **Динамический** | Прочный, менее чувствительный к шуму | Shure SM7B, Electro-Voice RE20 |

### Бюджетные варианты

| Микрофон | Цена | Для чего подходит |
|----------|------|-------------------|
| **AT2020** | ~$100 | Начало, домашняя студия |
| **Rode NT1** | ~$150 | Чистый конденсатор |
| **Shure SM7B** | ~$400 | Грязный, плотный вокал (рэп, рок) |

## Установка микрофона

### Расстояние и направление

| Расстояние | Эффект |
|-----------|--------|
| **5-10 см** | Близкий эффект (proximity effect): больше низких частот, интимный звук |
| **15-25 см** | Баланс между близким эффектом и естественным звучанием |
| **30+ см** | Более «комнатный» звук, больше реверберации помещения |

### Поп-фильтр

- **Обязателен** для вокала — снижает ударные согласные (П, Б, Д)
- Устанавливается **между ртом и микрофоном** на расстоянии 5-10 см
- Альтернатива: направляйте микрофон **под углом 30-45°** от прямой линии рта

### Настройка в DAW

| Параметр | Значение |
|----------|---------|
| **Buffer size** | 128-256 сэмплов (низкий латент для мониторинга) |
| **Direct Monitor** | Включите, если буфер маленький |
| **Gain staging** | Целевой пик -12 dBFS |

## Техника записи

### Работа с артистом

1. **Разминка голоса** — 5-10 минут перед записью
2. **Демонстрация** — покажите пример подачи, если нужно
3. **Несколько дублей** — всегда записывайте 3-5 проходов
4. **Комплименты** — мотивация улучшает результат

### Компиляция (comping)

После записи соберите лучшие фрагменты из разных дублей в один «идеальный» трек.

| DAW | Инструмент |
|-----|-----------|
| **FL Studio** | Take editor в Playlist |
| **Ableton** | Compile takes в Arrangement View |
