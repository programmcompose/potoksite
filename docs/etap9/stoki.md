# Стоки — Пассивный доход от музыки

Стоковые площадки — это маркетплейсы, где вы можете продавать музыку, биты, звуковые эффекты и саундтреки. Это **пассивный доход**: загрузили один раз — продаёте многократно.

## BeatStars — Продажа битов

BeatStars — крупнейшая платформа для продажи битов в мире.

### Как начать

| Шаг | Действие |
|-----|----------|
| 1 | Зарегистрируйтесь на [BeatStars](https://www.beatstars.com/) |
| 2 | Заполните профиль — фото, описание, жанры |
| 3 | Загрузите биты с правильными тегами |
| 4 | Установите цены и типы лицензий |
| 5 | Подключите платёжную систему (PayPal, Payoneer) |

### Типы лицензий

| Лицензия | Что включает | Цена (пример) |
|----------|-------------|---------------|
| **Basic Lease** | MP3, личное использование | $29.99 |
| **Premium Lease** | WAV + MP3, до 100K продаж | $49.99 |
| **Unlimited Lease** | WAV + stems, неограниченно | $99.99 |
| **Exclusive Rights** | Полные права, только один покупатель | $300-$2000+ |

### Советы по продаже на BeatStars

- **Обновляйте каталог регулярно** — минимум 2-3 бита в неделю
- **Используйте правильные теги** — жанр, настроение, BPM, тональность
- **Создавайте привлекательные обложки** — визуал влияет на клик
- **Работайте с отзывами** — положительные отзывы повышают доверие

!!! tip
    BeatStars интегрируется с YouTube, SoundCloud и Instagram. Используйте **BeatStars Player** для встраивания плеера на свои площадки.

## Pond5 — Стоковая музыка

Pond5 — одна из крупнейших площадок для стокового контента: музыка, видео, изображения, звуковые эффекты.

### Что продавать

| Категория | Примеры | Спрос |
|-----------|---------|-------|
| **Корпоративная музыка** | Фон для презентаций, реклам | Высокий |
| **Кино и ТВ** | Саундтреки, атмосферные треки | Средний |
| **Подкасты** | Интро, аутро, фоновая музыка | Растущий |
| **Звуковые эффекты** | SFX, переходы, атмосфера | Высокий |

### Как заработать на Pond5

1. **Создавайте универсальный контент** — корпоративная музыка продаётся стабильно
2. **Оптимизируйте описания** — ключевые слова для поиска
3. **Загружайте в высоком качестве** — WAV 24-bit минимум
4. **Будьте последовательны** — чем больше каталог, тем больше продаж

## AudioJungle — Музыка для веб-контента

AudioJungle (часть Envato Market) — площадка для музыки, используемой в веб-проектах, видеоиграх, мобильных приложениях.

### Особенности

| Параметр | Описание |
|----------|----------|
| **Эксклюзивность** | Можно выбрать эксклюзивную продажу (выше процент) |
| **Процент продавца** | 50% (неэксклюзив) / 70% (эксклюзив) |
| **Формат** | WAV, MP3, stems по запросу |
| **Категории** | Музыка, звуковые эффекты, подкасты |

### Что хорошо продаётся

- **Корпоративные треки** — upbeat, motivational, corporate
- **Эмбиент и лоу-фай** — для фона видео и подкастов
- **Электроника** — для игр и приложений
- **Акустика** — для YouTube-видео и презентаций

## Сравнение площадок

| Площадка | Аудитория | Порог входа | Потенциал дохода |
|----------|-----------|-------------|------------------|
| **BeatStars** | Артисты, битмейкеры | Низкий | Средний-высокий |
| **Pond5** | Режиссёры, маркетологи | Средний | Средний |
| **AudioJungle** | Веб-разработчики, дизайнеры | Средний | Средний-высокий |
| **Epidemic Sound** | Контент-мейкеры | Высокий (по заявке) | Высокий |
| **Artlist** | Видеомейкеры | Высокий (по заявке) | Высокий |

## Стратегия пассивного дохода

### Комбинированный подход

Не полагайтесь на одну площадку. **Диверсифицируйте**:

1. **BeatStars** — для битов и хип-хоп продакшна
2. **Pond5** — для универсальной стоковой музыки
3. **AudioJungle** — для веб-контента и игр
4. **Собственный сайт** — для прямых продаж

### Сколько можно заработать

| Уровень | Треков в каталоге | Месячный доход (пример) |
|---------|-------------------|------------------------|
| **Начинающий** | 10-20 треков | $50-$200 |
| **Средний** | 50-100 треков | $300-$1000 |
| **Продвинутый** | 200+ треков | $1000-$5000+ |

!!! important
    **Пассивный доход — не мгновенный.** Первые продажи могут прийти через 3-6 месяцев. Ключ — количество и качество каталога.
