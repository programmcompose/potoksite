# Социальные сети — Продвижение через платформы

Соцсети — ваш главный инструмент продвижения в 2024-2026 годах. Алгоритмы работают на вас, если вы понимаете их логику.

## Карта платформ

| Платформа | Аудитория | Формат контента | Для чего лучше |
|-----------|-----------|-----------------|----------------|
| **VK** | Россия, СНГ | Посты, аудио, видео, сообщества | Локальная аудитория, паблики по темам |
| **TikTok** | Глобальная | Короткие видео (15-60 сек) | Вирусный охват, молодой аудиторий |
| **YouTube** | Глобальная | Длинные видео, Shorts | Обучение, процесс работы, биты |
| **Telegram** | Россия, СНГ | Посты, аудио, каналы | Личный бренд, прямое общение |
| **SoundCloud** | Глобальная | Аудио | Портфолио битов, демо, коллабы |

## VK — ВКонтакте

### Паблики по темам

VK — самая сильная платформа в России для музыкального контента. Стратегия:

1. **Найдите паблики**, где ваша целевая аудитория уже сидит
2. **Не спамьте ссылками** — создавайте полезный контент
3. **Сарафанное радио** — просите друзей и коллег поделиться

### Как работать с VK

- **Создайте сообщество** с вашим ником/брендом
- **Публикуйте:** процесс создания битов, таймлапсы, советы по продакшну
- **VK Музыку** — загружайте треки через дистрибьютора (они подключены к VK)
- **Аудиозаписи** в паблике — короткие фрагменты битов для привлечения внимания

!!! tip
    VK отлично работает для **локального рынка**. Если вы из России или СНГ — это ваша основная площадка для нетворкинга.

## TikTok

### Российский TikTok vs Зарубежный

| Аспект | Российский TikTok | Зарубежный TikTok |
|--------|-------------------|-------------------|
| **Аудитория** | RU, BY, KZ, UZ | Глобальная |
| **Язык контента** | Русский | Английский |
| **Тренды** | Локальные мемы, рэп, поп | Глобальные тренды, hip-hop, electronic |
| **Монетизация** | Сложнее (ограничения) | Creator Fund, бренды, мерч |

### Стратегия для битмейкеров

1. **Показывайте процесс** — таймлапсы создания бита, «до/после» сведения
2. **Звук — это король** — используйте свои биты как звук для других видео
3. **Образовательный контент** — «как сделать 808», «секреты сведения»
4. **Тренды** — адаптируйте тренды под музыкальную тематику

## YouTube

### Длинные видео и Shorts

**YouTube Shorts** — аналог TikTok, но с преимуществом: контент остаётся на канале и работает на SEO.

| Формат | Что снимать | Частота |
|--------|-------------|---------|
| **Shorts** | Быстрые советы, «секрет за 30 сек», фрагменты битов | 3-5 в неделю |
| **Длинные видео** | Разбор бита, процесс сведения, обзор плагинов | 1-2 в неделю |
| **Биты и инструменталы** | Полные треки с визуалом | По мере выпуска |

### пример: You Suck at Producing

Канал, который построили на **честных разборах продакшна**. Они не показывают «идеальный процесс» — они показывают реальный, с ошибками и исправлениями.

- **Подписчики:** 1M+
- **Стратегия:** честность + юмор + полезный контент
- **Монетизация:** Patreon, мерч, курсы

## Telegram (Запретограм)

### Создание канала

Telegram — идеальная платформа для **прямой связи с аудиторией**:

1. **Создайте канал** — ваши мысли, процесс работы, новости
2. **Создайте чат** — для общения с подписчиками
3. **Регулярный контент** — минимум 3 поста в неделю

### Что постить

- **Процесс работы** — скриншоты проектов, мысли о звуке
- **Советы** — короткие лайфхаки по продакшну
- **Рекомендации** — плагины, сэмпл-паки, книги
- **Личное** — ваши истории, мотивация, неудачи

## SoundCloud

SoundCloud — это **ваше портфолио**. Здесь ищут биты артисты, продюсеры, и другие битмейкеры.

### Стратегия

1. **Загружайте биты регулярно** — минимум 1 в неделю
2. **Тэги** — используйте правильные теги для поиска
3. **Реагируйте на комментарии** — это создаёт комьюнити
4. **Репосты** — делитесь чужими работами, и вас тоже будут репостить

!!! note
    SoundCloud — это не про миллионы прослушиваний. Это про **нахождение ваших людей**: артистов, которые ищут биты, продюсеров, которые хотят коллаб.

## Создание страниц и ведение аккаунтов

### Единый стиль

- **Никнейм** — одинаковый на всех платформах
- **Аватар** — узнаваемый, минималистичный
- **Описание** — кто вы, что делаете, как связаться
- **Ссылки** — перелинковка между платформами

### Контент-план

| День | Платформа | Контент |
|------|-----------|---------|
| Пн | TikTok + Shorts | Таймлапс бита (30 сек) |
| Ср | VK + Telegram | Совет по продакшну |
| Пт | Все платформы | Новый бит / фрагмент трека |
| Вс | YouTube | Разбор процесса (длинное видео) |

### Примеры успешных аккаунтов

- **[Internet Money](https://www.instagram.com/internetmoney/)** — продюсерский коллектив, который строит личный бренд через соцсети
- **[Pi'erre Bourne](https://www.instagram.com/pierrbourne/)** — показывает процесс, делится звуками, вдохновляет
- **[Dora](https://www.tiktok.com/@doratheband)** — продюсер, который использует TikTok для продвижения своих битов
