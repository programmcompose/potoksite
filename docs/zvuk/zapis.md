# Запись инструментов

Качественная запись — 80% качества финального продукта. Плохую запись
невозможно «исправить» на этапе сведения.

## Запись вокала

### Подготовка

1. **Комната** — чем меньше отражений, тем лучше. Шкаф с одеждой,
   палатка для вокала, обработанная комната
2. **Поп-фильтр** — обязательно между губами и микрофоном (10–15 см)
3. **Уровень** — пик на -12 dB, не допускайте клиппинга

!!! warning
    **Никогда не записывайте с клиппингом.** Перезапись лучше, чем
    попытка исправить обрезанный сигнал.

### Техника записи

| Параметр | Рекомендация |
|----------|-------------|
| Дистанция до микрофона | 15–20 см |
| Усиление (gain) | Пик на -12 dB |
| Фантомное питание | +48V (для конденсаторных) |
| Low-cut фильтр | 60–80 Гц (убирает шум кондиционера) |
| Мониторинг | Наушники, не динамики |

## Запись гитары

### Варианты записи

| Метод | Качество | Сложность | Применение |
|-------|----------|-----------|-----------|
| **DI-бокс** | Чистый, гибкий | Низкая | Электрогитара, обработка после |
| **Микрофон → Камертон** | Естественный | Средняя | Акустическая гитара, кабинет |
| **Ампли-симулятор** | Зависит от плагина | Низкая | Быстрый результат |
| **Микрофон + Кабинет + DI** | Максимальное | Высокая | Профессиональная запись |

### Микрофонирование акустической гитары

- Позиция: между 12-м и 14-м ладами, 30–40 см от гитары
- Микрофон: конденсаторный, кардиоида
- Избегайте наведения на звукотвор — будет «бубнение»


## Общие советы

- [ ] Всегда делайте «soundcheck» — записывайте 10 секунд перед сессией
- [ ] Используйте наушники для мониторинга, а не мониторы
- [ ] Записывайте с запасом по уровню (-12 dB пик)
- [ ] Делайте несколько дублей — выбор на этапе сведения
- [ ] Коммуницируйте с музыкантом — дайте обратную связь
