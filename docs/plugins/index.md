# Плагины — подборка для изучения

Здесь собраны подробные обзоры VST-плагинов: компрессоры, эквалайзеры,
ревербераторы, дилеи, сатурация и эффекты.

Каждая статья — это практическое руководство: что делает плагин, где его
ставить, как настраивать и какие есть плюсы/минусы.

## Подборки

| Сборка | Описание |
|--------|----------|
| [Подборка от Meraki](meraki-collection.md) | Компрессоры, EQ, реверб, дилеи, сатурация, эффекты — авторская коллекция |
| [Waves Abbey Road](waves-abbey-road.md) | Эмуляция оборудования Abbey Road Studios: ленты, реверб, компрессоры |
| [Waves CLA](waves-cla.md) | Плагины Chris Lord-Alge: компрессоры, канальные стрипы, эффекты |
| [Waves MannyM](waves-mannym.md) | Плагины Manny Marroquin (9× Grammy): дилеи, EQ, реверб, дисторшн |
| [Waves Maserati](waves-maserati.md) | Плагины Tony Maserati: вокал, бас, ударные, гитары, мастеринг |

> [!TIP]
> Не нужно знать все плагины сразу. Выберите 1–2 из каждой категории
> и изучите их глубоко — это лучше, чем поверхностно пробовать 50 штук.
