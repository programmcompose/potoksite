# Базовые VST-плагины

**Ваша база: (пакеты проще чем искать)**

- **FabFilter Total Bundle** 2020+ годов *(абсолютно лучшие VST, и они нужны вам для любой музыки для хита)*
  Ваша база в огромном пакете от разработчика — [найти плагины](https://t.me/+MhltD1Tlu1FjZmY6)

- **Kilohearts - Complete Bundle** ✅ *(идея для экономии: Kilohearts Ultimate, это почти всё, но если хотите максимум — вот этот пакет для продюсеров-перфекционистов, а ещё он бесплатный, потому плагины сами себя окупают)*
  Ваша база в огромном пакете от разработчика — [найти плагины](https://t.me/+MhltD1Tlu1FjZmY6)

- **Valhalla DSP** (ревербы, мультифункциональные плагины, бесплатные плагины нужны во всех DAW)

> **Важно:** Если **нет** опыта в работе со звуком, и вы только начали, вам не нужно знать каждый плагин, достаточно выучить базовые и не перегружать себя их в DAW (например для начального этапа) будет достаточно, чтобы вы не стали перегружены. Но по мере накопления опыта, постепенно добавляйте новые плагины, чтобы в ваш арсенал, как инструменты для выжигания из звука максимум.

**Примечание:** Ниже будет подробный перечень/описание плагинов из всех пакетов перечисленных выше плагинов.

## 1. Динамическая обработка

Для контроля громкости и/или динамического диапазона звука.

![FabFilter Pro-C 2](https://www.fabfilter.com/img/products/pro-c-2-intro.jpg?v=1710150538)

- Компрессоры в динамике — плагины для контроля громкости в динамике звука. *(FabFilter Pro C 2, Khs Compressor, Khs Dynamics)*
- Лимитеры в динамике — плагины для ограничения максимальной громкости сигнала. *(FabFilter Pro L 2, Khs limiter)*
- Экспандеры в динамике — плагины для усиления разницы в динамике сигнала. *(FabFilter Pro G)*
- Нойз-гейт (Noise Gate) в динамике — плагины для подавления фоновых шумов (особенно для записи вокала). *(тоже из FabFilter Pro G)*

---

## 2. Тональная коррекция ✅

Коррекция тембра и балансировка частотного спектра.✅

![Тональная коррекция](https://global.discourse-cdn.com/flex030/uploads/mixmasterforum/optimized/1X/2a092e5cf802b951947bc47963e24ac824c10309_2_690x457.png)

- Эквалайзеры (EQ) в динамике — плагины для коррекции частотного спектра (басы, средние, высокие). *(FabFilter Pro Q 3, Khs Carve EQ, Khs 3 Band EQ, Khs Slice EQ)*
- Фильтры — плагины для обрезки или выделения определённых частот (HPF, LPF, Band-pass). *(**FabFilter Pro Q 3, khs Ladder Filter, Khs Nonlinear Filter**, также очень полезные плагины Khs Comb Filter, Khs Filter, Khs Formant Filter, FabFilter Pro Micro, FabFilter Pro Simplon, FabFilter Pro Volcano 3)*
- Сатурация в динамике — плагины для добавления тёплого звучания. *(есть сатурация как в пакете Khs так и FabFilter, но не менее интересными будут RC-20, Type-A)*
- Стерео — плагины для управления шириной и балансом стереополы. *(FabFilter Pro DS)✅*

---

## 3. Пространственные эффекты ✅

Создание глубины пространства и атмосферы.✅

![Пространственные эффекты](https://prosound.ixbt.com/news/2023/november/03/image4b.jpg)

- Реверберация (Reverb) в динамике — плагин для создания эффекта пространства (имитация комнаты, зала, плэйн). *(FabFilter Pro R, Khs Convolver, очень интересны все плагины Valhalla DSP)*
- Задержка (Delay) в динамике — плагин для создания эффекта эхо и повторений звука. *(из FabFilter Pro Timeless 3, но обычно достаточно встроенного Delay в DAW, опять же вот и встроенные Delay, Echo, Fruity Delay и т.д.)*
- Хорус (Chorus) в динамике — плагин для создания эффекта "хора" звука, часто используется для утолщения и обогащения тембра гитар, клавиш и синтезаторных звуков. *(Khs Chorus и очень интересный Khs Ensemble)*
- Фленджер (Flanger) в динамике — плагин для создания эффекта "самолёта" или "пролёта". *(Khs Flanger)*
- Фейзер (Phaser) в динамике — плагин для создания эффекта фазового сдвига. *(Khs Phaser)*
- Панорамирование (Panning) в динамике — распределение звука в стереополе. *(встроено в микшерной консоли любой DAW, но если нужен более 3D панорамирование то вот плагин [DearVR 2](https://www.dear-reality.com/products/dearvr-pro-2))*

---

## 4. PITCH и обработка вокала

Коррекция высоты тона и обработка вокала.✅

![Pitch Shifter](https://kilohearts.com/resources/media_dynamic/offers/pitch_shifter/thumb_900.png)

- Пич-шифтеры в динамике — плагины для изменения высоты тона (часто для эффектов). *(Khs Pitch Shifter и обычно встроенный в вашу DAW)*
- Тюнеры — плагины для точной настройки высоты тона. *(в FL STUDIO и ABLETON есть встроенные тюнеры, если вы хотите больше, вот плагин от Kilohearts)*
- Гармонизаторы — плагины для создания гармоний из одного голоса. *(для того чтобы создать из одного голоса гармонию вот плагины [CHROMA](https://www.xynth.audio/plugins/chroma), [WAVES HARMONY](https://www.waves.com/plugins/waves-harmony), [Vocal Harmonizer Plugin](https://www.antarestech.com/products/vocal-effects/harmony-engine) и т.д.)*
- Тюнеры (Auto-Tune) — автоматическая коррекция высоты тона, часто для характерного эффекта. *(в составе пакета плагинов WAVES есть отличный тюнер, называется — [Waves Tune Real-Time](https://www.waves.com/plugins/waves-tune-real-time))*

## 5. Модулятивные эффекты

Динамическое изменение параметров звука с помощью модуляции.
(здесь важно из базовых плагинов ниже перечисленных плагинов выучить хотя бы один плагин)

![Модулятивные эффекты](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTcZheKfR_eg4TEc-lKgT2-16jVdWVS6vesIQ&s)

- Тремоло в динамике — плагин для модуляции громкости звука с помощью периодического изменения амплитуды. *(в пакете Khs есть отличный тремоло, опять же если хотите SOUNDTOYS, опять же вот и встроенные — tremolator, очень часто на тремоло можно заменить, но обязательно хотя бы выучить один)*
- Вибрато в динамике — плагин для модуляции высоты тона с помощью периодического изменения частоты. *(MVibrato от мелодии плагина от meldaproduction, либо просто для эксперимента и изучения на инструменте из Kontakt)*
- Ринг-модулятор в динамике — плагин для создания металлических и необычных тембров. *(Khs Ring mod)*
- LFO (Low-Frequency Oscillator) в динамике — генератор для модуляции параметров (громкость, частота и т. д.). *(LFOtool от разработчика Serum, опять же если в Shaperbox плагине Volume Shaper)*

## 6. Дисторшн и овердрайв ✅

Агрессивное окрашивание тембра для добавления характера звуку.✅

![FabFilter Pro-SATURN 2](https://www.fabfilter.com/img/products/saturn-2-screenshot.jpg)

- Дисторшн (Distortion) в динамике — плагин для сильного искажения сигнала, часто используемый в роке и электронике. *(Khs Distorstion, Khs Phase Distorstion, FabFilter Pro Saturn, Faturator)*
- Овердрайв (Overdrive) в динамике — плагин для более мягкого, тёплого искажения сигнала. *(FabFilter Pro Saturn)*
- Фаззер (Fuzz) в динамике — плагин для экстремального искажения. *(FabFilter Pro Saturn)*
- Биткраш — плагин для снижения качества звука, создания эффекта "из радио". *(FabFilter Pro Saturn, Khs Bitcrush, Khs Shaper, Khs Shaper Table)*

---

## 7. Сэмпл- и временная обработка ✅

Манипуляции с семплами и временными параметрами.✅

![Portal](https://splice-res.cloudinary.com/image/upload/v1573227192/gear/share_images/Portal-product_shareImage.png)

- Гранулярные плагины в динамике — плагины для разбивки семпла на микрочастицы "зерна" и их обработки. *([Quanta](https://www.audiodamage.com/products/ad055-quanta-2), Fruity Granulizer, [Granulator II](https://www.ableton.com/en/packs/granulator-ii/) — в 12 пакетах из 16 плагинов и 11 пресетов формата amxd внутри)*
- Статтер (Stutter) в динамике — плагин для создания эффекта заикания звука. *(Glitch 2)✅*
- Реверс — плагин для обратного воспроизведения семпла. *(обычно встроенный плагин DAW будет вам полезен)*
- Тайм-стретчинг — плагин для изменения темпа семпла без изменения высоты тона. *(обычно встроенный плагин DAW будет вам полезен)*

---

## 8. Экспериментальные эффекты

Плагины для нестандартной обработки звука.

![Snap Heap](https://kilohearts.com/resources/media_dynamic/offers/snap_heap/thumb_400.png)

- Мульти-эффекты — плагины позволяющие прогонять звук через несколько эффектов одновременно (цепочка, параллель + сухой + ещё всё что надо). *(Multipass, [Glitch 2](https://illformed.com/))*
- Генераторы шума — плагины для создания белого, розового, коричневого и других типов шума.
