# 🏅 Коллекция бейджей

Добивайтесь целей в обучении и получайте бейджи! Каждый бейдж даёт **+50 XP** и приближает вас к следующему уровню.

## Как получить бейджи

Бейджи начисляются **автоматически** при выполнении условий:

| Редкость | Цвет | Описание |
|----------|------|----------|
| `Обычный` | Серый | Доступные с первых шагов |
| `Необычный` | Зелёный | Требуют регулярного обучения |
| `Редкий` | Синий | Для постоянных учеников |
| `Эпический` | Фиолетовый | Для преданных студентов |
| `Легендарный` | Оранжевый | Вершина мастерства |
