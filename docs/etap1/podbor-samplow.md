# Подбор сэмплов и структура бита

## Подбор сэмплов друг к другу

Хороший бит начинается с правильного выбора сэмплов. Вот ключевые принципы:

### Один пак

Сэмплы из одного пака обычно записаны в едином стиле и хорошо сочетаются между собой.

### Один жанр

Не смешивайте сэмплы из разных жанров на начальном этапе. Если делаете трэп — используйте трэп-сэмплы.

### Наслушка и музыкальный опыт

Чем больше музыки вы слушаете, тем лучше понимаете, какие звуки работают вместе.

### Баланс громкости

Сэмплы должны быть сбалансированы по громкости ещё до начала сведения:

- Кик — громкий, но не перегруженный
- Снейр — чёткий, слышимый поверх кика
- Хэты — на фоне, не перекрывают основные элементы

!!! tip
    Слушайте бит на разных громкостях. Если на низкой громкости не слышно снейра — что-то не так.

## Структура бита

### Пирамида

Представьте бит как пирамиду:

```
        Мелодия / Вокал
       /                \
      /   Перкуссия FX   \
     /                     \
    /       Драмка          \
   /_________________________\
          Бас
```

- **Фундамент** — бас и кик
- **Середина** — снейр, хэты, перкуссия
- **Верхушка** — мелодия, вокал, эффекты

### Лестница

Структура трека как лестница — каждый элемент добавляется постепенно:

1. **Интро** — атмосферное вступление
2. **Строфа** — драмка + бас
3. **Припев** — добавляется мелодия
4. **Бридж / Смена** — контрастная секция
5. **Аутро** — затухание

!!! note
    Не перегружайте бит с первых секунд. Давайте слушателю привыкнуть к каждому элементу по отдельности.
