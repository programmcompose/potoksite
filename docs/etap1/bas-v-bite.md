# Бас в бите

## Сэмпл баса

### 808 Бас

**808** — легендарный бас из драм-машины Roland TR-808. Длинный, низкочастотный, с характерным искажением. Основа трэпа и хип-хопа.

### Риз Бас (Reese Bass)

**Риз-бас** — глубокий, дрожащий бас, созданный с помощью двух синтезаторных осцилляторов, слегка расстроенных друг относительно друга. Популярен в дарк-трэпе и дрилле.

## Мелодия уже с басом

Некоторые сэмплы содержат и мелодию, и бас в одном файле. В этом случае:

- Можно использовать как есть
- Можно отделить бас с помощью EQ (срезать высокие частоты для баса, низкие — для мелодии)

## Определение ноты баса

### Подбор корневой ноты

Корневая нота баса должна соответствовать тональности мелодии. Если мелодия в Cm, бас должен начинать с ноты C.

### Тональность сэмпла

Определите тональность сэмпла мелодии, чтобы подобрать совместимый бас.

### SPAN и Convert to MIDI (FL Studio)

**SPAN** — бесплатный плагин для определения нот в аудио:

1. Загрузите сэмпл в SPAN
2. Плагин покажет, какие ноты содержатся в сэмпле
3. Используйте **Convert to MIDI**, чтобы преобразовать аудио в MIDI-клип

!!! tip
    SPAN — незаменимый инструмент для работы с сэмплами. Он показывает тональность, ноты и даже предлагает гармоничные аккорды.

## Тональность

### Что такое тональность

**Тональность** — система организации нот вокруг центральной ноты (тоники). Определяет «цвет» и настроение музыки.

### Мажор

Мажорные тональности звучат ярко, радостно, открыто.

```
C мажор: C D E F G A B C
```

### Минор

Минорные тональности звучат мрачно, напряжённо, меланхолично.

```
C минор: C D Eb F G Ab Bb C
```

!!! note
    В хип-хопе и трэпе чаще используется минор — он создаёт нужное настроение.

## Прописание баса в Паттерне MIDI

### Пиано Ролл (Piano Roll)

**Piano Roll** — интерфейс для написания мелодий и баса в виде нот на сетке.

### В FL Studio

1. Откройте Piano Roll для канала баса
2. Расставьте ноты, следуя тональности мелодии
3. Используйте корневые ноты на сильных долях

### В Ableton Live

1. Откройте Piano Roll на MIDI-треке с басом
2. Пишите ноты, используя шкалу (Scale) для ограничения тональностью
3. Используйте MIDI Effects > Scale для автоматического ограничения нот

## Главные ошибки написания баса

### Слишком много нот

Бас не должен быть перегружен. Чем меньше нот — тем мощнее звучание. Используйте длинные ноты на корнях аккордов.

### Бас не в тон

Самая частая ошибка. Бас должен быть в той же тональности, что и мелодия. Используйте SPAN или другие инструменты для проверки.

### Cut Self в FL Studio

**Cut Self** — функция в FL Studio, которая автоматически обрезает предыдущую ноту при начале новой. Полезно для 808-баса, чтобы избежать наложения.

### Нет оборудования, воспроизводящего басовый спектр

Обычные наушники и колонки не воспроизводят частоты ниже 40-50 Гц. Вы можете не слышать, как звучит бас на больших системах.

### Неподходящий сэмпл

Не каждый бас подойдёт к вашему биту. Экспериментируйте, но помните про жанровую совместимость.
