# Работа с микшером

## Что такое микшер

**Микшер** — панель управления громкостью, панорамой и эффектами каждой дорожки в проекте.

## Настройка громкости

### Базовые правила

| Дорожка | Уровень (dB) | Примечание |
|---------|-------------|------------|
| Кик | -6 to -3 dB | Фундамент, должен быть слышен |
| Снейр | -8 to -5 dB | Чёткий, но не давит на кик |
| Хэты | -12 to -8 dB | На фоне |
| Бас | -6 to -4 dB | В паре с киком |
| Мелодия | -10 to -6 dB | Зависит от плотности |

### Главный принцип

Начинайте с кика и баса, затем постепенно добавляйте остальные элементы. Каждый новый звук должен вписываться в уже существующую картину.

## Расположение дорожек

Организуйте микшер логически:

```
1-4:   Кик, Снейр, Клэп, Перкуссия (ударные)
5-8:   Бас (808, саб)
9-12:  Мелодия, вокал, пэды
13-16: FX, переходы, атмосфера
```

!!! tip
    Присваивайте дорожкам цвета и имена. Через месяц вы скажете себе спасибо за порядок.

### Панорама (Pan)

- Кик и бас — по центру
- Снейр — по центру или слегка смещён
- Хэты и перкуссия — можно расставлять по ширине
- Мелодия — зависит от аранжировки
