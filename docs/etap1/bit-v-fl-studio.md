# Бит в FL Studio

## Добавление звуков в Плейлист

**Плейлист (Playlist)** — основное рабочее пространство FL Studio, где вы расставляете паттерны и аудио-клипы по временной шкале.

### Шаги:

1. Добавьте звуки в **Channel Rack**
2. Создайте паттерн с помощью шаг-секвенсора
3. Перетащите паттерн из Channel Rack в Плейлист
4. Дублируйте и редактируйте секции для построения структуры

### Горячие клавиши FL Studio

| Действие | Клавиша |
|----------|---------|
| Открыть Channel Rack | `F6` |
| Открыть Плейлист | `F5` |
| Открыть Пиано Ролл | `F7` |
| Открыть Микшер | `F9` |
| Открыть Браузер | `F8` |
| Play/Stop | `Пробел` |
| Запись | `F10` |
| Undo | `Ctrl+Z` |
| Redo | `Ctrl+Shift+Z` |
| Выделить всё | `Ctrl+A` |
| Копировать | `Ctrl+C` |
| Вставить | `Ctrl+V` |
| Сохранить проект | `Ctrl+S` |
| Рендер | `Ctrl+R` |

!!! tip
    Освойте горячие клавиши — это ускоряет работу в 3-5 раз.

## Типичный рабочий процесс

1. **Подбор сэмплов** — выберите кик, снейр, хэты, бас
2. **Создание паттерна** — запрограммируйте драмку в Channel Rack
3. **Добавление мелодии** — импортируйте или создайте мелодический сэмпл
4. **Написание баса** — пропишите бас в Piano Roll
5. **Структурирование** — расставьте секции в Плейлисте
6. **Микширование** — настройте уровни в Микшере
7. **Рендер** — экспортируйте финальный трек
