# 📊 Личная статистика

Отслеживайте свой прогресс, уровень и активность в курсе «18 ПОТОК».

## Система уровней

Каждое действие приносит **XP (опыт)**:

| Действие | XP |
|----------|-----|
| Просмотр страницы | +5 |
| Начало сессии | +3 |
| Пройденный урок | +25 |
| Получение бейджа | +50 |
| Повышение уровня | +100 |

Чем выше уровень, тем больше XP нужно для следующего. Максимум — **50 уровней**.

## Управление данными

### Экспорт
Сохраните прогресс в JSON-файл для резервной копии или переноса на другое устройство.

### Импорт
Загрузите ранее экспортированный файл для восстановления прогресса.

### Сброс
Полный сброс всех данных геймификации. **Внимание:** действие необратимо!

!!! warning "Конфиденциальность"
    Все данные хранятся **только в вашем браузере** (LocalStorage). Никакая информация не отправляется на сервер.
