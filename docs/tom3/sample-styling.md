# Стилизация сэмплов

Техники трансформации сэмплов — от лёгкого окрашивания до полного разрушения.

## Ретро звучание

| Инструмент | Что делает |
|---|---|
| **RC-20 (Retrocolor)** | Эмуляция старого оборудования: кассета, радио, телефон, биткрашер. Mix-контроль для плавной регулировки эффекта |
| **Tape-эмуляции** | Ленточная сатурация + wow/flutter (дрейф питча). Добавляет «теплоту» и «аналоговость» |

## Атмосферность

| Техника | Результат |
|---|---|
| **Reverb + Low-pass** | Звук «под водой» или «в другой комнате» |
| **Reverse reverb** | «Всасывающий» хвост перед ударом — классический эффект для snares и свипов |
| **Granular-эффекты** | Зернистая текстура, «размытый» во времени звук |

## Фильтрация сэмплов и драмки

| Фильт | Применение |
|---|---|
| **Auto-filter** | Автоматическое открытие/закрытие фильтра по времени. «Sweep»-эффект |
| **High-pass на драм-бусе** | Убрать низкочастотную грязь из перкуссии |
| **Resonant filter** | Подчёркивание конкретной частоты — «кислотный» звук |

## Эксперименты с WARP и питчем

Изменение темпа и высоты — мощный инструмент трансформации.

| Инструмент | Возможности |
|---|---|
| **ShaperBox 3 (RX)** | Wavefolder, bitcrusher, ring modulator, pitch shifter. Полное перестроение звука |
| **EFX2 / EFX Motions** | Пресеты трансформации: от лёгкого окрашивания до полного «разрушения» сэмпла |

**Warp-манипуляции:**
- Ускорение сэмпла + повышение питча = энергичный, «нервный» звук
- Замедление + понижение = тёмный, «дабstep» характер
- Reverse + pitch shift = «призрачный» эффект

!!! tip "Экспериментируйте"
    Не бойтесь «портить» сэмплы. Часто самый интересный звук рождается из «сломанного» оригинала.
