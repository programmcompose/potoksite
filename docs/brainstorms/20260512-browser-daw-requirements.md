## Обзор

Создать браузерную DAW (Digital Audio Workstation) в стиле Ableton Live как интерактивный учебный инструмент для сайта potoksite. Демонстрирует принципы работы DAW: дорожки, секвенсор, микшер, transport.

## Цель

**Учебный инструмент** — показать, как устроена DAW изнутри: дорожки, секвенсоры, микширование, transport. Не коммерческий продукт, а интерактивная демонстрация для курса по музыкальному продакшену.

---

## Архитектура

### Технологии
- **Tone.js** (Web Audio API) — аудио-движок (уже используется на сайте)
- **Чистый JS** — без фреймворков (соответствует существующим инструментам)
- **HTML/CSS** — стилизация под Ableton Live

### Размещение
- `docs/tools/browser-daw/index.html` — основная страница
- `docs/tools/browser-daw/samples/` — сэмплы для драмов
- Интеграция в `docs/tools/index.html` (карточка инструмента)

## Инструменты

| Дорожка | Тип | Звук |
|---------|-----|------|
| Kick | Drum | Реальный сэмпл |
| Snare | Drum | Реальный сэмпл |
| Hi-Hat (closed) | Drum | Реальный сэмпл |
| Hi-Hat (open) | Drum | Реальный сэмпл |
| Bass | Melodic | Tone.js синтез |
| Lead/Melody | Melodic | Tone.js синтез |

**Гибридный подход:** Drums — реальные сэмплы (реалистичный ритм), Bass + Melody — Tone.js синтез (self-contained, без файлов).

## Секвенсор

### Arrangement View (единственный view)
- **Step Sequencer (Grid)** для драмов — 16 шагов (4 такта по 4/4), каждая строка = драм-инструмент
- **Piano Roll** для Bass и Melody — полноценный drag-and-drop (перетаскивание нот, изменение длины)
- Snap grid (quarter, eighth, sixteenth)
- Визуальный playhead (бегущая линия воспроизведения)

### Transport
- Play / Stop
- Tempo (BPM slider, диапазон 60-180, default 120)
- Metronome (вкл/выкл)
- Loop (вкл/выкл, повтор паттерна)

---

## Микшер

На каждой дорожке:
- **Volume** (fader)
- **Pan** (L-R)
- Mute кнопка

Master канал:
- Master Volume
- Master Mute

## UI/UX

### Стиль
- Визуальная стилистика **Ableton Live** (тёмная тема, характерные цвета: оранжевый playhead, серый фон, цветные дорожки)
- Arrangement View — дорожки расположены вертикально, время горизонтально
- Цветовое кодирование дорожек (drums — один цвет, bass — другой, melody — третий)

### Адаптация
- **Десктоп** — основной
- **Мобильная** — базовая адаптация (уменьшенные элементы, touch-friendly grid)

### Сохранение
- **Нет** — сессия сбрасывается при обновлении страницы (учебный инструмент)

## Ограничения

- Без сохранения проектов
- Без VST-плагинов (EQ, компрессор и т.д.)
- Без экспорта аудио
- Без загрузки пользовательских сэмплов
- Без MIDI-подключения
