# Saturation / Сатурация

Сатурация — контролируемый перегруз сигнала. Добавляет характер, теплоту и «живость» цифровому звуку.

## Что делает сатурация

| Эффект | Как проявляется |
|---|---|
| **Добавление гармоник** | Генерирует новые частоты (обертоны), которых не было в исходном звуке. Звук становится «богаче» |
| **Уплотнение звука** | Гармоники заполняют «пустоты» в спектре. Тонкий звук становится толстым |
| **Острота звука** | Верхние гармоники добавляют чёткости и атаки без жёсткого EQ-буста |

## Яркость через сатурацию

Сатурация — альтернатива EQ для добавления яркости. Вместо того чтобы приподнять высокие частоты на эквалайзере, вы «разогреваете» сигнал — и гармоники сами создают верх.

| Подход | Результат |
|---|---|
| EQ boost высоких | Яркость, но может стать «цифровой» и резкой |
| Сатурация | Яркость + теплота + характер. Звук «живой» |

## Какой плагин подобрать

| Тип сатурации | Характер | Примеры плагинов |
|---|---|---|
| **Tube (ламповая)** | Тёплая, мягкая, чётные гармоники | Tape, Warm, Vintage |
| **Tape (ленточная)** | Округлая, «аналоговая» теплота | RC-20, Kramer Master Tape |
| **Transformer** | Плотная, «упакованная» | Decapitator, Saturn |
| **Diode / FET** | Агрессивная, с характером | Distressor, 3013 |
| **Digital / Wavefolder** | Экспериментальная, «кислотная» | ShaperBox, EFX2 |

!!! tip "Как выбрать"
    Для **теплоты** — Tape/Tube. Для **плотности** — Transformer. Для **характера** — Diode/FET. Для **экспериментов** — Digital.

## Примеры плагинов

| Плагин | Тип | Характер |
|---|---|---|
| **Soundtoys Decapitator** | Multi-mode saturator | 4 режима (E, P, X, N) — от ламповой теплоты до агрессивного distortion |
| **Waves Kramer Master Tape** | Tape saturator | Эмуляция ленточного магнитофона — округлая теплота, мягкие верха |
| **FabFilter Saturn 2** | Multi-band saturator | Сатурация на каждую частотную полосу отдельно — тонкий контроль окраса |

## Дисторшн

Дисторшн — это сатурация на максимуме. Когда перегруз становится настолько сильным, что форма волны «обрезается».

| Контекст | Применение |
|---|---|
| **Гитарные плагины** | Tube Screamer, Overdrive, Fuzz — классика для гитар и не только. Можно кидать на бас, синтезатор, даже вокал |
| **На синтезаторе** | Wavefolder / Bitcrusher — для агрессивного electronic-звuka |
| **На драм-машине** | Лёгкий distortion на bus-е барабанов = «клей» и характер |

## Ошибки перегруза

!!! warning "Частые ошибки"
    - **Слишком много gain** — звук превращается в кашу, теряется чёткость
    - **Сатурация на суб-басу** — гармоники в низах создают грязь. Сатюрируйте только середину и верх баса
    - **Забыть про Mix knob** — хороший плагин имеет Dry/Wet. Начните с 20–40% wet
    - **Сатурация до компрессора** — компрессор «съест» гармоники. Ставьте сатурацию после компрессии

## Сатурация как естественный компрессор

Сатурация не только добавляет гармоники — она сама по себе сжимает сигнал. Мягкие гармоники работают как «естественный компрессор», уплотняя звук без потери динамики.

### Практический приём: параллельная сатурация

1. Создайте **parallel bus** — продублируйте сигнал на отдельный трек
2. На параллельном треке **откройте сатурацию на максимум** — до агрессивного перегруза
3. Смешивайте параллельный сигнал с сухим: **10–30% wet**
4. Результат: **уплотнение и характер без потери динамики**

| Wet % | Эффект |
|---|---|
| 10–20% | Лёгкое уплотнение, «клей» |
| 20–30% | Заметное уплотнение, характер |
| 30–50% | Агрессивный параллельный перегруз |

### Почему это работает

| Обычная сатурация | Параллельная сатурация |
|---|---|
| Сжимает весь сигнал | Сжимает только гармоники |
| Теряет динамику | Сохраняет транзиенты |
| Один плагин на треке | Максимальный перегруз + чистый сигнал |
