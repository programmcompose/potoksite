# Мастеринг

Мастеринг — это **финальный этап** производства музыки. Цель мастеринга:

1. Сделать трек конкурентоспособным по громкости
2. Обеспечить консистентность между треками альбома
3. Адаптировать звук под целевую платформу (Spotify, Apple Music, винил)

!!! warning
    Мастеринг **не может исправить плохой микс**.
    Если микс проблемный — вернитесь к сведению.

## Цепочка мастеринга

Типичная цепочка плагинов на мастер-канале:

```
Mixer Output → EQ → Compressor → Saturation → Limiter → Reference (сравнение) → Export
```

## Инструменты мастеринга

### 1. Мастеринг EQ

| Тип | Применение |
|-----|-----------|
| Linear Phase | Без фазовых сдвигов, для тонкой настройки |
| Parametric | Гибкая настройка пиков и впадин |
| Graphic | Быстрые корректировки по полосам |

!!! tip
    Используйте корректировки **не более ±2 dB** на мастер-канале.
    Большие изменения — признак проблем в миксе.

### 2. Мастеринг компрессор

| Параметр | Значение |
|----------|---------|
| Ratio | 1.5:1 — 2:1 |
| Attack | 10–50 мс |
| Release | Auto или 50–200 мс |
| Gain Reduction | 1–3 dB |

### 3. Лимитер

Лимитер предотвращает клиппинг и повышает общую громкость.

| Параметр | Значение |
|----------|---------|
| Ceiling | 0 dB |
| True Peak | 0 dBTP |

### 4. Референс-сравнение

Плагин для сравнения вашего мастеринга с референсными треками.
Позволяет оценить громкость и баланс в реальном времени.

| Плагин | Назначение |
|--------|-----------|
| ADPTR Metric AB | LUFS-метрики, сравнение с референсом |
| Reference (Plugin Alliance) | A/B/C/D сравнение треков |

## Экспорт

| Параметр | Значение |
|----------|---------|
| Format | WAV |
| Bit Depth | 24-bit (архив), 16-bit (релиз) |
| Sample Rate | 44.1 кГц (CD), 48 кГц (streaming) |

## Чек-лист мастеринга

- [ ] Микс завершён и одобрён
- [ ] Референсный трек загружен в плагин для сравнения
- [ ] EQ — тонкие корректировки (±1–2 dB)
- [ ] Компрессия — glue, 1–3 dB gain reduction
- [ ] Лимитер — ceiling 0 dB, true peak 0 dBTP
- [ ] Сравнение с референсом через ADPTR Metric AB / Reference
- [ ] Проверка на разных системах
- [ ] Экспорт WAV 24-bit/44.1 кГц
- [ ] LUFS в целевом диапазоне
