# Chorus / Хорус

Хорус создаёт эффект «множественности» — один звук превращается в несколько слегка расстроенных копий.

## Что делает хорус

| Эффект | Как проявляется |
|---|---|
| **Водянистый звук** | Характерный «плеск» — звук кажется «мокрым», «текучим». Классическийchorus-эффект на гитарах и клавишах |
| **Наслоенность и широта** | Несколько «голосов» расходятся в стереополе. Тонкий звук становится широким и «толстым» |
| **Винтажное звучание** | Аналоговые хорусы (Roland JC-120, Boss CE-1) дают тёплый, «ретро» характер |

## Размытие транзиентов

Хорус «размазывает» атаку звука — начальный удар (transient) становится мягче.

| Когда хорошо | Когда плохо |
|---|---|
| Пэды, клавиши, гитары — транзиенты не важны | Kick, snare, бас — транзиенты критичны для чёткости |

!!! warning "Не ставьте хорус на"
    Kick, snare, sub-bass. Хорус «размоет» удар и звук потеряет punch.

## Параметры хоруса

| Параметр | Влияние |
|---|---|
| **Rate** | Скорость модуляции. Медленно = плавное «дыхание», быстро = «вибрация» |
| **Depth** | Глубина расстройки. Больше = более выраженный эффект |
| **Voices** | Количество «голосов». 2 = минимум, 5+ = «стена звука» |
| **Spread** | Стерео-ширина расхождения голосов |

## Примеры плагинов

| Плагин | Тип | Характер |
|---|---|---|
| **Chorus Dimension** | Multi-chorus | Несколько chorus-голосов с независимой rate/depth — от тонкого расстрой до «стены звука» |
| **Arturia Called the Chorus** | Analog chorus | Эмуляция аналоговых хорусов — тёплый, «ретро» характер, винтажное звучание |
| **UAD Roland JC-120** | Guitar amp/model | Эмуляция легендарного комбо — винтажный chorus, «водянистый» характер, классика для гитар и клавиш |
