# Работа с референсами

Референсные треки — ваш ориентир. Сравнение с профессиональной музыкой — один из самых эффективных инструментов сведения.

## Зачем референсы

| Проблема | Решение через референс |
|---|---|
| «Мой бас не такой толстый» | Сравните спектр баса с референсом |
| «Вокал не выдвигается» | Проверьте уровень и середину вокала в референсе |
| «Трек не够 громкий» | Сравните LUFS, не пики |
| «Не понимаю, что не так» | A/B сравнение покажет разницу |

## Использование SpectraLayers

| Функция | Применение |
|---|---|
| **Спектральное сравнение** | Наложение спектра вашего трека на референс. Визуально видно, каких частот не хватает |
| **Анализ громкости** | Сравнение RMS/LUFS в реальном времени |
| **Стерео-изображение** | Проверка ширины стереополя |

!!! tip "Как сравнивать правильно"
    Уровни вашего трека и референса должны быть **одинаковыми по громкости**. Иначе мозг будет думать, что громкий трек — «лучше». Используйте плагин для матчирования громкости (например, Magic AB, ADPTR Metric AB).

## Другие плагины для референсов

| Плагин | Особенность |
|---|---|
| **Reference (Plugin Alliance)** | A/B/C/D переключение до 4 треков. Автоматическое матчирование громкости |
| **ADPTR Metric AB** | LUFS-метрики + спектральное сравнение + A/B |
| **Magic AB (Magic Audio)** | Мгновенное переключение, матчирование громкости и баса |
