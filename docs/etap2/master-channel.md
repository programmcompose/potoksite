# Мастер-канал

Мастер-канал — финальная точка, где весь микс собирается воедино. Обработка на мастере — тонкая и аккуратная.

## Чек перед рендером

### Тональный баланс

Проверка частотного распределения микса.

| Инструмент | Что делает |
|---|---|
| **Tonal Balance Control (Youlean / iZotope)** | Визуализация спектра в реальном времени с цветовой зоной «нормы». Показывает, каких частот мало или много |
| **Ozone 12 Stabilizer** | Анализирует микс и автоматически компенсирует изменения при сведении. Если вы крутите EQ — Stabilizer держит баланс |

### Уплотнение пачки звука

| Инструмент | Роль |
|---|---|
| **Лимитер / Клиппер** | Финальный контроль пиков. «Прижимает» микс, делает его громче и плотнее |
| **Speccraft** | Мультиполосный имиджер + EQ. Позволяет «сужать» грязные частоты и «расширять» чистые |
| **Soothe2** | Динамический де-ессер + резонанс-киллер. Гасит проблемные резонансы в реальном времени |

### LUFS — громкость трека

| Жанр | Целевой LUFS (Integrated) |
|---|---|
| Hip-Hop / Trap | -8 … -6 |
| Electronic / EDM | -9 … -7 |
| Pop | -10 … -8 |
| Rock | -12 … -10 |

!!! note "True Peak"
    Держите True Peak на уровне **-1.0 dBTP**. Это оставляет headroom для конвертации в MP3/AAC без артефактов.

### Ориентировка на референсы

Сравнивайте свой микс с профессиональными треками в том же жанре. Референс — ваш компас.
