# Reverb / Реверберация

Реверберация имитирует отражения звука в пространстве. Превращает «сухой» звук в объёмный и глубокий.

## Что делает реверб

| Эффект | Описание |
|---|---|
| **Пространство** | Создаёт ощущение «комнаты» — звук не висит в вакууме |
| **Хвост (Tail)** | Звук затухает после основного сигнала. Длина хвоста = размер «комнаты» |
| **Глубина** | Дальние элементы получают больше реверба, близкие — меньше. Формируется 3D-картина микса |

## Как правильно добавлять реверб

| Принцип | Практика |
|---|---|
| **Меньше = больше** | Начните с минимума. Если слышите реверб явно — его уже много |
| **Через Send/Return** | Не вешайте реверб на каждый трек отдельно. Создайте 1–2 реверб-буса и посылайте на него треки |
| **Фильтруйте реверб** | Срежьте низы (HPF ~300 Гц) и верхи (LPF ~8 кГц) на самом ревербе — иначе он «забивает» микс |
| **Pre-Delay** | Задержка перед началом реверба (20–60 мс). Помогает сохранить чёткость атаки |

## Глубина в треке

Располагайте элементы в «пространственной плоскости»:

| Позиция | Реверб | Пример |
|---|---|---|
| **Передний план** | Минимум / нет | Lead-вокал, основной kick, lead-синт |
| **Средний план** | Умеренный | Бэк-вокал, гитара, pads |
| **Задний план** | Много | Атмосферные текстуры, ambient-слои |

## Чёткость vs Размытость

!!! warning "Ошибка: размытый микс"
    Если после добавления реверба трек стал «мыльным» и нечётким:
    - Уменьшите Decay (хвост)
    - Добавьте Pre-Delay
    - Срежьте низкие частоты на реверб-бусе (HPF)
    - Используйте shorter reverb (Room вместо Hall)

**Планы в треке** — это слои глубины. Каждый элемент должен занимать свою «позицию» от слушателя. Реверб — главный инструмент для создания этих планов.
