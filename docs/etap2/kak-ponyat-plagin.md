# Как понять плагин

## Осцилоскоп из ShaperBox 3 — визуальное понимание плагинов

Когда ты работаешь с VST-плагином и не уверен, что именно он делает со звуком — осцилоскоп помогает увидеть изменения в форме волны.

## Как настроить

1. **Добавь плагин, который ты хочешь понять** на микшер-канал
2. **После него поставь ShaperBox 3** в режиме **Oscilloscope**
3. **Меняй настройки** на плагине, который изучаешь — и смотри, как меняется waveform в ShaperBox 3

## Что это даёт

| Без осцилоскопа | С осцилоскопом |
|---|---|
| Понимание только на слух | Понимание на слух + визуальное |
| «Звук стал громче» — неясно как | waveform показывает амплитуду явно |
| «Звук стал плотнее» — абстрактно | waveform показывает транзиенты и пики |
| Трудно закрепить понимание | Визуальная форма закрепляет связь «настройка → результат» |

## Что визуализирует осцилоскоп

| Что меняется | Что видно на waveform |
|---|---|
| **Громкость / амплитуда** | Высота пиков волны — выше = громче, ниже = тише |
| **EQ / частоты** | Форма волны — срез частот меняет «текстуру» линии |
| **Компрессия** | Пики «приплющиваются» — gain reduction визуален |
| **Saturation** | waveform становится «грубее», транзиенты плотнее |
| **Reverb / Delay** | «Хвосты» после транзиента — эхо видно как повторные волны |
| **Clipper** | Пики «обрезаны» — flat tops на waveform |

## Практика

Для каждого плагина — меняй по одному параметру, наблюдай waveform, закрепляй связь:

> «Когда я кручу **Attack** компрессора → waveform показывает, что транзиент [сохраняется / срезается]»

Это закрепляет понимание, как плагин работает с **громкостью** и **амплитудой** звука — не абстрактно, а визуально.
