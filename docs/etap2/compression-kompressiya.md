# Compression / Компрессия

Компрессор — самый важный и самый сложный инструмент сведения. Управляет динамикой сигнала.

## Что делает компрессия

| До компрессии | После компрессии |
|---|---|
| Громкие участки очень громкие | Громкие участки приглушены |
| Тихие участки теряются | Тихие участки относительно слышнее |
| Большая разница между пиками и «дном» | Разница уменьшена — звук ровнее |

**Простыми словами:** компрессор — это «автоматическая громкость». Когда сигнал становится слишком громким, он тихо его «крутит».

## Как услышать компрессию

| Признак | Что значит |
|---|---|
| Звук стал «ровнее», пиков меньше | Компрессия работает |
| Звук «дышит» — громкость пульсирует | Release слишком быстрый |
| Звук «сплющен», без жизни | Перекомпрессия — уменьшите ratio или поднимите threshold |
| Атака «удара» сохранилась | Attack настроен правильно (достаточно медленный) |

## Параметры компрессора

| Параметр | Что делает | Типичные значения |
|---|---|---|
| **Threshold** | Порог срабатывания. Всё выше — сжимается | -30…-10 dB |
| **Ratio** | Насколько сильно сжимается | 2:1 (мягко), 4:1 (средне), 8:1+ (жёстко) |
| **Attack** | Сколько времени компрессор «ждёт» перед сжатием | 1…100 мс |
| **Release** | Сколько времени компрессор «отпускает» | 50…500 мс |
| **Makeup Gain** | Компенсация потери громкости | По слуху |

### Attack: ключевой параметр

| Attack | Результат |
|---|---|
| **Быстрый** (1–10 мс) | Транзиент сжат → звук мягкий, «округлый» |
| **Медленный** (30–100 мс) | Транзиент «проскакивает» → звук «бьёт», punch сохраняется |

### Release: синхронизация с темпом

Release должен «уложиться» в один такт или полтакта. Если release синхронен с BPM — компрессия «дышит» в ритм трека.

## Применение

| Контекст | Цель | Подход |
|---|---|---|
| **Мелодия** | Выровнять динамику, «склеить» звук | Мягкая компрессия (2:1–4:1), slow attack, gain reduction 2–4 dB |
| **Драм-машина** | Контроль пиков, добавление punch | Быстрый attack на snare, medium на kick, bus-компрессия для «клея» |
| **Bus / Group** | «Склейка» группы треков | Очень мягко: 1.5:1, 1–2 dB GR, slow attack |

## Multiband компрессия

Обычный компрессор сжимает весь сигнал целиком. **Multiband** делит сигнал на частотные полосы и сжимает каждую независимо.

| Полоса | Что сжимать | Зачем |
|---|---|---|
| **Низкие** (20–200 Гц) | Жёстче (4:1–6:1) | Контроль суб-баса, чтобы он не «улетал» |
| **Средние** (200–4000 Гц) | Мягче (2:1–3:1) | Выравнивание вокала/мелодии |
| **Высокие** (4 кГц+) | Очень мягко или не сжимать | Сохранение «воздуха» и чёткости |

**Когда использовать:** когда проблема локализована в одной частотной полосе. Например, бас «улетает» только в низах — сжимайте только низы, не трогая середину.
