# Основные понятия сведения

Перед погружением в плагины — разберём, какие характеристики звука вы будете формировать.

## Характеристика звука

| Характеристика | Что это | Как добиться |
|---|---|---|
| **Широкий звук** | Звук «раскрыт» в стереополе, занимает левый и правый канал | Панорамирование, chorus, stereo-imaging плагины, ping-pong delay |
| **Яркий звук** | Чёткий, «блистающий», с акцентом на высокие частоты | Приподнять 5–15 кГц на EQ, добавить сатурацию, использовать air-полку |
| **Плотный звук** | Звук «толстый», наполненный, без пустот | Сатурация, компрессия, наложение слоёв, заполнить середину (200–500 Гц) |
| **Глухой / пустой звук** | Проблема: звук «уткнулся в стену» или «провалился» | Проверить EQ — возможно срезано слишком много высоких; добавить яркости |
| **Атмосферный звук** | Пространственный, «объёмный», с ощущением глубины | Reverb, delay, filtering (low-pass для «под водой» эффекта) |
| **Сухой звук** | Близкий, интимный, без хвостов | Минимум или ноль reverb/delay, быстрый attack компрессии |

## Шпаргалка: диагностика звука

!!! tip "Как определить проблему"
    Задайте себе вопрос: **«Чего не хватает?»**
    - Звук тонкий → не хватает **плотности** (сатурация, слои)
    - Звук мутный → не хватает **яркости** (EQ highs, high-pass фильтр)
    - Звук плоский → не хватает **пространства** (reverb, delay)
    - Звук «размазан» → слишком много **пространства** (убрать reverb)

## Частотная шпаргалка

| Диапазон | Что живёт здесь | Если перебор | Если мало |
|---|---|---|---|
| 20–60 Гц | Sub-bass, фундамент | Гудение, «бубнение» | Нет веса |
| 60–250 Гц | Тело баса, warmth | Мутность | Пустота |
| 250–500 Гц | Плотность середины | «Картонный» звук | Тонкость |
| 500–2 кГц | Вокал, гитара, атака | Резкость | Не слышно |
| 2–6 кГц | Деталь, чёткость | Усталость ушей | Тусклость |
| 6–12 кГц | Presence, «близость» | Свист | Дальность |
| 12–20 кГц | Air, «воздух» | Шипение | Плоскость |
