Как это работает

Тренировка: Система применяет усиление на случайной частоте. Слушайте и угадайте, какая частота усилена.

День: Одна задача в день с фиксированным seed. Сравнивайте прогресс с друзьями!

Свободный: Экспериментируйте с настройками EQ без проверки ответов.

Bypass: Удерживайте кнопку (или B) для сравнения "до/после" в реальном времени.

Уровни: Лёгкий ±25%, Средний ±15%, Сложный ±5%, God ±2% от целевой частоты.
