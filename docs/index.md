## 📚 Структура курса

### [Этап №0 — Подготовка к продакшну](etap0/index.md)
Всё, что нужно знать перед началом: выбор DAW, оборудование, рабочее место, настройка драйверов, установка плагинов, оптимизация и организационные вопросы.

- **Выбор основной программы DAW** — FL Studio 25, Ableton Live 12, другие DAW
- **Какую DAW выбрать** — первый взгляд, визуал, логика работы, удобство
- **Необходимое оборудование** — WIN/MAC, наушники, мониторы, микрофон, MIDI-клавиатура, аудиокарта
- **Рабочее место** — комната, стол, расстановка мониторов
- **Аудио драйверы** — ASIO, внешняя звуковая карта
- **Установка программ и VST плагинов** — последовательная установка, сэмпл-паки
- **Оптимизация компа и ресурсов** — ЦП, задержка звука (буфер)
- **Программы для заметок** — Xmind, Obsidian, тетрадка
- **Организационные вопросы по курсу** — чаты, стримы, домашки, материалы, эвенты

### [Этап №1 — Первые Биты](etap1/index.md)
Создание первых треков: сэмплы, драмка, мелодия, бас, микшер и рендер.

- **Что такое сэмпл** — ударка, мелодический сэмпл, сэмпл-паки
- **Драмка и построение бита** — Kick, Snare, Clap, Hat; частотные диапазоны
- **Подбор сэмплов и структура** — один пак, жанр, баланс громкости
- **Работа с микшером** — громкость, панорама, расположение дорожек
- **Мелодия в бите** — поиск сэмпла, BPM, стретч/warp
- **Бас в бите** — 808, риз-бас, тональность, пиано-ролл
- **Мастер канал** — сумма звуков, главный принцип
- **Рендер трека** — громкость, качество, экспорт
- **Бит в FL Studio** — Channel Rack, паттерны, луп
- **Бит в Ableton** — Clip, Session/Arrangement View

### [Этап №2 — Сведение и VST Плагины](tom3/index.md)
Прикладные навыки для создания треков: обработка звука, плагины, мастеринг.

- **Основные понятия** — широкий, яркий, плотный, атмосферный звук
- **EQ / Эквализация** — срез частот, добавление, динамическая EQ
- **Saturation / Сатурация** — гармоники, уплотнение, дисторшн
- **Reverb / Реверберация** — пространство, глубина, планы в треке
- **Delay / Делэй** — повтор, фидбэк, пинг-понг
- **Chorus / Хорус** — ширина, наслонение, винтаж
- **Compression / Компрессия** — уравнивание громкости, multiband
- **Limiter и Clipper** — потолок, усреднение, сатурация
- **Стилизация сэмплов** — ретро, ShaperBox 3, EFX Motions
- **Автоматизация плагинов** — движение в структуре трека
- **Последовательность обработки** — порядок плагинов, сенды
- **Мастер-канал** — Tonal Balance, LUFS, Speccraft, Soothe2
- **Работа с референсами** — SpectraLayers, A/B сравнение
- **Слух** — частоты, тренировка, развитие
- **Психоакустика** — маскировка, Haas-эффект, погружение
- **Сведение (mixing)** — баланс, чёткость, пространство
- **Мастеринг** — финальная обработка для релиза

### [Этап №5 — Теория музыки для продюсеров](etap5/index.md)
Теория, которая сделает вашу музыку лучше.

- **Нотная грамота** — ноты, клавиатура, октавы
- **Аккорды и гармония** — мажор, минор, прогрессии
- **Ритм и перкуссия** — BPM, время, паттерны

### [Плагины — подборка для изучения](plugins/index.md)
Обзоры VST-плагинов: компрессоры, EQ, реверб, дилеи, сатурация.

- **Подборка от Meraki** — авторская коллекция инструментов
- **Waves Abbey Road** — эмуляция оборудования Abbey Road Studios
- **Waves CLA** — плагины Chris Lord-Alge
- **Waves MannyM** — плагины Manny Marroquin (9x Grammy)
- **Waves Maserati** — плагины Tony Maserati

### [Основы звукозаписи](zvuk/index.md)
Фундамент, без которого невозможно создавать музыку.

- **Что такое звук** — физика, частота, амплитуда, waveform
- **Оборудование** — микрофоны, аудиоинтерфейсы, мониторы, наушники
- **DAW** — обзор программ: Ableton, FL Studio, Logic Pro, Reaper
- **Запись инструментов** — вокал, гитара, живые инструменты
