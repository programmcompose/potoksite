# 🎵 Создание музыки — Интерактивный курс

## Добро пожаловать!

Это **полный курс по созданию музыки** — от физики звука до финального мастеринга.
Курс создан для того, чтобы вы могли пройти путь от полного новичка до уверенного
музыкального продюсера, работая в своей домашней студии.

---

## 🎯 Для кого этот курс?

| Уровень | Что вы получите |
|---------|----------------|
| **Новичок** | Понимание основ звука, выбор первого оборудования, работа в DAW |
| **Продвинутый** | Глубокое знание теории музыки, техники сведения, работа с плагинами |
| **Профессионал** | Справочник по мастерингу, глоссарий, чек-листы для работы |

> [!TIP]
> Не важно, с какого уровня вы начинаете — просто переходите к тому разделу,
> который вам интересен. Ссылки между главами помогут вам углублять знания.

---

## 📚 Структура курса

### Том I: Основы звукозаписи
Фундамент, без которого невозможно создавать музыку.

- **Что такое звук** — физика, частота, амплитуда, waveform
- **Оборудование** — микрофоны, аудиоинтерфейсы, мониторы, наушники
- **DAW** — обзор программ: Ableton, FL Studio, Logic Pro, Reaper

### Том II: Теория музыки для продюсеров
Теория, которая сделает вашу музыку лучше.

- **Нотная грамота** — ноты, клавиатура, октавы
- **Аккорды и гармония** — мажор, минор, прогрессии
- **Ритм и перкуссия** — BPM, время, паттерны

### Том III: Практика производства
Прикладные навыки для создания треков.

- **Запись инструментов** — вокал, гитара, живые инструменты
- **Сведение (mixing)** — EQ, компрессия, панорамирование
- **Мастеринг** — финальная обработка трека

### Дополнительно

- **Маршрут обучения** — пошаговый план на 12 недель
- **Глоссарий** — словарь терминов музыкального производства

## 🚀 Как пользоваться сайтом

1. **Навигация** — используйте левую боковую панель для перехода между разделами
2. **Поиск** — нажмите `Ctrl+K` (или `Cmd+K` на Mac) для быстрого поиска
3. **Оглавление** — правая панель показывает структуру текущей страницы
4. **Ссылки** — все термины связаны с глоссарием для быстрого изучения

> Этот сайт работает офлайн после загрузки. Все материалы доступны

## 🛠️ Системные требования

Для начала работы вам понадобится:

- **Компьютер** с минимальными характеристиками (4 ГБ RAM, 2 ядра)
- **Аудиоинтерфейс** (минимум 2 входа)
- **Наушники или мониторы**
- **DAW** (любая на ваш выбор — см. [обзор DAW](tom1/daw.md))
